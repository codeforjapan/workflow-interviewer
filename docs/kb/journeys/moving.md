# 転入・転出・転居 住民ジャーニー

> **このファイルの役割**
> 住民が「引越し」という1つの出来事でたどる行政接点の全体像を示す。
> 個々の業務フローは `workflows/` を参照。業務間の「穴」は `incident-catalog/` を参照。
> 自治体職員は**このファイルから読み始めることを推奨する**。

---

## ジャーニーの全体像

```
【住民の体験】
引越し決定
  │
  ├─ 転出前 ────────────────────────────────────────────────
  │   └─ 転出届（旧住所の市区町村へ）
  │       関連: マイナカード所持者はオンライン転出も可
  │
  ├─ 転入後14日以内 ─────────────────────────────────────────
  │   ├─ 転入届（必須・起点）
  │   │
  │   └─ 転入届が起点となって連鎖する手続き
  │       ├─ マイナンバーカード記載事項変更（当日・任意）
  │       ├─ 国民健康保険（加入・変更・脱退）
  │       ├─ 児童手当の住所変更／新規認定    ← 子どもがいる場合
  │       ├─ 介護認定の引き継ぎ申請          ← 65歳以上・要介護の場合
  │       └─ 生活保護の移管申請              ← 受給者の場合
  │
  └─ 転入後・随時 ────────────────────────────────────────────
      ├─ 印鑑登録（転入で旧登録は失効）
      ├─ 運転免許証の住所変更（警察署・免許センター）
      └─ 郵便転送（民間手続き）
```

## 業務連鎖マトリクス

転入届を起点として、**何が・いつ・誰に必要か**を示す。

| 連鎖する手続き | 対象者 | 期限 | 窓口 | 案内の責任 |
|---|---|---|---|---|
| マイナカード記載変更 | カード所持者 | 転入届と同日推奨 | 市民課 | 転入担当者 |
| 国民健康保険 加入 | 社保非加入者 | 転入日から14日 | 保険年金課 | 転入担当者 |
| 国民健康保険 脱退 | 転出元で国保加入 | 速やかに | 保険年金課 | 転入担当者 |
| 児童手当 新規認定 | 18歳以下の子あり | **転入日から15日以内**（支給開始月に影響） | 子育て支援課 | 転入担当者 |
| 介護認定 引き継ぎ | 要介護認定者 | **転入日から14日以内**（切れると認定失効） | 介護保険課 | 転入担当者 |
| 生活保護 移管 | 受給者 | 速やかに | 福祉課 | 転入担当者 |
| 印鑑登録 | 登録希望者 | いつでも | 市民課 | 任意案内 |

**→ 「転入担当者」が案内の責任を持つ手続きが6つある。**
これを担当者の個人知識に任せていることが、全国共通の「穴」の根源。

## 穴の連鎖マップ

この業務ジャーニーで発生しうるインシデントを、発生箇所と連鎖先で示す。

```
転入届受付
  │
  ├─[穴A] 介護保険証を持参していても担当者が気づかない
  │        → 14日以内申請を逃す → 認定失効 → サービス停止
  │        → incident-catalog/care-handover-14days.md
  │
  ├─[穴B] DV被害者の転入で新住所を複数課に連携する際に漏洩リスク
  │        → 子育て支援課の給付通知書に新住所が印字される
  │        → 福祉課の扶養照会で加害者に住所が判明する
  │        → incident-catalog/dv-cross-department.md
  │
  └─[穴C] 生活保護受給者の転入を市民課が把握しても福祉課に連携しない
           → 数週間、無保護状態が継続する
           → incident-catalog/welfare-transfer-gap.md
```

## 担当者向け：転入受付時のチェックフロー

```
転入届を受け取ったら、以下を順番に確認する

□ 1. 本人確認（必須）
       → workflows/_standardized-20/jyumin-ido/ 参照

□ 2. DV支援措置の確認（必須）
       システムでフラグ確認。フラグなしでも申告があれば係長へ即エスカレーション
       → incident-catalog/dv-cross-department.md 参照

□ 3. 世帯構成の確認（必須）
       ├─ 18歳以下の子がいる → 子育て支援課への案内（児童手当・15日ルール説明）
       ├─ 65歳以上がいる    → 介護保険証の有無を確認
       │                       持参あり → 介護保険課への案内（14日ルール説明）
       └─ 困窮の兆候がある  → 福祉課への案内または係長エスカレーション

□ 4. 生活保護受給証明書の確認（任意持参の場合）
       持参あり → 福祉課に連絡票を起こす

□ 5. 関連手続き一覧の案内票を渡す（必須）
       ※ 案内票は各自治体でカスタマイズ
```

## 自治体カスタマイズのポイント

このジャーニーファイルをForkして自治体版を作る際に、変更が必要な箇所。

| 項目 | 標準の記述 | 自治体が追記すること |
|---|---|---|
| 窓口の物理的配置 | 記述なし | 1F・2Fの配置、窓口番号、動線 |
| システム連携の有無 | 日次バッチ（一般的） | リアルタイム連携の有無 |
| DV対応の個室 | 推奨と記述 | 個室の場所・代替手段 |
| 案内票の様式 | テンプレート参照 | 自治体の実際の様式を添付 |
| エスカレーション先 | Level 1〜4で記述 | 担当者名・内線番号 |
