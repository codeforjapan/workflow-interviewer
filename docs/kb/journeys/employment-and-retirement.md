# 就職・退職 住民ジャーニー

> **このファイルの役割**
> 住民が「退職」または「就職」という出来事でたどる行政接点の全体像を示す。
> 個々の業務フローは `workflows/` を参照。業務間の「穴」は `incident-catalog/` を参照。
> 自治体職員は**このファイルから読み始めることを推奨する**。

---

## ジャーニーの全体像

```
【住民の体験】

【フェーズ1：退職時】

退職決定
  │
  └─ 退職日 ──────────────────────────────────────────────────
      │
      ├─[退職日～14日以内] ────────────────────────────────────
      │   ├─ 国民健康保険 加入届（必須・起点）
      │   │   前職の健康保険喪失証明書を持参
      │   │
      │   ├─ 国民年金 第1号加入届（必須）
      │   │   退職日時点で自動脱退、自分で加入届を出す必要あり
      │   │
      │   └─ 個人住民税 普通徴収への切替
      │       ├─ 残税額一括徴収（最大2か月分の予定納税含む）
      │       │   ※ 予想外の高額請求として認識されることが多い
      │       └─ 普通徴収納付書が郵送される
      │
      └─[退職後・随時] ────────────────────────────────────────
          ├─ 国民年金免除・猶予申請（収入激減の場合）
          │   申請期限: 納付期限から2年以内
          │
          └─ 確定申告（翌年2月末まで）
              ├─ 医療費控除（退職後の医療費が対象）
              ├─ 失業給付受給時の兼ね合い
              └─ 雑損控除（災害時の所得控除）


【フェーズ2：就職時（再就職・初就職）】

就職（社保加入企業への入社）
  │
  └─ 就職日 ──────────────────────────────────────────────────
      │
      ├─[就職日～14日以内] ────────────────────────────────────
      │   ├─ 国民健康保険 脱退届（必須）
      │   │   社保の資格取得証明書を持参
      │   │   ※ 自動脱退されない自治体が多い → 住民が申請必須
      │   │
      │   ├─ 国民年金 第2号自動切替（自動処理）
      │   │   社保加入により厚生年金へ自動的に切り替わる
      │   │   （住民の申請は不要）
      │   │
      │   └─ 個人住民税 特別徴収への切替（給与天引き）
      │       └─ 事業者（新勤務先）→ 市区町村で自動連携
      │           ※ 住民は主体的な申請不要だが、時間差あり
      │
      └─[就職後・随時] ────────────────────────────────────────
          └─ マイナンバー等の扶養申告手続き
              （勤務先の人事担当へ報告）
```

## 業務連鎖マトリクス

退職・就職時に、**何が・いつ・誰に必要か**を示す。

| 連鎖する手続き | 対象者 | 期限 | 窓口 | 案内の責任 |
|---|---|---|---|---|
| **【退職時】** |
| 国民健康保険 加入 | 社保喪失者全員 | 退職日から14日以内 | 保険年金課 | 退職者が窓口来訪時の担当者 |
| 国民年金 第1号加入 | 60歳未満の退職者 | 退職日から14日以内 | 保険年金課 | 国保加入時の同一課で案内 |
| 個人住民税 普通徴収切替 | 退職者全員 | 速やかに | 税務課 | 税務課（前職からの連携） |
| 個人住民税 残税一括徴収の説明 | 特別徴収から普通徴収へ | 退職時 | 税務課 | 税務課（重要：周知不足の穴） |
| 国民年金 免除・猶予申請 | 収入激減者 | 納付期限から2年以内 | 保険年金課 | 窓口での任意案内 |
| 確定申告 | 医療費控除等対象者 | 翌年2月末 | 税務署（一部は市） | 税務署・市税務課 |
| **【就職時】** |
| 国民健康保険 脱退 | 社保新規加入者 | 就職日から14日以内 | 保険年金課 | 保険年金課（要・主動申請） |
| 国民年金 第2号への切替 | 厚生年金加入者 | 自動処理 | 保険年金課（参考） | 勤務先人事部・日本年金機構 |
| 個人住民税 特別徴収切替 | 就職者全員 | 給与支払開始から | 税務課 | 勤務先が市町村に申告 |

**→ 「退職時」には複数の課の手続きが並行発生し、住民が手続き漏れするリスクが高い。**

## 穴の連鎖マップ

この業務ジャーニーで発生しうるインシデントを、発生箇所と連鎖先で示す。

```
退職届出
  │
  ├─[穴A] 退職後2週間「何をすればいいかわからない」空白
  │        → 国民健康保険・国民年金の未加入期間が生じる
  │        → 無保険状態で医療受診が必要になる
  │        → 年金納付記録が途絶える
  │        → incident-catalog/retirement-uninsured-gap.md
  │
  ├─[穴B] 個人住民税の残額一括徴収を事前に説明されない
  │        → 「月10万円だと思ったら20万円請求された」と混乱
  │        → 予想外の納税義務で困窮化のきっかけになる
  │        → 滞納に至るリスク
  │        → incident-catalog/tax-lump-sum-shock.md
  │
  ├─[穴C] 国民年金免除申請を知らず、そのまま未納状態が続く
  │        → 将来の年金受給資格を失うリスク（25年→10年に短縮されても基礎年金1/4喪失）
  │        → 老齢期の貧困化
  │        → incident-catalog/pension-nonpayment-cascade.md
  │
  └─[穴D] ハローワーク手続きを優先し、市区町村での手続きを後回し
           → 失業給付の受給を始めるが、国保・年金はまだ未加入
           → 2～3か月経ってから「あ、市役所に行かなきゃ」と気づく
           → 遡及加入が複雑になる
           → incident-catalog/hwk-priority-gap.md
```

## 担当者向け：退職者受付時のチェックフロー

```
「退職した」と窓口に来た住民への初回対応チェックリスト

□ 1. 本人確認と退職日の確認（必須）
       退職日が14日以上前の場合、対応が変わる
       → workflows/_standardized-20/kokumin-hoken/ 参照

□ 2. 前職の「健康保険資格喪失証明書」の確認
       持参なし → 「勤務先または前の市町村に依頼してください」と説明

□ 3. 個人住民税の在来納付方法を確認
       ├─ 前職で特別徴収 → 普通徴収への切替手続きが自動か確認
       ├─ 残税額の説明      → 「予定納税が含まれる」ことを特に強調
       └─ 税務課への紹介   → 専門的質問は即座に税務課へ

□ 4. 「国保・年金・税の3点セット」として案内（重要）
       □ 国民健康保険 加入届（保険年金課）
       □ 国民年金 第1号加入届（保険年金課、同時進行）
       □ 個人住民税 普通徴収切替確認（税務課）

       「14日以内がデッドラインです」と明記した案内票を渡す

□ 5. 困窮の兆候がないか確認
       ├─ 「今後の生活基盤はありますか？」と簡単に聞く
       ├─ 失業給付の見込み額が不確実な様子 → 福祉課へエスカレーション
       └─ 年齢が55歳以上で就職見込みが不明 → 福祉課への相談を勧める

□ 6. ハローワーク手続きとの時系列を確認
       市役所に来る前にハローワークに行く可能性が高い
       → 「こちらの手続きも同じ期限です」と同等の重要性を伝える
       → ハローワークとの連携手帳（あれば配布）

□ 7. 年金免除・猶予申請の説明（条件付き）
       「収入が大きく減った場合は、後で免除申請ができます」
       あくまで「その時に相談を」のレベルで案内（過度な説明は混乱招く）
```

## 自治体カスタマイズのポイント

このジャーニーファイルをForkして自治体版を作る際に、変更が必要な箇所。

| 項目 | 標準の記述 | 自治体が追記すること |
|---|---|---|
| 国保・年金・税の一括案内手段 | 記述なし | 実物の案内票・チラシをファイルに添付 |
| システム連携の有無 | 自動連携なしと想定 | リアルタイム連携の有無・遅延時間 |
| 税務課との連携 | 税務課が対応と記述 | 残税説明の実施状況・トラブル事例 |
| ハローワークとの協定 | 記述なし | 連携協定の有無・窓口同時対応の可否 |
| 福祉課へのエスカレーション | Level 1～3で記述 | 実際の係長名・内線番号 |
| 年金免除申請の説明レベル | 簡潔と記述 | 実際の説明時間・資料の有無 |
| 失業給付との兼ね合い | 一般的な注意程度 | 給付中の住民税額・国保料のシミュレーション例 |
