# おくやみ（死亡・遺族手続き）住民ジャーニー

> **このファイルの役割**
> 「おくやみ」は自治体手続きの中で最も多くの窓口をまたぐジャーニーのひとつ。
> 悲嘆の中にある遺族が、期限のある手続きを複数抱えながら行政を回る。
> 各手続きの担当課がバラバラに動き、**ワンストップ化が最も求められている領域**。
> 個々の業務フローは `workflows/` を参照。業務間の「穴」は `incident-catalog/` を参照。

---

## ジャーニーの全体像

```
【遺族の体験】
死亡（自宅・病院・施設）
  │
  ├─ 7日以内【期限あり】─────────────────────────────────
  │   ├─ 死亡届提出（戸籍窓口）
  │   │   必要書類: 死亡診断書（医師が交付）
  │   │   発行物: 火葬許可証（同窓口で同時発行）
  │   │   →戸籍の死亡記載
  │   │
  │   └─ 火葬許可証を得る
  │       →火葬場に提示して火葬実施
  │
  ├─ 14日以内【期限あり】────────────────────────────────
  │   ├─ 世帯主変更届（住民記録課）← 世帯主が亡くなった場合のみ
  │   ├─ 国民健康保険 資格喪失届（保険年金課）
  │   ├─ 後期高齢者医療 資格喪失届（保険年金課）←75歳以上の場合
  │   └─ 介護保険 資格喪失届（介護保険課）←要介護認定者の場合
  │
  ├─ できるだけ速やかに──────────────────────────────────
  │   ├─ 年金受給停止手続き（日本年金機構）
  │   │   関連: 保険年金課で案内可
  │   │
  │   ├─ 未支給年金の請求（5年以内）
  │   │   条件: 亡くなった月分まで支給を受けていない場合
  │   │
  │   └─ 遺族年金の請求（遺族基礎年金・遺族厚生年金）（5年以内）
  │       遺族年金の種類により条件あり
  │
  └─ 相続関連（数か月以内）───────────────────────────────
      ├─ 固定資産税 所有者変更登記（法務局登記完了後）
      │   関連: 税務課で相続人を確認・ファイリング
      │
      ├─ 準確定申告（相続開始から4か月以内）（税務署）
      │   条件: 亡くなった者に所得があった場合（給与・事業・不動産など）
      │
      └─ 相続放棄（相続開始から3か月以内）（家庭裁判所）
          条件: 負債が資産を上回る場合など
```

## 業務連鎖マトリクス

死亡届を起点として、**何が・いつ・誰に必要か**を示す。

| 連鎖する手続き | 対象者 | 期限 | 窓口 | 案内の責任 |
|---|---|---|---|---|
| 死亡届 | 必須（届出義務者） | **7日以内** | 戸籍課 | — |
| 火葬許可証の申請 | 必須（火葬実施前） | 7日以内 | 戸籍課（同時発行） | 戸籍担当者 |
| 世帯主変更届 | 世帯主が亡くなった場合 | **14日以内** | 住民記録課 | 戸籍担当者 |
| 国民健康保険 資格喪失届 | 国保加入者 | **14日以内** | 保険年金課 | 戸籍担当者 |
| 国民健康保険 保険証返却 | 国保加入者 | 14日以内 | 保険年金課 | 保険担当者 |
| 後期高齢者医療 資格喪失届 | 75歳以上の医療制度加入者 | **14日以内** | 保険年金課 | 戸籍担当者 |
| 後期高齢者医療 保険証返却 | 75歳以上の医療制度加入者 | 14日以内 | 保険年金課 | 保険担当者 |
| 介護保険 資格喪失届 | 要介護認定者・介護保険加入者 | **14日以内** | 介護保険課 | 戸籍担当者 |
| 介護保険 保険証返却 | 要介護認定者 | 14日以内 | 介護保険課 | 介護保険担当者 |
| 年金受給停止届 | 年金受給者 | 速やかに | 日本年金機構（郵送可） | 保険年金課が案内 |
| 未支給年金の請求 | 遺族 | **5年以内** | 日本年金機構 | 保険年金課が案内 |
| 遺族基礎年金・遺族厚生年金の請求 | 遺族（要件あり） | **5年以内** | 日本年金機構 | 保険年金課が案内 |
| 固定資産税 所有者変更 | 相続人 | 登記完了後 | 税務課 | 税務課が案内 |
| 準確定申告 | 故人に所得があった場合 | **相続開始から4か月以内** | 税務署 | 税務課が案内 |
| 相続放棄 | 相続人（必要な場合） | **相続開始から3か月以内** | 家庭裁判所 | 福祉課等が案内 |
| 各種給付の停止（児童手当） | 故人が受給者だった場合 | 速やかに | 子育て支援課 | 戸籍担当者が案内 |
| 各種給付の停止（生活保護） | 故人が受給者だった場合 | 速やかに | 福祉課 | 福祉課で対応 |

**→ 「戸籍担当者」が案内の責任を持つ手続きが6つある。**
これを担当者の個人知識に任せていることが、全国共通の「穴」の根源。

## 穴の連鎖マップ

この業務ジャーニーで発生しうるインシデントを、発生箇所と連鎖先で示す。

```
死亡届受付
  │
  ├─[穴A] 「おくやみ窓口」がなく、遺族が複数課をたらい回しにされる
  │        → 期限のある手続き（14日・4か月・3か月）を見落とす
  │        → 年金返納請求が来たり、給付を受け損なったりする
  │        → incident-catalog/death-multiwindow-chaos.md
  │
  ├─[穴B] 年金受給停止の連絡が遅れ、過払い年金の返還請求が来る
  │        → 遺族が「どうして返さなくちゃいけないのか」と混乱
  │        → 福祉的困難（返納資金がない遺族）へ発展
  │        → incident-catalog/overpayment-pension.md
  │
  ├─[穴C] 未支給年金・遺族年金の請求を知らず、受給できるはずの給付を受け取らない
  │        → 5年で時効消滅。最大数百万円の喪失
  │        → 本来は保険年金課が案内する責務
  │        → incident-catalog/unpaid-pension-loss.md
  │
  ├─[穴D] 死亡届受付時に「次にすること一覧」を渡す仕組みがない
  │        → 遺族が何をすべきか把握できない
  │        → 期限を過ぎてから「あ、申請漏れだ」と気づく（その時は遅い）
  │        → incident-catalog/death-no-checklist.md
  │
  └─[穴E] 準確定申告（4か月以内）の期限を担当者が案内しない
           → 申告漏れ・延滞税発生
           → 遺産分割協議時に「故人の税額は？」という問題が生じる
           → incident-catalog/preliminary-tax-deadline.md
```

## 担当者向け：死亡届受付時のチェックフロー

```
死亡届を受け取ったら、以下を順番に確認する

□ 1. 届出人の確認と死亡診断書の確認（必須）
       ├─ 届出義務者か確認
       ├─ 死亡診断書が医師署名・押印されているか
       └─ 死亡の日時・場所が明確か
       → workflows/_standardized-20/shikin-todoke/ 参照

□ 2. 戸籍電算システムへ登録（必須）
       → 火葬許可証を発行

□ 3. 遺族の状況を確認（必須）
       ├─ 世帯主が亡くなったか → 世帯主変更届が必要（14日以内）
       ├─ 65歳以上か → 介護保険証・後期高齢者医療証の有無を確認
       ├─ 子がいるか → 児童手当受給中なら給付停止が必要
       ├─ 困窮の兆候がないか → 福祉課へ
       └─ 年金受給者か → 保険年金課への案内が必要

□ 4. 年金受給者の確認（任意だが強く推奨）
       ┌─ 質問例: 「故人が年金を受け取っていましたか？」
       │
       ├─ YES の場合
       │  └─ 【必ず保険年金課に連絡票を起こす】
       │     保険年金課が年金受給停止を案内する責任がある
       │
       └─ NO または 不明
          └─ 質問履歴を記録

□ 5. 関連手続き一覧の案内票を渡す（必須）
       ┌─ 案内票に以下を含める
       │  • 世帯主変更届（14日以内）
       │  • 国民健康保険 資格喪失（14日以内）
       │  • 後期高齢者医療 資格喪失（14日以内）
       │  • 介護保険 資格喪失（14日以内）
       │  • 年金受給停止（速やかに）
       │  • 未支給年金・遺族年金の請求（5年以内）
       │  • 準確定申告（4か月以内・税務署）
       │  • 相続放棄（3か月以内・家庭裁判所）
       │
       └─ 各課の窓口番号・内線をすべて記載

□ 6. 各課への連絡票を起こす（重要）
       ├─ 保険年金課
       │  └─ 世帯主が亡くなった、年金受給者、医保加入者の別
       ├─ 介護保険課
       │  └─ 要介護認定の有無
       ├─ 子育て支援課
       │  └─ 児童手当受給者だった場合
       └─ 福祉課
          └─ 困窮の兆候、生活保護受給者だった場合

□ 7. 【遺族向けチェックリストを渡す】 ★最も重要
       ※ テンプレートは本ファイル下部に記載
```

## 遺族向け案内票（テンプレート）

死亡届受付時に配布すべき案内票。自治体カスタマイズ必須。

```

このたびは、お亡くなりになられましたお悔やみを申し上げます。

以下の手続きをお手伝いいたします。
期限が決まっているものがありますので、ご確認ください。

【 7日以内 】
□ 火葬許可証 → 本日、戸籍課でお渡しします

【 14日以内 】
□ 世帯主変更届（世帯主が亡くなった場合）
   窓口: 住民記録課 ☎ ____

□ 国民健康保険 資格喪失届 + 保険証返却
   窓口: 保険年金課 ☎ ____

□ 後期高齢者医療 資格喪失届 + 保険証返却（75歳以上）
   窓口: 保険年金課 ☎ ____

   窓口: 介護保険課 ☎ ____

□ 固定資産税 所有者変更（相続登記完了後）
   窓口: 税務課 ☎ ____

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
ご質問やご不明な点は、お気軽にお問い合わせください。
おくやみご相談窓口（設置自治体の場合）
☎ ____（内線 ____）
```

## 自治体カスタマイズのポイント

このジャーニーファイルをForkして自治体版を作る際に、変更が必要な箇所。

| 項目 | 標準の記述 | 自治体が追記すること |
|---|---|---|
| おくやみワンストップ窓口の設置有無 | 「推奨」と記述 | 実際の設置有無・窓口名 |
| 案内票の様式 | テンプレート参照 | 自治体の実際の様式を添付 |
| 年金事務所との連携協定 | 記述なし | リアルタイム連携の有無 |
| 準確定申告の案内担当 | 税務課 | 市民向け説明資料の有無 |
| 困窮世帯への福祉相談体制 | 記述なし | 生活保護・福祉資金の相談窓口 |
| 相続放棄の案内体制 | 記述なし | 司法書士・弁護士の紹介制度の有無 |
| マイナンバーカードの返却手続き | 記述なし | 必要に応じて戸籍課で対応 |
| 窓口の物理的配置 | 記述なし | 1F・2Fの配置、窓口番号、動線 |
| エスカレーション先 | Level 1〜4で記述 | 担当課長・班長の内線番号 |
