# 年度サイクル（予算編成〜執行〜決算） 🔄 O1

> **このファイルの役割**
> 行政組織が1年間で経験する「予算編成〜執行〜決算」の時系列構造を示す。
> 個々の業務フローは `workflows/` を参照。業務間の「穴」は `incident-catalog/` を参照。
> **新人職員・異動職員は「このファイルから読み始める」ことを推奨する。**

---

## ジャーニーの全体像

年度サイクル（O1）は、以下の4つの主要フェーズで構成される：

```
【全庁の体験】

年度開始 (4月)
  ├─────────────────────────────────────────
  │ [フェーズ1] 年度開始・前年度決算整理
  │   4月: 新年度開始・前年度出納整理期間・新人職員着任
  │   5月: 前年度決算整理・専決・税務通知準備
  │
  ├─────────────────────────────────────────
  │ [フェーズ2] 上半期（執行開始〜予算編成準備）
  │   6月: 固定資産税納期・個人住民税通知発送
  │   7月〜8月: 予算編成ヒアリング開始・概算要求受付
  │
  ├─────────────────────────────────────────
  │ [フェーズ3] 補正予算・予算編成本格化
  │   9月: 9月議会（補正予算・決算認定）
  │   10月〜11月: 予算要求書提出・査定・改築繰り返し
  │
  └─────────────────────────────────────────
    [フェーズ4] 年度末・新年度準備
    12月: 12月議会・予算編成大詰め
    1月〜2月: 当初予算案策定・議案作成・市長査定
    3月: 3月議会（予算議決）・年度末執行集中・人事内示
    ↓
    翌年4月に循環
```

### 各月の詳細フロー

#### **4月：新年度開始・引き継ぎ・前年度出納整理期間**

```
新年度初日(4/1)
  ├─ 新職員が配置される
  │   └─ 引き継ぎ資料がない → 担当者に直撃
  ├─ 前年度出納整理期間（4/1〜4/30）
  │   └─ 去年の支出が今月まで処理できる
  ├─ 税務・福祉系の異動に伴う資格要件確認
  └─ 新規事業の開始手続き
```

**主要業務**：
- 新年度人事異動が発令される
- 前年度の支出を遅延なく処理する期間開始
- 新規アルバイト・嘱託職員の配置
- 税務システムの年度更新
- 福祉サービスの申請受付再開（保育など）

**関係課**：全課（人事異動全課を巻き込む）

**リスク**：
- 前年度の決算書を作成する作業が4月中に並行して起きる → 担当者が多忙
- 継続事業の引き継ぎ資料がない → 新担当が初からアドホック対応
