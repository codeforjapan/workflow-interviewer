# 監査・検査対応サイクル 🔄 O4

> **このファイルの役割**
> 定期監査・決算審査・行政評価・国県検査など、自治体が年間を通して対応する各種監査・検査の構造を可視化する。
> 「証拠を事後に作る」「対応が形式的になる」という病理の構造と、予防的な対応のあり方を示す。

---

## サイクル概要

自治体は年間を通して複数の監査・検査に対応する。主なものは以下の4種類で、それぞれ実施主体・時期・目的が異なる。

| 種別 | 実施主体 | 主な時期 | 目的 |
|---|---|---|---|
| **定期監査** | 監査委員（独立機関） | 通年（年2〜3回） | 財務・事務の適正性確認 |
| **決算審査** | 監査委員 | 7〜8月 | 決算書の正確性審査 |
| **行政評価** | 首長部局 or 外部評価委員 | 9〜11月 | 施策・事業の有効性評価 |
| **国・県の補助金検査** | 国・都道府県 | 随時（補助事業ごと） | 補助金の適正執行確認 |
| **外部監査** | 外部監査人（公認会計士等） | 随時（契約による） | 包括外部監査・個別外部監査 |

---

## タイムライン

| 時期 | 主な監査・検査 | 発生するリスク・穴 |
|---|---|---|
| **4〜5月** | 前年度補助金の実績報告審査 | 書類の不備・対象外経費が発覚するタイミング |
| **6月** | 定期監査（上半期） | 実地検査の直前に証拠書類を整備する「駆け込み対応」 |
| **7〜8月** | 決算審査（監査委員） | 決算書の数値と証拠書類の照合。担当異動後の最初の山場 |
| **9月議会** | 決算認定（議会審議） | 監査意見書・決算書の内容を議員が審議 |
| **10〜11月** | 行政評価（事業評価） | 評価のための数値が用意されておらず事後収集になる |
| **11〜12月** | 定期監査（下半期）または随時監査 | 上半期指摘事項の改善状況確認 |
| **随時** | 国・県補助金検査（実地） | 通知から2週間程度での対応を求められることがある |

---

## 穴の連鎖

```
[日常業務] 証拠書類の管理が平時からずさん（口頭決裁・記録なし）
  └→ [監査直前] 証拠書類を事後作成・整備する「追い書き」が発生
       └→ 監査委員・検査官の不信感 → 再調査・是正勧告
            └→ 職員の業務量増大・改善対応のための残業
                 └→ 翌年度も同じパターンが繰り返される
```

```
[補助金交付後] 実績報告の管理が担当者個人に依存
  └→ [担当者異動（O3交差）] 補助金台帳・証拠書類の所在が不明に
       └→ [国・県検査] 証拠書類が提出できず → 補助金返還命令
            → hojokin-kanri/gap-notes.md 参照
```

```
[監査委員の指摘] 是正勧告が出る
  └→ [組織の反応] 「次回までに改善」と回答するが改善されない
       └→ [翌年度監査] 同じ指摘が繰り返される（慢性的な未改善）
            └→ 議会・住民への説明責任の問題
```

## 監査対応の病理構造

### なぜ「証拠の事後作成」が起きるか

```
平時: 口頭決裁・記録なし → 業務が速く回る（短期的メリット）
         ↓
監査前: 「この案件、記録ない」と気づく
         ↓
対応: 「記録を作る」（追い書き、日付の遡り） ← 違法リスク
```

平時からの記録習慣がないことが根本原因。「監査のために書類を作る」という本末転倒が常態化する。

### なぜ指摘事項が改善されないか

| 要因 | 内容 |
|---|---|
| 担当者の異動 | 指摘を受けた担当者が異動し、後任者が経緯を知らない（O3交差） |
| 改善コストの高さ | システム改修・規程改正が必要な指摘は予算・時間がかかる |
| フォロー体制なし | 監査委員は是正勧告するが、改善の実施確認は自治体任せ |
| 優先順位の低さ | 「監査委員の指摘」が首長・管理職の優先アジェンダに入らない |

## 担当者向け: 監査対応チェックリスト

### 定期監査・決算審査 事前準備

```
□ 1. 監査対象期間の決裁書類・証拠書類がすべて揃っているか確認
□ 2. 前回監査の指摘事項と改善状況を整理しておく
□ 3. 補助金台帳・発注台帳の最新化を確認
□ 4. 会計担当・財政担当と事前すり合わせを行う
□ 5. 監査委員からの質問想定問答を準備する
```

### 国・県補助金検査 事前準備

```
□ 1. 補助金交付決定通知・実績報告・精算書類一式を一か所に集める
□ 2. 補助対象経費の領収書・支出証拠書類を時系列で整理
□ 3. 交付条件・補助対象経費の解釈基準を再確認
□ 4. 担当者異動がある場合は前任者に確認事項を事前リストアップ
```

## バックオフィスジャーニーとの交差点

| 交差するOコード | 交差の内容 | リスク |
|---|---|---|
| **O1** (年度サイクル) | 決算審査は年度決算（O1）の直接的なアウトプット | 予算執行の記録不備が決算審査で発覚 |
