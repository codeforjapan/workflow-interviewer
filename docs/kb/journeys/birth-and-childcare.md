# 妊娠・出産・子育て 住民ジャーニー

> **このファイルの役割**
> 住民が「妊娠・出産・子育て」という長期のライフイベントでたどる行政接点の全体像を示す。
> 個々の業務フローは `workflows/` を参照。業務間の「穴」は `incident-catalog/` を参照。
> 自治体職員は**このファイルから読み始めることを推奨する**。

---

## ジャーニーの全体像

```
【住民の体験】
妊娠発覚
  │
  ├─ 妊娠初期（8〜12週） ────────────────────────────────────────
  │   └─ 妊娠届出・母子手帳交付（約10週が目安）
  │       関連: 伴走型相談支援（令和5年度〜・努力義務）
  │
  ├─ 妊娠中（13〜39週）────────────────────────────────────────
  │   ├─ 妊婦健診（公費負担・14回分）
  │   ├─ 特定妊婦の把握と支援（妊娠届時のアセスメント）
  │   │   ※10代・DV被害・困窮世帯
  │   └─ 国民年金産前産後免除申請（出産予定日4か月前から可能）
  │       ※出産予定日の4か月前〜産後翌々月末
  │
  ├─ 出産（分娩日）────────────────────────────────────────────
  │
  ├─ 出生後14日以内 ─────────────────────────────────────────
  │   ├─ 出生届（14日以内・必須）
  │   │   └─ 同時受付: 児童手当申請（出生翌日から15日以内が開始月ルール）
  │   │
  │   └─ 出生届と連鎖する手続き
  │       ├─ 子の国民健康保険加入（社保非加入世帯）
  │       └─ 出生後の伴走支援面談（令和5年度〜）
  │
  ├─ 出生後〜生後4ヶ月 ──────────────────────────────────────
  │   ├─ 乳児訪問（生後4週間以内推奨）
  │   └─ 予防接種スケジュール開始（ロタ・ヒブ・小児肺炎球菌等）
  │
  ├─ 生後12ヶ月 ─────────────────────────────────────────────
  │   ├─ 1歳児健診（指導）
  │   └─ 予防接種スケジュール継続（MR第1期等）
  │
  ├─ 生後18ヶ月 ─────────────────────────────────────────────
  │   └─ 1歳6ヶ月児健診（法定・必須）
  │
  ├─ 2歳〜3歳 ────────────────────────────────────────────────
  │   └─ 保育所等利用申請（入所希望月の2〜3ヶ月前）
  │
  └─ 3歳 ──────────────────────────────────────────────────────
      ├─ 3歳児健診（法定・必須）
      └─ 就学前の成長確認
```

## 業務連鎖マトリクス

妊娠届出を起点として、**何が・いつ・誰に必要か**を示す。

| 手続き | 対象者 | 期限 | 窓口 | 案内の責任 |
|---|---|---|---|---|
| 妊娠届・母子手帳交付 | 妊娠確認者 | 妊娠判明後・速やかに | 健康推進課・保健センター | - |
| 伴走型相談支援（初期面談） | 妊娠届出者全員 | 妊娠届出時またはその後 | 保健センター | 妊娠届担当者 |
| 妊婦健診（14回公費負担） | 妊娠届出者 | 妊娠初期〜出産直前 | 医療機関（妊婦健診事業） | - |
| 国民年金産前産後免除申請 | 国保加入の妊婦 | 出産予定日4ヶ月前〜産後翌々月末 | 保険年金課 | 妊娠届担当者（案内） |
| **出生届** | 出産者 | **出生後14日以内** | 市民課・出張所 | - |
| **児童手当申請** | 15歳以下の子親権者 | **出生翌日から15日以内** | 子育て支援課 | **出生届担当者** |
| 子の国民健康保険加入 | 社保非加入世帯 | 出生後速やかに | 保険年金課 | 出生届担当者 |
| 乳児訪問 | 生後新生児 | 生後4週間以内推奨 | 保健センター・訪問スタッフ | - |
| 予防接種スケジュール（定期） | 生後2ヶ月〜 | 月齢別（4週間隔等） | 医療機関 | 乳児訪問時・健診時案内 |
| 1歳6ヶ月児健診 | 生後18ヶ月児 | 18ヶ月到達時 | 保健センター | 健康推進課（集団通知） |
| 3歳児健診 | 生後36ヶ月児 | 36ヶ月到達時 | 保健センター | 健康推進課（集団通知） |
| 保育所等利用申請 | 認可保育園希望者 | 入所希望月の2〜3ヶ月前 | こども園課・保育支援センター | 申請者の事前確認 |

**→ 「出生届担当者」が児童手当の15日ルール案内の責任を持つ手続きが重要。**
これを担当者の個人知識に任せていることが、支給開始の遅れにつながる全国共通の「穴」。

## 穴の連鎖マップ

この業務ジャーニーで発生しうるインシデントを、発生箇所と連鎖先で示す。

```
妊娠届出（初期接点）
  │
  ├─[穴A] 出生届受付時に児童手当申請の15日ルールを案内しない
  │        または案内が不十分（「後で手続きすればいい」と安心させる）
  │        → 申請日が16日以上遅延
  │        → 支給開始が翌月以降になり、支給月が1〜2ヶ月遅れる
  │        → 新生児期の費用困窮に対応できない
  │        → incident-catalog/child-allowance-15day-window.md
  │
  ├─[穴B] 妊娠届時の伴走型相談支援が形骸化・チェックリストなし
  │        または10代妊娠・DV被害・困窮の兆候を見落とす
  │        → 特定妊婦の把握が遅れ、妊娠中の支援につながらない
  │        → 出産直前または産後に急に相談される
  │        → 虐待リスク増加、産後うつの悪化
  │        → incident-catalog/specific-pregnant-identification.md
  │
  ├─[穴C] 妊娠中に国民年金産前産後免除申請がされない
  │        または出産予定日4ヶ月前のタイミングを逃す
  │        → 納付期限を過ぎると未納扱いになる
  │        → 将来の年金受給額が減額される
  │        → 10年以上後に発覚することもある
  │        → incident-catalog/maternity-pension-exemption-missed.md
  │
  ├─[穴D] 出生届直後、複数の窓口をはしご（児童手当・健保・出生届確認）
  │        各課から別々の書類・説明を受ける
  │        → 親が疲弊し、指示を受けた手続きを後回しにする
  │        → 予防接種の案内を見落とす、健診の日程を忘れる
  │        → 乳児の健康管理の遅延
  │        → incident-catalog/newborn-procedure-fragmentation.md
  │
  └─[穴E] 伴走支援で特定妊婦と判定されても引き継ぎ先がない
           または引き継ぎ後も断片的な支援しかない
           → 特定妊婦が出産後に孤立する
           → 虐待相談機関への連携が遅れる
           → incident-catalog/high-risk-pregnant-handoff.md
```

## 担当者向け：出生届受付時のチェックフロー

```
出生届を受け取ったら、以下を順番に確認する

□ 1. 出生の事実確認（必須）
       ├─ 出生日（14日以内か確認）
       ├─ 保護者の確認
       └─ 医師の証明書添付の確認
       → workflows/_standardized-20/koseki/ 参照

□ 2. 世帯構成の確認（必須）
       ├─ 保護者が日本国籍か
       ├─ 社会保障受給中か（生活保護・児童扶養手当等）
       └─ 困窮の兆候がないか

□ 3. 児童手当申請の15日ルール説明（必須）
       ├─ 「支給開始月は申請日の翌月になる」と説明
       ├─ 「15日以内の申請で当月分からの支給が開始される」と説明
       ├─ 出生翌日（出生日+1日）から数えて15日以内であることを確認
       └─ 申請用紙を渡し、子育て支援課への導線を示す
       → incident-catalog/child-allowance-15day-window.md 参照

□ 4. 子の国民健康保険加入の確認（任意持参の場合）
       ├─ 親が社保加入か確認（社保加入なら不要）
       └─ 社保未加入 → 保険年金課への案内票を渡す

□ 5. 出生届に基づく伴走支援の初期通知（新規対象者向け）
       ├─ 保健センター訪問日時の案内
       ├─ 予防接種スケジュール票の配布
       └─ 乳児健診の予約案内
       → workflows/_standardized-20/boshi-techo/ 参照

□ 6. 関連手続き一覧の案内票を渡す（必須）
       ├─ 児童手当申請（期限：出生翌日から15日以内）
       ├─ 子の健保加入（期限：速やかに）
       ├─ 予防接種通知（自動送付される自治体が多い）
       └─ 乳児訪問の日程確認（自動案内または申込）
       ※ 案内票は自治体でカスタマイズ
```

## 自治体カスタマイズのポイント

このジャーニーファイルをForkして自治体版を作る際に、変更が必要な箇所。

| 項目 | 標準の記述 | 自治体が追記すること |
|---|---|---|
| 妊娠届の受付窓口 | 「保健センター」で統一 | 市役所本庁での受付実施の有無、時間 |
| 伴走型相談支援の担当者 | 「保健センター」と記述 | 専任者の有無、相談日時、個別面談の実施率目標 |
| 児童手当申請の同時受付 | 「出生届時の案内」と記述 | 同一窓口での申請受付の可否、分離している場合の動線 |
| 予防接種クーポンの送付 | 「乳児訪問時・健診時」 | 出生届後の自動送付タイミング、郵送の有無 |
| 乳児訪問の対象 | 「生後4週間以内推奨」 | 実施率目標、対象外となる条件（社保加入妊婦の乳幼児等） |
| 保育所利用申請の時期 | 「入所希望月の2〜3ヶ月前」 | 実際の当自治体の申込締切日 |
| 出生届と児童手当の書類 | 別様式と想定 | 統一様式がある場合の記述 |
| DV対応 | 記述なし | 特定妊婦・DV被害者への受付対応（個室・非通知等） |
