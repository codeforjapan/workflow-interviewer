# 高齢者・介護 住民ジャーニー

> **このファイルの役割**
> 住民が「65歳到達」から「おくやみ」までの行政接点の全体像を示す。
> 個々の業務フローは `workflows/` を参照。業務間の「穴」は `incident-catalog/` を参照。
> 自治体職員は**このファイルから読み始めることを推奨する**。

## ジャーニーの全体像

```
【住民の体験】
65歳到達
  │
  ├─ 介護保険第1号被保険者へ ────────────────────────────────────
  │   └─ 介護保険証の郵送（自動）
  │       関連: 高齢者向け福祉サービス案内
  │
  ├─ 75歳到達 ────────────────────────────────────────────────────
  │   ├─ 後期高齢者医療への自動移行
  │   ├─ 国民健康保険の脱退（自動）
  │   └─ 後期高齢者医療保険証交付（自動）
  │
  ├─ 状態変化（入院・転倒等がきっかけ） ──────────────────────────
  │   │
  │   └─ 要介護状態の発生
  │       └─ 介護が必要だと家族が認識
  │
  ├─ 介護認定申請フェーズ ──────────────────────────────────────
  │   ├─ 申請（本人・家族・地域包括支援センター経由）
  │   ├─ 認定調査（訪問で実施）
  │   ├─ 主治医意見書取得
  │   └─ 要介護認定審査会（判定）
  │
  ├─ ケアマネジメント開始 ──────────────────────────────────────
  │   ├─ ケアマネジャー選定・契約
  │   ├─ 居宅サービス計画（ケアプラン）作成
  │   └─ サービス利用開始（訪問介護・デイサービス等）
  │
  ├─ 継続的な支援 ────────────────────────────────────────────────
  │   ├─ 要介護度の変化時：区分変更申請
  │   ├─ 更新時期：更新申請（認定有効期間は3年以内）
  │   ├─ 月ごと：利用状況確認・ケアプラン見直し
  │   └─ 高額療養費：限度額適用認定証申請（後期高齢者医療）
  │
  ├─ 施設入所検討フェーズ ──────────────────────────────────────
  │   ├─ 在宅介護の継続が困難に
  │   ├─ 特別養護老人ホーム等の申込
  │   └─ 入所調整
  │
  └─ おくやみへの移行 ────────────────────────────────────────────
      ├─ 介護保険脱退
      ├─ 後期高齢者医療脱退
      └─ 遺族年金・葬祭費等の手続き
```

## 業務連鎖マトリクス

高齢化に伴う行政接点の**何が・いつ・誰に必要か**を示す。

| 手続き | 対象者 | タイミング | 窓口 | 案内の責任 |
|---|---|---|---|---|
| 介護保険証送付 | 65歳到達者 | 65歳誕生日前後（自動） | 介護保険課 | 介護保険課 |
| 高齢者向け福祉案内 | 65歳到達者 | 同上 | 介護保険課・地域包括支援センター | 介護保険課 |
| 後期高齢者医療への移行 | 75歳到達者 | 75歳誕生日前後（自動） | 保険年金課 | 保険年金課 |
| 国民健康保険脱退 | 75歳到達者で国保加入 | 同上 | 保険年金課 | 保険年金課 |
| 後期高齢者医療保険証交付 | 75歳到達者 | 75歳誕生日前後 | 保険年金課 | 保険年金課 |
| 介護認定申請 | 要介護が必要な65歳以上 | 状態変化時（本人・家族が判断） | 介護保険課・地域包括支援センター | 地域包括支援センター |
| 認定調査（訪問） | 介護認定申請者 | 申請後30日以内 | 介護保険課調査員 | 介護保険課 |
| 主治医意見書取得 | 介護認定申請者 | 申請と並行 | 医療機関 | 本人・家族（依頼） |
| 要介護認定審査会 | 介護認定申請者 | 調査後、30日以内（合計60日以内） | 介護保険課 | 介護保険課 |
| ケアマネジャー選定 | 要介護認定者 | 認定結果通知後 | 地域包括支援センター・ケアマネ事業所 | 介護保険課・地域包括支援センター |
| 居宅サービス計画作成 | 要介護認定者 | 認定後、サービス利用前 | ケアマネジャー | ケアマネジャー |
| サービス利用開始 | 要介護認定者 | ケアプラン作成後 | 訪問介護事業所等 | ケアマネジャー |
| 区分変更申請 | 要介護度変化が生じた者 | 状態変化時（随時） | 介護保険課・地域包括支援センター | 本人・家族・ケアマネジャー |
| 更新申請 | 要介護認定者 | 認定有効期間終了の60日前から | 介護保険課・地域包括支援センター | 介護保険課（通知） |
| 限度額適用認定証申請 | 後期高齢者医療保険加入者 | 高額療養費該当見込時 | 保険年金課 | 本人・医療機関 |
| 転入時：介護認定引き継ぎ | 転入時に要介護認定者 | **転入日から14日以内**（切れると失効） | 介護保険課 | 転入担当者（市民課） |
| 介護保険脱退 | 要介護認定者が転出・死亡時 | 転出届・死亡届の届出時 | 介護保険課 | 転出・福祉課等 |

**→ 「介護保険課」「地域包括支援センター」「転入担当者」の連携が，全国共通の「穴」の根源。**

## 穴の連鎖マップ

この業務ジャーニーで発生しうるインシデントを，発生箇所と連鎖先で示す。

```
介護保険第1号被保険者到達（65歳）
  │
  ├─[穴A] 75歳誕生日前後の保険証の「切れ目」問題
  │        → 旧国保証の有効期限切れ〜後期高齢者医療証到着の間に医療機関受診
  │        → 資格確認が取れず10割負担，その後返納手続きで混乱
  │        → incident-catalog/healthcare-cert-gap.md
  │
  ├─[穴B] 要介護になっても「申請しなければサービスが始まらない」を家族が知らない
  │        → 状態変化しても申請を放置 → 自費で介護を継続 → 家族疲弊・虐待へ
  │        → 地域包括支援センターの訪問・啓発が不足
  │        → incident-catalog/care-application-unknowing.md
  │
  ├─[穴C] 転入時の介護認定引き継ぎ14日ルールを担当者が案内しない
  │        → 転入届受付時に要介護者と気づかない，または気づいても案内しない
  │        → 期限切れ → 認定失効 → サービス停止 → 一からやり直し
  │        → 福祉課と市民課の連携欠落
  │        → incident-catalog/care-handover-14days.md
  │
  ├─[穴D] 介護者（配偶者・子ども）の疲弊が見えず，虐待・老老介護の危機を見逃す
  │        → ケアマネジャーと地域包括支援センターの定期訪問が形式的
  │        → 要介護度判定のみで「支える側」の状態まで見ない
  │        → 孤立した介護環境が放置される
  │        → incident-catalog/caregiver-burnout.md
  │
  └─[穴E] 複合的なニーズ（困窮×介護，障害×介護，ヤングケアラー等）への気づきが遅い
           → 福祉課・障害者支援課・子育て支援課との情報共有がない
           → 個別対応のみで，制度横断的な支援が機能しない
           → incident-catalog/complex-social-needs.md
```

## 担当者向け：要介護認定の「入り口」チェックフロー

### A. 転入受付時に要介護者がいると判明した場合

```
転入届を受け取ったら、以下を順番に確認する

□ 1. 世帯構成確認時に65歳以上の者を把握
       ├─ 現在介護保険証を持っているか聞く
       └─ 「要介護」「要支援」の認定を受けているか聞く

□ 2. 要介護認定者と判明した場合
       ├─ 「転入日から14日以内に『介護認定引き継ぎ申請』が必要」と説明
       ├─ 紙（案内票）で渡す
       └─ 介護保険課への連絡票を起こす（その場で電話連絡も推奨）

□ 3. 要介護者の介護保険証・受給者証・認定決定通知書をコピー取得推奨
       → 介護保険課への引き継ぎ情報として活用

□ 4. 関連手続き一覧の案内票を渡す（必須）
       ※ 案内票は自治体でカスタマイズ
```

### B. 地域包括支援センターが「申請検討」を促す場面

```
地域包括支援センター職員が転倒・入院等で状態変化した高齢者を把握した場合

□ 1. 本人・家族に対する申請勧奨の声掛けをする
       「このような状態では、介護保険の『要介護認定』を受けるといいですよ」

□ 2. 申請書類の取得方法を複数案内
       ├─ 介護保険課に取りに行く（市役所）
       ├─ 地域包括支援センター経由で代行申請
       ├─ 郵送請求（自治体による）
       └─ オンライン申請（自治体による）

□ 3. 主治医情報を確認
       「かかりつけの病院はありますか？」
       → 医者がいない場合は地域包括がサポート

□ 4. 面談・訪問で要介護度の変化を継続モニタリング
       （申請→認定までの30日待機期間中に進行することもある）
```

## 自治体カスタマイズのポイント

このジャーニーファイルをForkして自治体版を作る際に，変更が必要な箇所。

| 項目 | 標準の記述 | 自治体が追記すること |
|---|---|---|
| 地域包括支援センターの担当圏域 | 記述なし | 地域圏ごと（中央地域、西部地域等）、所在地、連絡先 |
| ケアマネジャー紹介の仕組み | 地域包括がマッチング推奨 | 自治体の紹介リスト、優先紹介事業所、料金体系 |
| サービス事業所の周知方法 | 地域包括からの案内 | 自治体ホームページのリンク、パンフレット所在地 |
| 限度額適用認定証の申請窓口 | 保険年金課 | 郵送申請の可否、オンライン申請の有無 |
| 更新申請の通知方法 | 前年度の書類に記載 | 通知タイミング（有効期限の60日前）、対象者抽出リスト |
| 転入時の「14日ルール」案内票 | テンプレート参照 | 自治体の実際の様式、多言語対応の有無 |
| 要介護度判定の異議申し立て | 認定結果の受け取り日から60日 | 自治体の異議申し立て受付窓口、手数料の有無 |
| 施設入所時の対応 | ケアマネが調整 | 自治体が紹介する施設リスト、入所判定会議の有無 |
| おくやみ対応との連携 | 福祉課が引き継ぎ | 死亡届受付時の介護保険脱退自動処理、遺族への説明順序 |
