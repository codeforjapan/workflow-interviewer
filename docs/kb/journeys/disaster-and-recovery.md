# 被災・生活再建 住民ジャーニー

> **このファイルの役割**
> 「被災（台風・地震・洪水・火災等）」から「生活再建」に至るまで住民がたどる行政接点の全体像を示す。
> 被災ジャーニーは他のライフイベントジャーニーと根本的に異なる点がある：
> **通常の窓口機能が損なわれた状態で、多数の住民が一斉に手続きを求める。**
> 個々の業務フローは `workflows/` を参照。業務間の「穴」は `incident-catalog/` を参照。

---

## ジャーニー全体像（4フェーズ）

```
【フェーズ1: 緊急避難（発災〜72時間）】

災害発生（地震・洪水・台風等）
  │
  ├─ 避難指示・勧告の発令
  │   発令主体: 市区町村長（気象庁・国土交通省情報に基づく）
  │
  ├─ 避難行動
  │   ├─ 自力避難可能な住民 → 指定緊急避難場所・指定避難所へ
  │   ├─ 要配慮者（高齢者・障害者等）→ 個別避難計画に基づく支援付き避難
  │   │   ※ 計画が未整備の場合: 民生委員・自主防災組織が独自判断で支援
  │   └─ 自主避難者 → 自主避難所（法定外）へ
  │
  └─ 安否確認・行方不明者把握
      ├─ 避難者名簿の作成（避難所単位）
      ├─ 行方不明者情報の収集（警察・消防・自衛隊と連携）
      └─ 要支援者の安否確認（避難行動要支援者名簿の活用）
           ⚠️ 名簿が関係機関に共有されていない場合、機能しない → INC-D01参照

【フェーズ2: 応急対応（発災後〜1〜2週間）】

避難所での生活開始
  │
  ├─ 罹災証明書の申請受付開始
  │   │（市区町村が申請窓口・被害認定調査を実施）
  │   │ ⚠️ 調査員不足で発行に数週間〜数か月かかる場合あり
  │   │ ⚠️ 一部損壊では多くの支援制度を利用できない → 住民が混乱
  │   └─ 罹災証明書を受け取る → 各支援制度への申請へ
  │
  ├─ 応急修理制度の申請（半壊・準半壊）
  │   申請先: 市区町村
  │   上限: 約25.7万円（国費補助）
  │
  ├─ 福祉避難所への移送（要配慮者）
  │   ⚠️ 福祉避難所の場所・受け入れ条件を知らない住民・職員が多い → INC-D02参照
  │
  ├─ 被災者への情報提供（支援メニューの周知）
  │   問題: 情報が避難所単位でバラバラ・担当課が多すぎる
  │   → 「罹災証明書を取ったら次は何をすれば？」をまとめた案内が必要
  │
  └─ 仮設住宅・みなし仮設の申請受付
      申請先: 都道府県・市区町村
      ⚠️ 全壊・大規模半壊が優先。半壊以下は対象外の場合あり

【フェーズ3: 生活再建支援（発災後〜数か月）】

罹災証明書取得後
  │
  ├─ 被災者生活再建支援金の申請（全壊・大規模半壊・中規模半壊）
  │   申請先: 都道府県（市区町村経由）
  │   支給額: 基礎支援金（最大100万）+ 加算支援金（最大200万）
  │   ⚠️ 申請期限あり（発災から13か月以内が多い）
  │
  ├─ 義援金の受け取り
  │   受け取り先: 市区町村（配分委員会の指示に基づく）
  │   ⚠️ 配分が完了するまで数か月〜1年以上かかる場合あり
  │   ⚠️ 被災者生活再建支援金と別制度（両方申請が必要）
  │
  ├─ 災害弔慰金の申請（死亡者の遺族）
  │   申請先: 市区町村（福祉課等）
  │   支給額: 最大500万円（生計維持者の死亡）
  │
  ├─ 各種税・保険料の減免申請
  │   ├─ 固定資産税の減免（税務課）
  │   ├─ 国民健康保険料の減免（保険課）
  │   └─ 個人住民税の減免（税務課）
  │   ⚠️ 申請しないと減免されない制度が多い
  │
  ├─ 災害援護資金・生活福祉資金の申請（貸付）
  │   申請先: 市区町村（援護資金）・社協（生活福祉資金）
  │   ⚠️ 「貸付＝返済必要」であることを必ず説明
  │
  └─ 被災者台帳（市区町村が作成）への登録確認
      → 支援の進捗管理・漏れ防止のための管理台帳
      ⚠️ 台帳未整備の場合、支援の重複・漏れが発生

【フェーズ4: 長期復興（発災後〜数年）】

  ├─ 住宅再建（自力再建・補助利用）
  │   ├─ 被災者生活再建支援金（加算支援金: 住宅再建方法に応じて支給）
  │   ├─ 民間保険（地震保険・火災保険）の保険金請求
  │   │   ⚠️ 行政の被害認定と保険査定は独立（区分が異なる場合あり）
  │   └─ 住宅ローン（既存ローン＋再建ローンの二重負担問題）
  │       → 自然災害債務整理ガイドライン（破産回避のための特別手続き）
  │
  ├─ 仮設住宅からの退去・住民票の移動
  │   ├─ 新住所への転入届（住民記録課）
  │   ├─ 国保・介護保険等の住所変更
  │   └─ みなし仮設からの退去調整（賃貸期間延長交渉等）
  │
  ├─ 精神的健康の問題への対応
  │   PTSD・孤独死・アルコール問題・コミュニティ崩壊
  │   → 保健センター・心理専門職・社協による継続支援
  │
  └─ コミュニティ再建
      ├─ 自治会・町内会の再編
      └─ 仮設住宅コミュニティから恒久住宅への移行支援
```

## 手続き連鎖マトリクス

| 手続き | 対象者 | 期限 | 窓口 | 案内責任 |
|---|---|---|---|---|
| 避難先の登録 | 避難した全住民 | 速やかに | 避難所・避難者名簿 | 避難所担当職員 |
| 罹災証明書の申請 | 住家被害のある住民 | 受付開始次第（期限あり） | 市区町村窓口・郵送 | 市区町村（広報・避難所案内） |
| 被害認定不服申立（第2次調査申請） | 認定区分に異議のある住民 | 罹災証明書受領後の一定期間 | 市区町村担当課 | 罹災証明書交付時に説明 |
| 応急修理制度の申請 | 半壊・準半壊世帯 | 着工前（修繕開始後は対象外） | 市区町村建設・住宅課 | 罹災証明書交付時に案内 |
| 被災者生活再建支援金の申請 | 全壊・大規模半壊等世帯 | 発災から13か月以内が多い | 都道府県（市区町村経由） | 市区町村・支援窓口 |
| 義援金の受取 | 被災者（罹災証明書保有者） | 配分委員会決定後 | 市区町村指定窓口・口座振込 | 市区町村 |
| 災害弔慰金の申請 | 死亡した被災者の遺族 | 発災から3年以内が多い | 市区町村福祉課 | 市区町村（おくやみ対応時） |
| 国保・介護保険料の減免申請 | 被災した被保険者 | 各制度の定める期限 | 市区町村各担当課 | 担当課からの周知 |
| 固定資産税の減免申請 | 被災した不動産所有者 | 年度内に申請が必要なことが多い | 税務課 | 税務課からの周知 |
| 仮設住宅の入居申請 | 全壊・大規模半壊等世帯 | 公募期間内 | 都道府県・市区町村 | 市区町村・避難所 |
| 住民票の移動（避難先・仮設住宅） | 避難長期化住民 | 実態の住所に合わせて | 住民記録課 | 住民自身（任意） |

## 穴の連鎖マップ

```
被災
  │
  ├─【穴1】要配慮者の避難支援が機能しない
  │         個別避難計画未整備・名簿が関係機関に共有されていない
  │         → incident-catalog/INC-D01-vulnerable-person-list-failure.md
  │
  ├─【穴2】福祉避難所の情報が住民・担当職員に届いていない
  │         「直接来ないで」ルールが機能不全を引き起こす
  │         → incident-catalog/INC-D02-welfare-shelter-unknown.md
  │
  ├─【穴3】被害認定区分への不服が申立制度を知らないまま放置
  │         一部損壊 → 支援制度対象外で生活再建できない
  │         → incident-catalog/INC-D03-disaster-certificate-dispute.md
  │
  ├─【穴4】「申請しなければもらえない」支援制度の案内が届かない
  │         - 被災者生活再建支援金（期限あり）
  │         - 国保・介護保険料の減免（申請必要）
  │         - 固定資産税の減免（申請必要）
  │
  └─【穴5】義援金と被災者生活再建支援金を同一視して片方しか申請しない
            財源・申請先・支給時期がまったく異なる
            → concepts/disaster-relief-payment.md 参照
```

## 担当者向け: 避難所窓口チェックリスト（初動期）

```
□ 1. 避難者名簿を作成し、氏名・住所・世帯人数・要配慮者フラグを記録
□ 2. 要配慮者（車いす・人工透析・医療的ケア等）を把握し、福祉避難所への移送を検討
□ 3. 避難行動要支援者名簿と照合し、安否不明者を防災担当に報告
□ 4. 罹災証明書申請受付の開始予定を周知する
□ 5. 「何の支援が受けられるか」ワンペーパーの案内を配布する
□ 6. 住民票の現住所と避難先の乖離を記録し、郵便物転送・各種申請の対応を案内
□ 7. 車中泊避難者の把握（駐車場等を巡回し、支援物資・情報を届ける）
□ 8. 要配慮者の医療・介護サービスの継続確認（主治医・ケアマネへの連絡）
```

---

## 担当者向け: 生活再建窓口チェックリスト（応急〜復興期）

```
□ 1. 罹災証明書交付時に「次のステップ」（支援金申請・保険料減免等）を一覧で案内
□ 2. 不服申立（第2次調査）の方法と期限を交付時に説明
□ 3. 「全壊/大規模半壊」→ 被災者生活再建支援金の申請案内（期限確認）
□ 4. 「半壊/準半壊」→ 応急修理制度・住宅ローン特例等の案内
□ 5. 死亡被災者がいる場合 → 災害弔慰金の案内（おくやみジャーニーと連動）
□ 6. 国保・介護保険料・固定資産税減免の申請手続きを各課に依頼・周知
□ 7. 社協の生活困窮相談窓口（生活福祉資金・災害VC）への誘導
□ 8. 長期化する場合: 精神的支援（DPAT・保健師訪問）への接続
```
