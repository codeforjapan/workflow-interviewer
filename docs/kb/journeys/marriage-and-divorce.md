# 結婚・離婚 住民ジャーニー

> **このファイルの役割**
> 住民が「結婚」「離婚」という2つの出来事でたどる行政接点の全体像を示す。
> 個々の業務フローは `workflows/` を参照。業務間の「穴」は `incident-catalog/` を参照。
> 自治体職員は**このファイルから読み始めることを推奨する**。

---

## ジャーニーの全体像

```
【住民の体験 - フェーズ1：婚姻】

婚姻決定
  │
  ├─ 婚姻届受付 ──────────────────────────────────────────────
  │   └─ 婚姻届（当事者または代理人が提出）
  │       関連: 夜間・休日でも受け付け可能な自治体多数
  │
  ├─ 婚姻後・14日以内 ────────────────────────────────────────
  │   ├─ 氏・本籍の変更（戸籍に記載）
  │   │   ├─ 結婚に伴って氏を変える場合 → 戸籍変更手続き
  │   │   └─ 本籍地の変更 → 婚姻届で指定可能
  │   │
  │   └─ 転居がない場合でも行う手続き
  │       ├─ マイナンバーカード記載変更（氏変更した場合）
  │       ├─ 国民年金の種別変更
  │       │   └─ 配偶者が会社員 → 第3号被保険者への切替
  │       ├─ 健康保険の切替・扶養追加
  │       │   ├─ 配偶者が社保加入 → 扶養手続き（会社経由）
  │       │   ├─ 配偶者が個人事業主 → 国保加入手続き
  │       │   └─ 自分が社保加入 → 配偶者の国保脱退手続き
  │       └─ 印鑑登録の変更（氏変更した場合）
  │
  └─ 転居が伴う場合
      └─ moving.md のジャーニーと併行（転入届・転出届も必須）

─────────────────────────────────────────────────────────────────

【住民の体験 - フェーズ2：離婚】

離婚決定
  │
  ├─ 離婚成立 ────────────────────────────────────────────────
  │   ├─ 協議離婚：夫婦の合意で離婚届を提出
  │   ├─ 調停離婚：家庭裁判所の調停を経て離婚届を提出
  │   └─ 審判離婚：家庭裁判所の審判を経て離婚届を提出
  │
  ├─ 離婚届受付・14日以内 ────────────────────────────────────
  │   ├─ 離婚届（当事者または代理人が提出）
  │   │
  │   └─ 離婚届が起点となって連鎖する手続き
  │       ├─ 氏・本籍の変更（戸籍に記載）
  │       │   ├─ 婚姻時に氏を変えた方が復氏する場合
  │       │   ├─ 旧姓に戻すか別の本籍を選択
  │       │   └─ 離婚後3ヶ月以内なら婚氏続称の届出も可能
  │       │
  │       ├─ 転居・住民票の分離（転居する場合）
  │       │   └─ moving.md のジャーニーも併行
  │       │
  │       ├─ 国民年金の種別変更
  │       │   └─ 第3号被保険者 → 第1号被保険者への切替
  │       │       （または配偶者の扶養から外れて社保加入）
  │       │
  │       ├─ 国民健康保険加入
  │       │   └─ 配偶者の扶養から外れた場合
  │       │
  │       ├─ 児童手当の受給者変更（親権者へ）
  │       │   ├─ 子がいる場合、親権者が変更になったら必須
  │       │   └─ 支給停止期間が生じないよう注意
  │       │
  │       ├─ マイナンバーカード記載変更（氏変更した場合）
  │       │
  │       └─ 子の親権・戸籍変更（親権者以外の場合）
  │           └─ 家庭裁判所での手続きが別途必要
  │               （市区町村ではできない）
  │
  └─ 離婚後・随時 ────────────────────────────────────────────
      ├─ 印鑑登録の変更・新規登録（氏変更した場合）
      ├─ 運転免許証の氏変更（警察署・免許センター）
      └─ 各種保険料の手続き（生命保険等、民間手続き）
```

## 業務連鎖マトリクス

結婚・離婚をきっかけに、**何が・いつ・誰に必要か**を示す。

### 婚姻時の手続き

| 連鎖する手続き | 対象者 | 期限 | 窓口 | 案内の責任 |
|---|---|---|---|---|
| 婚姻届 | 結婚当事者 | 制限なし（日時選択可） | 戸籍・住民票課 | 住民の申請 |
| 住民票の世帯変更 | 婚姻届出者 | 婚姻届受理後自動 | 戸籍・住民票課 | 自動処理 |
| マイナカード記載変更 | カード所持者で氏変更 | **転入日から14日以内** | 市民課 | 婚姻担当者 |
| 国民年金 第3号への切替 | 配偶者が会社員の場合 | **14日以内（年金手帳持参）** | 保険年金課 | 婚姻担当者 |
| 国民健康保険 脱退 | 新たに被扶養者になる場合 | **14日以内** | 保険年金課 | 婚姻担当者 |
| 健康保険 扶養追加 | 配偶者が社保加入 | 速やかに | 配偶者の勤務先 | 婚姻担当者（案内） |

### 離婚時の手続き

| 連鎖する手続き | 対象者 | 期限 | 窓口 | 案内の責任 |
|---|---|---|---|---|
| 離婚届 | 離婚当事者 | 制限なし（日時選択可） | 戸籍・住民票課 | 住民の申請 |
| 戸籍の分籍・氏変更 | 婚氏を復氏する側 | 離婚後3ヶ月以内（婚氏続称選択可） | 戸籍・住民票課 | 自動処理（届け出後） |
| 住民票の分離 | 転居する側 | **14日以内** | 戸籍・住民票課 | 住民の申請 |
| マイナカード記載変更 | カード所持者で氏変更 | **14日以内** | 市民課 | 離婚担当者 |
| 国民年金 第1号加入 | 第3号から切替必要者 | **14日以内（年金手帳持参）** | 保険年金課 | 離婚担当者 |
| 国民健康保険 加入 | 被扶養者から外れる場合 | **14日以内** | 保険年金課 | 離婚担当者 |
| 児童手当 受給者変更 | 親権が変わった場合 | **15日以内（支給月に影響）** | 子育て支援課 | 離婚担当者 |
| 子の氏変更（家庭裁判所） | 親権者以外の親に引き取られた場合 | 親権決定後 | 家庭裁判所（家裁） | 離婚担当者（案内のみ） |

**→ 「婚姻・離婚担当者」が案内の責任を持つ手続きが計10〜11個。**
これを担当者の個人知識に任せていることが、全国共通の「穴」の根源。

## 穴の連鎖マップ

この業務ジャーニーで発生しうるインシデントを、発生箇所と連鎖先で示す。

```
婚姻届受付
  │
  ├─[穴A] 国民年金第3号への切替案内が漏れる
  │        → 届け出ないまま時効（2年）を過ぎる
  │        → 遺族年金受給権が失われる可能性
  │        → 将来の年金額に影響
  │        → incident-catalog/pension-3rd-notification-gap.md
  │
  └─[穴B] 配偶者の健康保険切替案内が不十分
           → 国保喪失手続きを忘れたまま重複加入
           → 後年、医療費返還請求のトラブル
           → incident-catalog/duplicate-insurance.md

─────────────────────────────────────────────────────────────────

離婚届受付
  │
  ├─[穴C] 国民年金・国保加入を知らず無保険状態になる
  │        → 病気・怪我で高額医療費の負担
  │        → 年金加入記録の空白が将来に影響
  │        → incident-catalog/post-divorce-uninsured.md
  │
  ├─[穴D] 子の親権・戸籍変更が家庭裁判所の手続きだと知らない
  │        → 市区町村窓口で「ここではできない」だけで終わる
  │        → 子どもの戸籍が変わらず新住所に移せない
  │        → 学校転校・各種申請で証明書が不備
  │        → incident-catalog/child-name-change-guidance.md
  │
  ├─[穴E] DV被害者が離婚届を提出した際に加害者への情報漏洩
  │        → 住所・転居先が加害者に判明（住民票異動で）
  │        → 児童手当の給付通知が新住所で郵送される
  │        → 扶養照会で加害者に住所が特定される
  │        → incident-catalog/dv-post-divorce-leakage.md
  │
  └─[穴F] 親権変更に伴う児童手当の受給者切替が遅延
           → 支給月が1ヶ月以上空く
           → 新親権者が生計を担うのに給付がない期間が発生
           → incident-catalog/child-allowance-handover-gap.md
```

## 担当者向け：離婚届受付時のチェックフロー

```
離婚届を受け取ったら、以下を順番に確認する

□ 1. 本人確認・書類の完全性（必須）
       ├─ 協議離婚：夫婦署名・押印・証人2名
       ├─ 調停・審判：調停調書または審判書を確認
       └─ workflows/_standardized-20/jyumin-ido/ 参照

□ 2. DV支援措置の確認（最優先）
       システムでフラグを確認。フラグなしでも申告があれば係長へ即エスカレーション
       ├─ DV加害者が住所を知らないようにする必要がある
       ├─ 住民票の閲覧制限をかける
       ├─ 子育て支援課・福祉課への情報遮断
       └─ incident-catalog/dv-post-divorce-leakage.md 参照

□ 3. 世帯構成・子どもの有無を確認（必須）
       ├─ 子どもがいるか
       │   ├─ YES → 親権者は誰か
       │   │         ├─ 親権変更がある場合：子育て支援課に児童手当受給者変更を案内
       │   │         │   └─ **15日ルール説明**（支給月に影響）
       │   │         └─ 親権者以外が子を引き取る場合：家庭裁判所での氏変更手続きを案内
       │   │
       │   └─ NO → 手続きスキップ
       │
       └─ 特に子がいて転居する場合 → moving.md も併行

□ 4. 婚姻時に氏を変えたか確認（必須）
       ├─ YES → 復氏するか / 婚氏続称するか選択を確認
       │         └─ マイナカード記載変更を案内（14日ルール）
       │
       └─ NO → 戸籍記載のみ

□ 5. 転居の有無を確認（必須）
       ├─ YES → moving.md のチェックフロー も実施
       └─ NO → 以下に続く

□ 6. 健康保険・年金の状況を確認（必須）
       ├─ 現在、国民年金の第3号被保険者か
       │   YES → 第1号加入または配偶者の扶養から外す案内（14日ルール）
       │
       ├─ 現在、配偶者の健康保険被扶養者か
       │   YES → 国保加入手続きを案内（14日ルール）
       │
       └─ 現在、国保加入者か
           └─ 配偶者が国保から脱退することを案内

□ 7. 関連手続き一覧の案内票を渡す（必須）
       ※ 案内票は各自治体でカスタマイズ
       ※ DV事案の場合は特別な案内票を用意
```

## 自治体カスタマイズのポイント

このジャーニーファイルをForkして自治体版を作る際に、変更が必要な箇所。

| 項目 | 標準の記述 | 自治体が追記すること |
|---|---|---|
| 婚姻・離婚届の受付時間 | 制限なし | 休日・夜間届出の対応可否 |
| DV対応の個室 | 推奨と記述 | 個室の場所・代替手段・職員の訓練状況 |
| 年金事務所への連携 | 日次バッチ（一般的） | リアルタイム連携の有無・タイムラグ |
| 児童手当の15日ルール | 記述のみ | 前月割れ防止の自治体独自運用（ある場合） |
| 家庭裁判所への案内資料 | テンプレート参照 | 子の氏変更手続き（家裁）の詳細案内を添付 |
| 子育て支援課との連携 | 日次バッチ（一般的） | リアルタイム通知の有無・システム連携状況 |
| 案内票の様式 | テンプレート参照 | 自治体の実際の様式を添付 |
| エスカレーション先 | Level 1〜4で記述 | 担当者名・内線番号・DV対応責任者 |
