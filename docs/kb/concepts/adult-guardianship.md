# 概念競合（役割型）: 成年後見

## 概要

「成年後見人がいる」という情報だけでは、その人が何をどこまで代理できるかがわからない。**法定後見には後見・保佐・補助の3類型があり、代理権と同意権の範囲が全く異なる**。また任意後見は家庭裁判所の審判前であれば効力がなく、窓口では混乱が生じやすい。自治体の業務フローの多くが「成年後見人が来た場合」の扱いを曖昧にしたまま運用されている。

---

## 法定後見の3類型

```
判断能力の程度（低い ←──────────── 高い）
  │
  ├─【後見】 判断能力を「欠く常況」
  │   └─ 後見人が原則として全ての法律行為を代理
  │       本人は一定の日用品購入等以外の法律行為を単独でできない
  │
  ├─【保佐】 判断能力が「著しく不十分」
  │   └─ 保佐人が「重要な財産行為」（民法第13条）に同意または代理
  │       本人は日常的な行為は単独でできる
  │       代理権は審判で個別に付与（付与されない場合も）
  │
  └─【補助】 判断能力が「不十分」
      └─ 補助人が特定の行為（審判で個別に決定）のみ同意または代理
          本人の行為能力は最も広く保たれる
          本人の同意なしに補助審判の申立ては不可
```

| 類型 | 申立て要件 | 代理権 | 同意権 | 取消権 |
|---|---|---|---|---|
| **後見** | 精神上の障害で判断能力欠く常況 | 全般（家庭裁判所が付与） | 日用品等以外は本人行為を取消可 | ○（後見人が取消可） |
| **保佐** | 精神上の障害で判断能力著しく不十分 | 審判で個別に付与された範囲 | 民法第13条の重要行為 | ○（同意なき行為を取消可） |
| **補助** | 精神上の障害で判断能力不十分 | 審判で個別に付与された範囲 | 審判で個別に付与された範囲 | ○（同意なき行為を取消可） |

---

## 任意後見との違い

```
【法定後見】
  → 家庭裁判所が職権で後見人を選任
  → 本人の判断能力が失われた「後」に申立て
  → 本人の意思は反映されにくい

【任意後見】
  → 本人が判断能力があるうちに、将来の代理人を自分で選んで契約
  → 家庭裁判所による「任意後見監督人」の選任後に効力発生
  → 任意後見監督人が選任されるまでは効力なし（← 窓口での混乱原因）
```

⚠️ **「任意後見契約書がある」だけでは後見人として認められない。**
任意後見監督人選任の審判書の写しがあって初めて効力が発生する。

## 各行政手続きにおける後見人の扱い

| 手続き | 後見類型 | 後見人の関与範囲 | 注意点 |
|---|---|---|---|
| **介護保険認定申請** | 後見・保佐・補助いずれでも可 | 後見人が申請書に署名・代理申請可 | 認定結果の通知先を後見人に変更できる |
| **生活保護申請** | 後見類型で異なる | 後見人が申請書に署名可（後見）。保佐・補助は本人意思確認優先 | 保護決定通知・保護金品は本人名義が原則。後見人の口座への振込は要確認 |
| **住民票の届出** | 後見人が届出義務者になる場合あり | 後見人が転入・転出届を代理 | 支援措置（DV等）との競合に注意 |
| **障害者手帳の申請** | いずれでも可 | 後見人が代理申請可 | 手帳の交付は本人名義 |
| **施設入所契約** | 後見・保佐 | 後見人が契約当事者になることが多い | 施設側が「後見人の代理権確認書類」を求める |
| **選挙権の行使** | 後見であっても代理不可 | 本人のみ（一身専属的） | 後見人は代理投票不可 |
| **遺言の作成** | 後見であっても代理不可 | 医師2名の立会い等の手続きにより本人が作成可 | 後見人が遺言内容を決定する権限はない |

## 市区町村長申立て

本人・家族等による申立てが期待できない場合、市区町村長が後見等の審判を家庭裁判所に申立てることができる。

| 根拠法令 | 対象 |
|---|---|
| 老人福祉法第32条 | 65歳以上の高齢者 |
| 知的障害者福祉法第28条 | 知的障害者 |
| 精神保健福祉法第51条の11の2 | 精神障害者 |

**実務上の課題**: 申立て判断の基準が自治体ごとに異なり、「申立てを誰が判断するか」が曖昧なまま事例が放置されることがある（→ chiiki-houkatsu-shien-center.md のギャップ参照）。

---

## 成年後見制度利用促進法（平成28年）

国・地方公共団体・関係機関が連携して成年後見制度の利用を促進するための法律。
市区町村は「成年後見制度利用促進基本計画」に基づき、中核機関の設置・権利擁護支援ネットワークの構築が求められている。

## AIへの注意事項

⚠️ **AIは「成年後見人がいる」という情報から自動的に全ての手続きを後見人が代理できると判断してはならない。**

理由:
1. 後見・保佐・補助の3類型により代理権の範囲が異なる
2. 任意後見は監督人選任前は効力なし
3. 選挙権・遺言など一身専属的な行為は後見人でも代理不可
4. 生活保護の保護金品の受取口座など、制度ごとに細かい運用ルールがある

後見人の代理権確認は、審判書謄本（法定後見）または任意後見監督人選任審判書謄本（任意後見）を必ず確認すること。
