# 概念競合（役割型）: ヤングケアラー

## 概要

「ヤングケアラー」は**法律上の定義を持たない概念**でありながら、教育・福祉・保健の複数の制度にまたがって使われる。本来大人が担うべき家族のケア（介護・育児・家事・感情サポート等）を担っている18歳未満の子どもを指すが、「誰がこの子を把握・支援する責任を持つか」が制度上明確ではないため、**全ての機関が「うちの所管外」として見逃す**リスクが高い。

---

## ヤングケアラーの状態モデル

```
ヤングケアラーが置かれる状況
  │
  ├─【学校での見え方】
  │   遅刻・欠席の増加 / 授業中の居眠り / 宿題未提出
  │   → 学校は「生活習慣の問題」と判断しがち
  │   → 不登校統計に吸収され、背景が見えない
  │
  ├─【介護・福祉側での見え方】
  │   要介護高齢者・障害者のケアプランに「家族が担う部分」として組み込まれている
  │   → ケアマネ・相談支援専門員は「家族支援を受けている」と記録
  │   → 子どもがケアを担っているとは認識されない
  │
  ├─【子ども家庭福祉側での見え方】
  │   育児放棄（ネグレクト）・過度な家事分担との境界が曖昧
  │   → 虐待通告の対象になる場合とならない場合がある
  │   → 「本人が嫌がっていない」と児相が判断して対応しないことがある
  │
  └─【貧困・生活保護側での見え方】
      困窮世帯で親の代わりに家事・弟妹の世話をしている
      → ケースワーカーは世帯の生活状況を見るが、子どもの負担に焦点が当たりにくい
```

## 年齢による使える制度の変化

| 年齢 | 適用される主な制度 | ケアの責任者 |
|---|---|---|
| **18歳未満** | 子ども・子育て支援、児童福祉法、就学援助 | 親権者（保護者） |
| **18歳〜20代** | 介護保険の家族支援、障害者支援、若者就労支援 | 本人（成人）|

18歳の誕生日を境に「子ども」から「成人」に変わり、使える制度が一変する。**ヤングケアラー支援の多くは18歳未満を想定しており、18歳以上の若者ケアラーは制度の谷間に落ちやすい**。

## 「発見者」と「支援責任者」のズレ

| 発見する機関 | 発見の契機 | 支援責任者（本来） | ギャップ |
|---|---|---|---|
| 学校・担任 | 欠席増加、居眠り、宿題未提出 | 教育委員会・スクールソーシャルワーカー | 「家庭の問題」として担任の個人対応に留まる |
| 民生委員・児童委員 | 見守り訪問、地域での観察 | 市区町村の子ども福祉担当 | 「どこに連絡すればよいか」わからない |
| ケアマネジャー | 高齢者訪問時に子どもが家事担当と把握 | 市区町村・地域包括支援センター | ケアプランの範囲外として情報が止まる |
| 相談支援専門員 | 障害者支援計画策定時に家族の状況把握 | 市区町村の障害福祉担当 | 「子どもの問題は教育委員会へ」と振られる |
| 保健師 | 乳幼児健診、家庭訪問で把握 | 市区町村の健康福祉担当 | 情報共有の仕組みがない |

## 「虐待」「ネグレクト」との境界

ヤングケアラーは必ずしも虐待・ネグレクトではないが、重なるケースがある。

```
【虐待・ネグレクトに近い状況】
  └─ 親が子にケアを強制し、学校に行かせない
  └─ 子どもの食事・睡眠が犠牲になっている
  └─ 子どもが精神的苦痛を感じているのに親が無視
     → 児童相談所への通告対象になりうる

【ヤングケアラーだが虐待とは言えない状況】
  └─ 子ども自身がケアを選んでいる
  └─ 学業・睡眠・友人関係に大きな支障はない
  └─ 親が感謝しており関係性は良好
     → 児相への通告ではなく「伴走型支援」が適切
```

この判断は専門家（スクールソーシャルワーカー・児童相談所・保健師）が行うべきであり、学校担任やケアマネが単独で判断すべきではない。

## AIへの注意事項

⚠️ **AIは「ヤングケアラーである」という情報から自動的に支援機関・支援内容を決定してはならない。**

理由:
1. ヤングケアラーは法定概念ではなく、「どの機関が主担当か」が制度上決まっていない
2. 年齢・家族状況・ケアの内容によって関係する制度が異なり、複数機関の連携判断が必要
3. 虐待・ネグレクトとの境界判断は専門的アセスメントが必要であり、AIが自動分類できない
4. 本人が支援を望むか、秘密を守ってほしいか、学校への開示を望むかなど、本人意思の確認が前提
