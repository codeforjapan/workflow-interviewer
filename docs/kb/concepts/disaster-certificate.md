# 概念競合: 罹災証明書・被害認定

## 概要

「罹災証明書」は市区町村が発行する公的な被害証明書だが、どの被害認定区分を得るかによって利用できる支援制度が大きく変わる。同じ建物に対して、行政の被害認定と保険会社の査定が異なる結果を出すこともあり、住民は「自分が何の区分を取れたか」が何を意味するか理解できないまま各支援制度に申請しなければならない。

---

## 制度別定義一覧

| 証明書・認定 | 発行主体 | 根拠法令 | 目的・効果 |
|---|---|---|---|
| **罹災証明書** | 市区町村 | 災害対策基本法第90条の2 | 各種支援制度の利用資格を証明。区分（全壊/半壊等）が給付水準に直結 |
| **被害認定調査** | 市区町村（内閣府基準に基づく） | 内閣府「災害に係る住家の被害認定基準運用指針」 | 罹災証明書の区分を決定する調査。第1次・第2次（不服申立後）がある |
| **り災証明（損害保険用）** | 保険会社・損害保険鑑定人 | 約款に基づく（法定なし） | 火災保険・地震保険の支払い根拠。行政区分と連動しない場合が多い |
| **農業被害認定** | 農業共済組合・市区町村農業委員会 | 農業保険法 | 農業共済・収入保険の保険金算定。農地・農作物・農業用ハウス等 |
| **中小企業被害** | 商工会議所・市区町村産業課 | 中小企業信用保険法（特例） | 災害関係保証・セーフティネット融資の申請根拠 |

---

## 被害認定区分と主な支援制度の対応

```
被害認定区分（内閣府基準）
  │
  ├─【全壊】（損害割合 50%以上）
  │   └─ 被災者生活再建支援金: 基礎支援金100万円 + 加算支援金（最大200万円）
  │   └─ 応急仮設住宅への入居資格
  │   └─ 固定資産税の減免（多くの自治体で全額免除）
  │   └─ 解体撤去費用の公費負担（自治体判断）
  │
  ├─【大規模半壊】（損害割合 40〜50%未満）
  │   └─ 被災者生活再建支援金: 基礎支援金50万円 + 加算支援金（最大200万円）
  │   └─ 応急仮設住宅への入居は自治体判断
  │
  ├─【中規模半壊】（損害割合 30〜40%未満）※2020年追加
  │   └─ 被災者生活再建支援金: 基礎支援金なし + 加算支援金（最大100万円）
  │
  ├─【半壊】（損害割合 20〜30%未満）
  │   └─ 被災者生活再建支援金: 対象外
  │   └─ 修繕・補修の融資制度の対象になる場合あり
  │
  ├─【準半壊】（損害割合 10〜20%未満）※2020年追加
  │   └─ 応急修理制度の対象（上限25.7万円）
  │
  └─【一部損壊】（損害割合 10%未満）
      └─ 多くの公的支援制度の対象外
      └─ 民間保険・自己負担による修繕が基本
```

## 競合が顕在化する典型場面

### 場面1: 「半壊」と認定されたが保険は「全損」

- 内閣府基準では「半壊」（構造体の損傷割合で評価）
- 地震保険では「全損」（地震保険独自の評価基準）
- → 住民は「保険では全損なのに、なぜ被災者生活再建支援金が出ないのか」と混乱する

### 場面2: 不服申立（第2次調査）で区分が変わる

- 第1次調査で「半壊」と認定されたが、実態は「大規模半壊」相当と住民が判断
- 第2次調査（不服申立）を経て「大規模半壊」に変更されることがある
- → 不服申立の手続きを知らないまま支援を受け損なう住民が発生
- 不服申立の期限・手続き窓口を罹災証明書に明記しない自治体が多い

### 場面3: 「公費解体」対象かどうかの判断

- 全壊・大規模半壊であっても「公費解体」が自治体独自判断のため、認定区分だけでは利用可否がわからない
- 半壊でも「危険」と判断された場合に公費解体対象になる場合もある（自治体差）
- → 住民が解体業者に先に依頼してしまい、公費解体の対象から外れるケースがある

### 場面4: 賃貸住宅の借主の罹災証明書

- 建物の被害は建物所有者（家主）が認定を受ける
- 賃貸借主は「家財」の被害については罹災証明書を取得できるが、「住居の被害」区分は家主の認定を参照する
- → 借主が被災者生活再建支援金の対象外（自己所有でないため）であることを窓口で説明できないことがある

## 罹災証明書の発行フロー

```
災害発生
  ↓
市区町村が被害認定調査員を派遣（建物外観・内部調査）
  ↓
被害認定区分を決定（全壊/大規模半壊/中規模半壊/半壊/準半壊/一部損壊）
  ↓
罹災証明書を発行（申請者に交付）
  ↓
区分に不服がある場合 → 第2次調査（再調査）を申請
  ↓
各種支援制度への申請に罹災証明書を使用
```

⚠️ **大規模災害時は調査が追いつかず、証明書発行が数週間〜数か月遅延することがある。**
この遅延の間に各支援制度の申請期限が到来するため、「まず申請・後で証明書提出」を認める自治体と認めない自治体でさらに混乱が生じる。

---

## AIへの注意事項

⚠️ **AIは被害認定区分から自動的に利用可能な支援制度を列挙してはならない。**

理由:
1. 被災者生活再建支援法の対象となる「特定自然災害」に指定されていない災害では支援金が出ない
2. 自治体独自の上乗せ支援は自治体によって大きく異なる（内容・条件・期限が異なる）
3. 公費解体・応急修理の有無・上限額は自治体ごとの判断
4. 罹災証明書の区分と保険の査定区分は連動しないため、双方を説明する必要がある
5. 不服申立の可否・期限は自治体ごとに異なる
