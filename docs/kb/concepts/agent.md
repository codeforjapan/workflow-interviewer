# 概念競合（役割型）: 代理人

## 概要

「代理人」という言葉は行政手続きの現場で日常的に使われるが、**手続きごとに「誰が代理人になれるか」「代理の範囲はどこまでか」が全く異なる**。「家族が来ているから大丈夫」という判断が手続きの無効につながることもある。法的代理人・任意代理人・届出代行者を混同することが、全国共通の窓口トラブルの根源のひとつ。

---

## 代理人の類型

```
代理人の類型
  │
  ├─【法定代理人（Legal guardian / statutory agent）】
  │   法律によって当然に代理権が与えられる者
  │   └─ 例: 未成年者の親権者、被後見人の成年後見人
  │
  ├─【任意代理人（Voluntary agent）】
  │   本人が委任状等で代理権を与えた者
  │   └─ 例: 親族・弁護士・司法書士への委任
  │
  └─【届出代行者（Administrative filing proxy）】
      法的代理権はないが行政手続き上「代わりに来た人」として認められる場合がある
      └─ 例: 同一世帯員、施設職員、支援者同行
          ※ 手続きによっては法的効力が認められない場合がある
```

## 制度別・手続き別の代理人要件

| 手続き | 法的根拠 | 認められる代理人の範囲 | 必要な証明・書類 | 注意点 |
|---|---|---|---|---|
| **戸籍届出**（婚姻・死亡等） | 戸籍法第37条 | 届出義務者（親族等）が本人。一定の場合は使者（代書）による持参可 | 届出義務者の署名が原則必要 | 届出義務者以外が届出する「使者」は代理人ではない。内容を変更する権限なし |
| **住民異動届**（転入・転出等） | 住民基本台帳法第30条の46 | 本人・世帯主・世帯員が届出可。第三者は委任状必要 | 委任状・本人確認書類 | 世帯員は委任状なしで可。住所を別にする親族は第三者扱い |
| **介護保険認定申請** | 介護保険法第27条 | 本人・家族・地域包括支援センター・ケアマネ・成年後見人 | 特に制限なし（書類への署名） | 認定結果通知書は原則本人（後見人等）へ |
| **生活保護申請** | 生活保護法第24条 | 本人申請が原則。支援者同行は「同行」であり代理申請ではない | 特になし | 「代わりに申請する」権限は原則なく、窓口で本人の意思確認が必要。ただし身体的困難等の場合に例外あり（→ INC-003） |
| **マイナンバーカード申請・受取** | 個人番号法 | 法定代理人のみ代理可（任意代理人は受取のみ一定条件下で可） | 法定代理権確認書類（戸籍謄本等）、委任状（任意代理の場合） | 本人確認の厳格化により「来られない」では代理が認められないことが多い |
| **固定資産税・住民税の申告** | 地方税法 | 税理士・任意代理人 | 委任状 | 委任状なしの家族申告は法的に有効だが確認が必要 |
| **後期高齢者医療の届出** | 高齢者医療確保法 | 家族・成年後見人等 | 委任状（任意代理の場合） | 施設に入所中の高齢者については施設職員が届出代行するケースが多い |

## 「法的代理人」と「支援者同行」の違い

生活保護申請の現場で最も混乱が多い。

```
【法的代理人】
  → 本人に代わって法的効力のある意思表示ができる
  → 例: 成年後見人が被後見人の生活保護を申請する
     （後見人が申請書に署名できる）

【支援者同行・同席】
  → 本人を「支える」「手伝う」立場
  → 法的に本人に代わって意思表示する権限はない
  → 例: NPO支援者が申請に同行する
     （申請書には本人が署名する必要がある）
```

「支援者が代わりに申請した」という扱いにすると、本人の意思が確認されていない申請として無効になるリスクがある。一方、「本人が来られないから申請できない」という水際作戦（→ INC-003）も問題。身体的困難・精神的危機状態にある場合は、支援者・社協・地域包括支援センターが第一連絡窓口として機能し、適切な形で本人申請に結びつける。

## 成年後見人と代理権の範囲

成年後見人であっても、代理できない行為がある（一身専属的行為）。

| 行為 | 成年後見人の代理可否 |
|---|---|
| 財産管理（預金・不動産） | ○（後見人が代理可） |
| 介護保険申請 | ○ |
| 住民票の届出 | ○（一定の場合） |
| 婚姻・離婚の意思表示 | ✕（一身専属的。本人の判断能力が必要） |
| 遺言の作成 | ✕（一身専属的） |
| 選挙権の行使 | ✕（本人のみ） |
| 生活保護の申請 | △（本人の意思確認が前提。後見人が支援する形が原則） |

→ 「成年後見人がいれば全て代理できる」は誤り。

---

## AIへの注意事項

⚠️ **AIは「家族が来ている」「支援者が代行する」という情報から自動的に代理権を認定してはならない。**

理由:
1. 手続きごとに代理人の要件が異なり、一律に「家族なら代理可」とは言えない
2. 法定代理人（親権者・成年後見人）と任意代理人では必要書類が異なる
3. 届出代行者（使者）は代理人ではなく、内容変更の権限を持たない
4. 生活保護申請における「代理申請」は本人意思の確認が前提であり、AIが自動判断できない

代理人の適格性確認は必ず窓口担当者が行うこと。
