# 概念競合: 不登校・長期欠席

## 概要

「不登校」は日常語として「学校に来ない子ども」を意味するが、**文部科学省の統計定義・就学義務法制・教育支援・福祉的支援のそれぞれで適用される概念が異なる**。「この子は不登校ですか」という問いへの答えが文脈によって変わり、支援のスタートラインを誰が引くかが曖昧になる。

---

## 制度別定義一覧

| 区分 | 定義 | 根拠 | 扱いの主体 |
|---|---|---|---|
| **不登校（文科省統計）** | 年間30日以上欠席し、かつ**病気・経済的理由以外**の者 | 文科省「児童生徒の問題行動等調査」の調査定義 | 学校・教育委員会が集計 |
| **長期欠席** | 年間30日以上欠席した全ての者（病気・不登校・その他含む） | 同調査 | 学校・教育委員会が集計 |
| **出席停止（感染症）** | 学校保健安全法第19条に基づく校長命令による欠席 | 学校保健安全法第19条 | 欠席日数にカウントされない |
| **出席停止（いじめ加害）** | 被害者保護のためにいじめ加害者に命令される出席停止 | 学校教育法第35条 | 欠席日数にカウントされない |
| **就学義務の不履行** | 保護者が正当な事由なく子を就学させない状態 | 学校教育法第17条 | 教育委員会が督促・罰則（過料）可能 |
| **教育機会確保法上の対象** | 不登校児童生徒（学校以外の場での学習機会確保の対象） | 義務教育の段階における普通教育に相当する教育の機会の確保等に関する法律（H28） | 学校・教育支援センター・フリースクール等 |

---

## 競合が顕在化する典型場面

### 場面1：「病気で休んでいる子」が実は不登校と認識されていない

医師の診断書がある「病気欠席」は統計上の「不登校」にカウントされない。しかし長期の病気欠席の背景に学校への心理的回避がある場合でも、学校側は「病気」として処理するため、支援につながりにくい。

### 場面2：就学義務と不登校支援の矛盾

就学義務は保護者が負う（学校教育法第17条）が、不登校は「強制登校」によって解決すべき問題ではないとされている（教育機会確保法第3条）。「学校に来なくても構わない」と伝えることが教育委員会として正しい対応であっても、保護者から「それは就学義務を放棄させているのでは」と問われることがある。

### 場面3：生活保護世帯・困窮世帯の子どもの不登校

貧困と不登校が重なる場合、教育委員会・学校・福祉事務所・ヤングケアラー支援の複数機関が関わりうるが、**各機関が「うちの所管ではない」と判断して支援の空白が生じる**。就学援助の申請ルートも自治体によって教育委員会経由か福祉部局経由かが異なる（→ `stakeholders/kyoiku-iinkai.md` 参照）。

### 場面4：フリースクール等への出席が「出席」と認められるか

教育機会確保法施行後、フリースクール・教育支援センターへの出席を学校の出席として認める扱いが広がっているが、認定の基準・手続きは校長・教育委員会の判断に委ねられており全国で統一されていない。出席認定されるかどうかで「不登校の統計数」も変わる。

## 「不登校」「長期欠席」「出席停止」の関係

```
年間の全欠席者
  │
  ├─ 出席停止（感染症・いじめ加害者への命令）── 日数計算に含まない
  │
  └─ 長期欠席（30日以上）
       │
       ├─ 病気理由      ─ 「不登校」にはカウントされない（統計上）
       ├─ 経済的理由   ─ 「不登校」にはカウントされない（統計上）
       └─ その他（不安・無気力・人間関係等）─ 「不登校」としてカウント
            └─ このうちフリースクール等で学んでいる場合
                 → 出席認定される場合・されない場合がある（校長裁量）
```

## AIへの注意事項

⚠️ **AIは「不登校」という情報から自動的に就学義務違反・支援対象・福祉的支援の必要性を判断してはならない。**

理由:
1. 「不登校」の定義が文脈（統計・就学義務・支援制度）によって異なり、同じ状態でも制度上の扱いが変わる
2. 就学義務の督促は教育委員会の権限であり、学校や福祉部局が単独で判断できない
3. 不登校と貧困・虐待・ヤングケアラーが重なる場合、複数機関の連携が必要であり、AIが自動的に支援ルートを決定できない
4. フリースクール出席の「認定」は校長の個別判断であり、一律に適用できない
