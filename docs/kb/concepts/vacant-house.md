# 概念競合: 空き家

## 概要

「空き家」は日常語として「誰も住んでいない家」を意味するが、**行政制度の文脈では5つの異なる法的カテゴリが存在し、それぞれ担当課・手続き・法的効果が全く異なる**。「空き家問題の相談が来た」とき、何の制度の話をしているかを確認しないと担当課が振り回しを行う典型的な問題が発生する。

---

## 制度別定義一覧

| 制度 | 概念・区分 | 定義 | 担当 | 主な効果 |
|---|---|---|---|---|
| **空家等対策特別措置法**（H26・R5改正） | 空家等 | 建築物またはこれに附属する工作物であって居住その他の使用がなされていないことが常態であるもの（土地含む） | 市区町村 | 固定資産税情報の内部利用・所有者調査が可能 |
| 同上 | 管理不全空家 | 空家等のうち、適切な管理が行われていないことにより景観等に悪影響を与えているもの（R5改正で新設） | 市区町村 | 勧告・命令可。**住宅用地特例の解除可（R5〜）** |
| 同上 | 特定空家 | 倒壊等著しく保安上危険・衛生上有害・著しく景観を損なう・その他周辺生活環境に悪影響を与えている空家等 | 市区町村 | 勧告→住宅用地特例解除、命令→行政代執行可 |
| **固定資産税（地方税法）** | 住宅用地特例の適用可否 | 住宅が建つ土地は課税標準を1/6（小規模）または1/3（一般）に軽減。特定空家・管理不全空家への「勧告」後は解除 | 税務課 | 特例解除で固定資産税が最大**6倍**に |
| **住民基本台帳法** | 居住実態との乖離 | 住民登録はあるが実際に居住していない「幽霊空き家」。逆に登録なしに居住している場合も | 市民課 | 住民票の職権消除・調査が可能（要件厳格） |
| **不動産登記法** | 所有者不明空き家 | 相続未登記等で所有者の連絡先不明。R6.4.1施行の相続登記義務化（3年以内）で一部解消予定 | 法務局（登記）・税務課（課税） | 行政代執行・課税通知の送達不能。INC-008直結 |
| **建築基準法** | 老朽危険建築物 | 著しく保安上危険または衛生上有害な建築物 | 建築指導課（市区町村または都道府県） | 勧告・命令・代執行（特定空家とは独立した手続き） |

## 競合が顕在化する典型場面

### 場面1：「空き家の固定資産税が6倍になった」という問い合わせ

空家等対策特別措置法上の「特定空家」または「管理不全空家」に認定されて**勧告**を受けると、住宅用地特例が解除されて固定資産税・都市計画税が最大6倍になる。しかし住民は「なぜ急に税額が上がったか」がわからず混乱する。税務課・市民生活課・建設課が別々に対応している自治体では、どの課が説明責任を持つかが曖昧になる。

### 場面2：住民票があるのに誰も住んでいない

子どもが独立した後も元の住所に住民票を残したまま施設に入所、あるいは単に手続きを忘れているケース。住民基本台帳上は「居住者あり」だが実態は空き家。空家等対策法の「常態として使用されていないことが常態」という認定の妨げになりうる。住民票の職権消除は要件が厳格で容易に行えない。

### 場面3：所有者不明の空き家への対応

相続登記が未了で所有者不明の空き家は、固定資産税の課税通知先の特定・行政代執行の名あて人の特定・空家等対策協議会への情報提供の全てで行き詰まる。2024年4月施行の相続登記義務化後も、義務化以前の未登記案件は長く残存する（→ INC-008）。

### 場面4：「危険な空き家」の通報への対応

住民から「隣の空き家が傾いて危険」という通報があったとき、対応窓口が自治体によって建設課・空家対策担当・建築指導課・市民相談窓口など分散している。特定空家認定の手続き・建築基準法上の危険建築物への対応・緊急危険回避措置の3つが別々の根拠法令で別々の担当課に帰属する。

## 空き家の状態と制度の対応関係

```
物理的な「空き家」（誰も住んでいない）
  │
  ├─【住宅用地特例の適用あり】← 通常の空き家状態
  │   └─ 固定資産税：1/6または1/3に軽減
  │
  ├─【管理不全空家に認定】（R5〜）
  │   └─ 勧告 → 住宅用地特例を解除可能
  │         → 固定資産税が最大6倍に
  │
  ├─【特定空家に認定】
  │   ├─ 勧告 → 住宅用地特例解除
  │   └─ 命令 → 行政代執行（費用は所有者負担）
  │
  └─【所有者不明】（相続未登記・住民票虚偽等）
      ├─ 課税通知が届かない → 不良債権化（INC-008）
      └─ 代執行の名あて人が確定できない
```

---

## 「空き家」と隣接する概念

- **「廃屋」**: 法的定義なし。おおむね特定空家の中でも倒壊リスクが高い状態を指す俗語。
- **「空地」**: 建物のない更地。空き家対策法の対象外。別の法的問題（雑草・不法投棄等）。
- **「所有者不明土地」**: 不動産登記法・所有者不明土地利用特措法の対象。空き家問題と重なるが別の法律。
- **「老朽住宅」**: 建築基準法上の危険建築物認定の対象。現に居住中でも認定可能。

## AIへの注意事項

⚠️ **AIは「空き家」という情報から自動的に固定資産税の税額・行政代執行の可否・所有者への連絡方法を判断してはならない。**

理由:
1. 「空き家」が空家等対策法上のどの区分か（空家等/管理不全空家/特定空家）によって法的効果が全く異なる
2. 住宅用地特例の解除は「認定＋勧告」というプロセスを経ないと不可
3. 所有者不明の場合は課税・代執行のいずれも通常手順では進められない
4. 担当課が自治体によって異なるため、「どこに相談すべきか」はKBだけでは決定できない
