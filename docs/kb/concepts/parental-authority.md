# 概念競合（役割型）: 親権者

## 概要

「親」「保護者」「親権者」は日常語では互換的に使われるが、行政制度の文脈では明確に異なる。特に**離婚・未婚・養子縁組・里親・共同親権（2026年以降）の場面では「誰が親権者か」が手続きの有効性に直結する**。「子どもの親が来た」だけでは親権者かどうか判断できないことがある。

---

## 親権者の類型

```
「子どもの親」の類型
  │
  ├─【実親（婚姻中）】
  │   → 父母共同親権（民法第818条第1項）
  │   → どちらか一方が来ても共同で代理可（日常行為の範囲内）
  │
  ├─【実親（離婚後）】
  │   → 従来：父母どちらか一方の単独親権（民法第819条）
  │   → 2026年5月〜：共同親権が選択可能（民法改正）
  │   ※ 離婚時の合意または裁判所の判断により決定
  │
  ├─【未婚の場合】
  │   → 出生時は母が単独親権
  │   → 父が認知後、親権者変更の届出で父母どちらも親権者になれる
  │
  ├─【養親（普通養子縁組）】
  │   → 養親が親権者となる（実親の親権も存続する場合あり）
  │
  ├─【養親（特別養子縁組）】
  │   → 養親が親権者となる（実親の親権は消滅）
  │   → 子が成人するまで離縁原則不可
  │
  └─【里親】
      → 親権は実親（または乳児院等の施設長）のまま
      → 里親は法的代理人ではない
      → 日常的な生活行為は可能だが、手術同意等は親権者が必要
```

---

## 2026年民法改正：共同親権の導入

2024年5月に民法が改正され、2026年5月施行。離婚後も父母双方が親権を持つ「共同親権」が選択可能になる。

| 項目 | 改正前 | 改正後（2026年5月〜） |
|---|---|---|
| 離婚後の親権 | 単独親権のみ | 共同親権または単独親権（選択制） |
| 決定方法 | 離婚届に記載（協議）または審判 | 同左。DV・虐待等がある場合は裁判所が単独親権を選択 |
| 窓口対応 | 親権者の証明は戸籍で確認 | 共同親権の場合「どちらか一方」で申請できる手続きと「双方の同意」が必要な手続きが分かれる |

**⚠️ 2026年以降、窓口では「共同親権か単独親権か」を戸籍で確認する必要が生じる。**

## 制度別の「親権者」の扱い

### 児童手当（児童手当法）

**受給者** = 「児童と生計を同一にする父または母」（監護者）が原則。
親権者≠監護者のケースがある（離婚後に非親権者が実際に子を育てている場合等）。

**実務上の問題**: 離婚後に親権のない親が実際の監護者の場合、手当の受給名義の変更手続きが必要。窓口では「親権者が請求者」と誤案内されることがある。

---

### 子ども・子育て支援（保育所入所等）

保育認定申請は**保護者**（親権者または監護者）が行う。親権のない同居親族が申請を求めてきた場合に対応に迷うケースがある。

---

### 健康保険の被扶養者認定

被扶養者（子）の認定は、健康保険加入者（父または母）が「生計維持者」であることが要件。離婚後に非監護親の健保に子を入れ続けることはできない（収入要件・生計同一要件を満たさなくなる）。

---

### 予防接種・健診の同意者

予防接種法上の被接種者への接種同意は、原則として**親権者（または監護者）**が行う。里親委託中の場合、施設長・里親が同意書に署名することが多いが、法的根拠の整理が自治体ごとに異なる。

---

### 就学義務

学校教育法上の就学義務は「保護者」（親権者・後見人・養育者）が負う。親権のない同居者が事実上子を養育している場合の扱いは自治体で異なる。

---

## 「実親」「養親」「里親」の行政手続き上の違い

| 場面 | 実親（婚姻中） | 実親（離婚後単独親権） | 養親（特別養子） | 里親 |
|---|---|---|---|---|
| 保育所入所申請 | 父母どちらでも可 | 親権者のみ | 養親 | 委託元（施設長等）+ 里親 |
| 児童手当の受給 | 所得の高い方が原則 | 親権者かつ監護者 | 養親 | 里親（委託要件を満たせば） |
| 医療行為への同意 | 父母どちらでも可 | 親権者 | 養親 | 親権者（実親・施設長等）が原則 |
| 住民票の世帯員として記載 | ○ | 実際に同居している親と同世帯 | 養親世帯 | 里親世帯 |

## AIへの注意事項

⚠️ **AIは「親が来た」「保護者が来た」という情報から自動的に親権者として手続きを進めてはならない。**

理由:
1. 離婚後の単独親権の場合、来庁した親が親権者でない可能性がある
2. 2026年以降は共同親権が存在し、一方の親の意向だけでは処理できない手続きが生じる
3. 里親は法的代理人ではなく、医療同意等の重要な場面で親権者確認が必要
4. 養子縁組の形態（普通・特別）によって親権の帰属が異なる

親権者の確認は戸籍謄本または住民票の続柄情報で行うこと。2026年以降は親権欄の記載確認が特に重要。
