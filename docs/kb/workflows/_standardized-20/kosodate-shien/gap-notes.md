# gap-notes: 子ども・子育て支援（保育所等利用）

**このファイルの役割**:
標準仕様書の「あるべきフロー」と自治体の現実の差分を記録する。
差分はそのままFDE提案の根拠になる。

---

## 標準仕様書が定めているが、現実に機能していないこと

### ギャップ1：利用調整の選考基準（スコア表）が自治体ごとに異なることによる混乱

**仕様書の記述**: 保育の必要度に基づいてスコア（点数）を算出し、スコア順に利用調整を行う

**現実に起きていること**:
- スコア表の設計（就労フルタイムが何点、パート勤務が何点、など）が自治体ごとに異なる
- 加点項目（ひとり親、きょうだい同一施設希望、認可外施設からの転園など）も自治体によって大きく異なる
- 転居者が「前の自治体では保育必要度の上位3割に入っていたのに、この自治体では下位3分の1に落ちた」と困惑する
- スコア同点時の優先順位（「第1子年齢が低い」か「母の就労時間が長い」かなど）も自治体による
- 保護者はスコア表を見ても「なぜこの点数？」と納得できず、問い合わせが殺到する（特に不承諾通知後）

**FDE提案の根拠**:
仕様書は「スコア表に基づく」と定めるが、スコア設計そのものは「市町村の保育実施責任」に基づいており、統一を強制できない。
しかし待機児童解消を目指すなら、「スコア計算の透明性向上」「スコア同点時の優先ルールの統一化」が有効。
全国FDE層で「モデルスコア表」「スコア説明の標準書式」を整備すれば、
転居時の混乱や「なぜこの結果？」の問い合わせを減らせる。

---

### ギャップ2：待機児童の定義が自治体によって異なり、統計比較が困難

**仕様書の記述**: 利用調整結果に基づき、入園できなかった児童を「待機児童」として管理

**現実に起きていること**:
- 「待機児童」の定義が自治体ごとに異なる
  - 第1希望施設に入園できなかったすべての児童を計上する自治体
  - 「認可保育所のみ希望」で希望にかなわなかった児童のみを計上する自治体
  - 認可外施設に入園した児童は除外する自治体と除外しない自治体
- 国の統計（「保育所待機児童数」）でも自治体によって定義が異なり、全国比較ができない
- 同じ県内の市町村でも定義が異なるため「A市では100人、B市では50人の待機児童」と報道されても、実際の子ども供給量の差がわからない
- 育休延長を目的に「入園できなかった」ことを利用する保護者も多く、実質的な「保育が必要な児童」を把握しにくい

**FDE提案の根拠**:
待機児童統計は社会的関心が高く、地方創生・少子化対策の重要指標だが、
定義の不統一が国の政策立案を困難にしている。
FDE層で「待機児童の統一定義」「第1希望・第2希望以降の区分管理」を標準化すれば、
国や県の待機児童解消施策の優先順位付けが正確になる。

---

### ギャップ3：認可外施設利用者への給付（新制度未移行施設の扱い）の説明が複雑

**仕様書の記述**: 子ども・子育て支援新制度に基づき、認可施設・認定こども園・小規模保育事業に対して施設型給付・地域型保育給付を実施

**現実に起きていること**:
- 認可外保育施設（新制度に移行していない認可外保育室など）を利用する保護者に対し、給付の対象・非対象が複雑に説明される
  - 認定こども園に入園できず、仕方なく認可外施設を利用している保護者は給付対象
  - 但し市町村が「教育・保育給付認定」を出した場合のみ
- 認可外施設の中でも「新制度への移行済み施設」と「未移行施設」で給付扱いが異なる
- 保護者向けの説明資料が「新制度該当」「非該当」の判定フロー自体が複雑で、わかりやすい図解が自治体にない
- 施設側（認可外事業者）からの問い合わせも多く、担当者が毎回説明に時間を費やしている

**FDE提案の根拠**:
新制度は経過措置の時間が長く（令和7年度まで）、「移行済み」「未移行」が混在している状況が続く。
標準仕様書は「新制度該当施設」を主たる対象としており、未移行施設への給付ロジックが仕様書に記載不十分。
FDE層で「認可外施設利用者向けの給付判定チェックシート」「保護者向け説明書き」の標準フォームを整備すれば、
全国の市町村窓口での説明の質がそろい、保護者の混乱が減る。

---

### ギャップ4：育休延長目的の「保活失敗」への対応方針が担当者によってばらつく

**仕様書の記述**: 保護者が利用申請を行い、認定・利用調整により利用可否を決定

**現実に起きていること**:
- 「入園できなかったので育休を延長する」という保護者が多い（育児・介護・休業法で育休は最大2年）
- 自治体によって「これは不正な保活か、正当な保育必要度の不在か」の解釈が異なる
- 待機児童としてカウントするか否かも、担当者の判断でばらつく
- 同じ自治体内でも「この申請は待機児童に数える」「この申請は除外する」と判定が異なることがある
- 保護者向けの説明では「入園できなくても育休延長できます（法律の権利です）」と言う担当者と
  「育休延長を目的にした申請は保育の必要性がないと見なす」と言う担当者がいて、
  同じ市役所内で矛盾した説明になっている

**FDE提案の根拠**:
これは仕様書の「空白領域」ではなく、制度設計上の「グレーゾーン」。
労働法（育児休業制度）と福祉制度（保育給付）の綾の中で、
「育休延長を目的とした保育申請をどう扱うか」のジャッジフローが統一されていない。
FDE層で「育休延長申請への対応フロー」「保護者向け説明スクリプト」を整備し、
同一自治体内での説明の一貫性を確保することが、保護者の公平性感覚を守る。

→ 関連法令: 育児・介護・休業法 第9条（育児休暇の期間）

## 標準仕様書が定めておらず、自治体が個別設計している領域

| 領域 | 内容 | 難易度 |
|---|---|---|
| スコア表の設計 | 就労形態・時間、求職活動の扱い、加点項目の選択 | 高 |
| 保育料減額基準 | 多子軽減の比率、所得基準、きょうだい同時入園の扱い | 高 |
| スコア同点時の優先ルール | 「第1子年齢順」か「就労時間」か「申請日時」か | 中 |
| 待機児童の定義と統計方法 | 第1希望施設のみか、認可外施設入園者を除くか | 中 |
| 認可外施設の給付判定フロー | 「新制度移行施設か」の確認、給付対象の説明 | 中 |
| 育休延長申請への対応 | 保育必要性があるか、待機児童に数えるか | 中 |
| きょうだい同一施設入園希望時の加点 | 加点の有無、加点幅（何点） | 低 |
| 利用者負担額（1号認定）の決定方法 | 「保育料基準額×ランク」か「定額」か | 低 |

## 差分の構造的な意味

```
子ども・子育て支援の「穴」は待機児童と選考基準の透明性の欠如から生じる

  保育需要（待機児童の発生）
    ↓ 利用調整時のスコア表が自治体ごとに異なる
    ↓ 保護者が「なぜこの判定？」と納得できない
    ↓ 転居者が「以前と基準が違う」と混乱
    ↓ 待機児童統計が自治体ごと定義が異なり、国の対策が効きにくい
      ↑全国一律の基準がないため、制度への信頼が地域で異なる

保育所不足は供給サイドの問題（定員・施設数）だが、
選考基準の不透明さは「制度信頼」の問題。
FDE層で「スコア設計の指針」「待機児童の統一定義」「選考結果の説明様式」を整備すれば、
待機児童は減らなくても「結果が納得できる」という体験ができ、
保護者の行政への信頼感が高まる。

特に転居時に「前の自治体と基準が違う」と気づく不公平感は、
各自治体が独立に設計した基準をFDE層で可視化・標準化することで軽減できる。
```
