# 固定資産税 概要

## 制度の概要

固定資産税は、シャウプ勧告を契機とする昭和25年の地方税制度の根本的改革で創設された財産税（地方税法第340〜369条）。固定資産（土地・家屋・償却資産）の保有と市町村の行政サービスとの受益関係に着目する「応益原則」に基づき、所得ではなく資産価値そのものに応じて所有者に課税する。どの市町村にも広く存在するため税源の偏りが小さく、市町村税としての基幹税目とされる。

## 課税の基本構造

課税主体は全市町村（東京都23区内は都が課税）。納税義務者は賦課期日（当該年度初日の属する年の1月1日）時点の所有者で、土地・家屋は登記簿上の所有者等、償却資産は申告のあった所有者等を固定資産課税台帳に登録して課税する。課税標準は「価格（適正な時価）」、標準税率1.4%。免税点は土地30万円・家屋20万円・償却資産150万円（令和9年度以後は家屋30万円・償却資産180万円に変更）。
評価は総務大臣が定める固定資産評価基準（地方税法第388条・第403条）に基づき市町村長が決定。土地は売買実例価額等を基礎に、宅地は地価公示価格等の7割を目途に評価。家屋は再建築価格×経年減点補正率で算定。償却資産は取得価額から経過年数分の減価を差し引いて算定（評価額の下限は取得価額の5%）。土地・家屋は原則3年ごとに評価替え（据置年度は据え置き）、償却資産は毎年評価替え。評価替え時は負担水準（前年度課税標準額÷当該年度評価額）のばらつきを均衡化する「負担調整措置」が働く。住宅用地には課税標準の特例（小規模住宅用地1/6、一般住宅用地1/3）、新築住宅には税額の減額特例（一般住宅3年間・3階建以上等5年間、税額1/2）がある。なお家屋の評価基準は令和8年6月26日総務省告示第239号で改正され、令和9年度分から適用される。

## 年間サイクル

年度初めに前年度課税台帳を継承し、住民基本台帳・法務局登記情報等と照合、名寄処理・賦課処理・調定処理を経て納税通知書を発行するのが基本サイクル。新築・地目変更・共有者変更・減免申請は随時発生する例外処理。

## よくある論点

相続未登記による名義ズレ（総務省も所有者不明土地問題として課題認識）、評価替え時の住民説明負担、農地転用後の地目変更遅延、新築住宅軽減措置の申告漏れなどが典型（詳細はgap-notes.md参照）。

## 用語集

固定資産評価基準（総務大臣が告示する評価の基準・方法・手続、地方税法第388条）／評価替え（土地・家屋の評価額を原則3年ごとに見直すこと）／負担調整措置（負担水準のばらつきを均衡化する税負担の調整）／償却資産（土地・家屋以外の事業用有形固定資産、毎年評価替え）／名寄処理・調定処理（台帳統合・税額確定の実務工程）。

## 関連部門・関連業務の傾向

一般に資産税務を扱う課を中心に、法務局・都道府県・税務署との通知連携、都市計画部門・農業委員会との地目確認、建築確認担当との新築把握が生じやすい。課名・組織名は自治体差が大きいため、名称よりも機能・連携関係として捉えるのが有用。
