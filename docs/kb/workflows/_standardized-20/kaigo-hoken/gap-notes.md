# gap-notes: 介護保険（要介護認定）

---

## ギャップ1：転入時の「14日ルール」が転入窓口に届かない

**仕様書の記述**: 転入による引き継ぎ認定申請は14日以内

**現実に起きていること**:
- 「14日以内に介護保険課へ申請が必要」という知識は介護保険課内にあるが、転入窓口（市民課）に伝わっていない
- 市民課担当者（特に新人・会計年度任用）は介護保険証を見ても「後で介護保険課に行ってください」とだけ言う
- 住民も「また別の窓口に行けばいい」と思い、14日が過ぎてから介護保険課に行く

**仕様書との齟齬**:
仕様書は介護保険システム内の処理を定めるが、
「転入窓口から介護保険課への情報連携」は定めない。
これは業務フロー設計の問題であり、システムでは解決しない。

**被害の構造**:
```
14日超過 → 通常申請扱い（最大30日待機）
  ↓
その間、要介護者はサービスを受けられない
  ↓
家族介護者が30〜44日間、無支援で介護
  ↓
介護疲弊 → 放棄・虐待のリスク
```

→ `incident-catalog/INC-002-care-handover-14days.md`

## ギャップ2：申請から30日の「暫定サービス空白」が案内されない

**仕様書の記述**: 申請から30日以内に認定通知

**現実に起きていること**:
- 担当者が「申請してから約30日で結果が出ます」と伝えて終わり
- **暫定ケアプランを使えば申請中もサービスを受けられる**という案内がされない
- この案内ができるのはベテラン担当者のみ（属人的）

**仕様書との齟齬**:
仕様書は認定処理フローを定めるが、
「認定前の暫定サービス利用の案内」は定めない。
介護保険法上は暫定ケアプランは許容されているが、
それを窓口で案内する義務規定はない。

## ギャップ3：「気づきの入口」が多様で拾いきれない

**仕様書の記述**: 申請者から窓口へ（インバウンド申請を前提）

**現実に起きていること**:
- 「申請の仕方がわからない」「自分が対象か知らない」高齢者が多い
- 本来の入口: 本人・家族が窓口来庁
- 実際の入口: 民生委員の通報、病院MSWからの連絡、転入届の際の偶発的な気づき
- 転入窓口でA主任が「介護保険証を持っていたら声をかける」という個人ルール

**仕様書との齟齬**:
仕様書はシステムの受付機能を定めるが、
「どうやって申請が必要な住民を発見するか」は定めない。
アウトリーチ・プロアクティブな案内はCfJの仕事領域。

## ギャップ4：認定審査会の開催頻度と処理遅延

**仕様書の記述**: 審査会の開催頻度は各自治体が設定

**現実に起きていること**:
- 月2回開催が標準だが、3月末・年度始めは申請が集中して審査会待ちが発生
- 「30日以内」の法定期限を超える事例が繁忙期に発生しうる
- 超過しそうな場合の住民への説明は担当者個人の判断

## ギャップ5：災害時の介護保険サービス継続（被災・避難との交差）

**通常の介護保険フロー**:
住民票のある市区町村が保険者。要介護認定を受けた被保険者は、ケアマネが作成するケアプランに基づいてサービスを受ける。

**大規模災害発生時に起きること**:

### 問題1: 避難所での介護サービス継続
- 要介護者が避難所（体育館等）に避難した場合、通常のホームヘルプ・デイサービス等が利用できない
- 事業者自体が被災してサービスが停止することがある
- 避難所での「生活援助」をどこまで行政・ボランティアが担うか不明確

### 問題2: 他市区町村への避難（保険者の継続）
- 住民票が被災地の市区町村のままで他市区町村に避難した場合、介護保険の保険者は元の市区町村のまま
- 避難先の介護サービス事業者を利用する場合、**被災地の市区町村への事前申請・照会が必要**（通常フロー）
- 大規模災害時には「避難先でのサービス利用を事後申請・事後精算可」とする特例が国から通知されることがある
- 特例通知の内容・期間は災害ごとに異なる

### 問題3: 被災による認定有効期間中の状態変化
- 被災のショック・生活環境の激変により、要介護度が変わる（特に高齢者は悪化しやすい）
- 通常の区分変更申請が必要だが、被災直後の混乱期に申請できない場合がある
- 特例: 被災地の市区町村が「被災要配慮者の区分変更申請の受付期限を延長する」措置を取ることがある

### 問題4: 仮設住宅でのケアマネ変更
- 仮設住宅入居後、担当ケアマネ・サービス事業者を変更する必要が生じる
- 仮設住宅の住所地と元の居住地が異なる場合、担当地域の変更が必要
- 引継ぎ不十分でケアプランの継続が途切れるリスク

**対策**:
- 要配慮者名簿（介護認定者のリスト）を防災担当と共有し、災害時の優先対応者を事前に特定しておく
- 避難所に介護専門職（介護士・ケアマネ等）を配置する体制を事前に設計
- 被災特例通知が出た場合の対応手順を事前にマニュアル化しておく
