# gap-notes: 予防接種

**このファイルの役割**:
標準仕様書の「あるべきフロー」と自治体の現実の差分を記録する。
差分はそのままFDE提案の根拠になる。

---

## 標準仕様書が定めているが、現実に機能していないこと

### ギャップ1：VRS入力の遅延と接種記録の不整合

**仕様書の記述**: 医療機関がVRS（ワクチン記録システム）へリアルタイムまたは即日に接種情報を報告

**現実に起きていること**:
- 医療機関の電子カルテ連携がまだ一部のみで、大多数は紙予診票のスキャン取込に頼っている
- スキャン画像の OCR 精度が低く、日付やワクチン種別の誤読が発生する
- 市町村に報告が到着するまで数日～1週間のタイムラグがあり、その間に別の医療機関で接種される場合の重複チェックが機能しない
- 転入者の接種記録がVRS全国ネットワークに未反映の場合、新自治体では旧記録が見えず「接種歴不明」として再接種指示を出してしまう

**FDE提案の根拠**:
仕様書は「VRS連携する」と定めているがリアルタイム性・精度の要件がない。
医療機関の電子カルテ連携率を上げることとVRS入力遅延の可視化（「いつ反映されるか」の保護者向け説明）が、
FDE層での標準化対象。

→ 関連業務: `workflows/_standardized-20/iryo-kikan/gap-notes.md`（医療機関の電子カルテ対応）

---

### ギャップ2：転入者の接種記録引継ぎの不確実性

**仕様書の記述**: 転入元市町村の接種記録を確認し、重複接種や接種漏れを防止する

**現実に起きていること**:
- 全国のVRSは繋がっているが、転入元自治体の接種記録が新自治体側で参照できるまで1～2週間かかる
- その間に新自治体が「記録なし」と判断して接種案内を出してしまい、保護者が混乱する
- 海外赴任から帰国した住民の場合、海外で受けた接種（特に英文記録）がVRSに登録できず、手作業での確認が必要
- 転入時に接種記録提出を案内しないと、保護者が古い帳面（母子健康手帳の紙記録）しか持参しないケースが多く、デジタル化に時間がかかる

**FDE提案の根拠**:
これはVRS全国ネットワークの構築自体が途上だからこその問題。
「転入受付時の標準的な説明スクリプト」「VRS反映待機中の取扱い」「海外接種の記録提出方法」が、
FDE介入で整備すべき領域。

### ギャップ3：HPVワクチン積極的勧奨再開（令和4年）に伴う個別案内の実施差

**仕様書の記述**: 定期接種対象者に対し、予防接種法に基づき保護者向け案内を個別送付する

**現実に起きていること**:
- HPVワクチン（子宮頸がん予防）は積極的勧奨が令和4年再開されたが、実施時期が自治体ごとにバラバラ
- 過去に接種勧奨がされなかった世代への「キャッチアップ接種」の実施者層・通知方法が自治体で異なる
- 自治体によって、保護者への「接種の重要性」の説明の詳しさが大きく異なり、接種率に直結している
- 同一県内でも隣の市との案内形式・説明内容が異なるため、転居した保護者が混乱する

**FDE提案の根拠**:
法令では「個別案内すること」を定めるが「どの程度の説明を含めるか」は定めない。
HPVワクチンのような社会的議論のある接種については、
「保護者の疑問への標準的な回答集」「説明補助動画」といった啓発ツールが各自治体で独立に作られている状況を、
FDE層で「全国標準」として整備すれば、転居時の混乱や接種率格差の是正につながる。

### ギャップ4：接種クーポン（無料化）の管理と対象者把握のばらつき

**仕様書の記述**: 定期接種対象者への無料接種を確保するため、クーポン（または法制度による医療機関への報告）を発行

**現実に起きていること**:
- 自治体によってクーポン券の発行形式（紙券 vs デジタル vs 医療機関への通知）がバラバラ
- 転出後の引っ越し先での接種時に「前の自治体のクーポンは使えない」と医療機関で言われ、保護者が自己負担するケースが多い
- 公費負担対象外の任意接種（HPVワクチンの年上世代、おたふく風邪など）との区別が住民に伝わりきらず、問い合わせが増える
- 高齢者インフルエンザ等の「毎年更新」が必要な任意接種クーポンの更新忘れが、実施機関・保護者の両方で起きている

**FDE提案の根拠**:
クーポン管理そのものは仕様書の対象外であり、各自治体が独立に設計している領域。
しかし転出入時のクーポン有効性確認フロー、市町村間でのクーポン互換性確保、
任意接種と定期接種の案内区分の標準化が、全国FDE層で定義すれば、
住民体験を大幅に改善できる。

## 標準仕様書が定めておらず、自治体が個別設計している領域

| 領域 | 内容 | 難易度 |
|---|---|---|
| VRS入力の品質管理フロー | 紙予診票スキャンの OCR 誤読検出・修正手順 | 高 |
| 接種記録の遡及修正 | VRS登録後の誤入力発見時の訂正ルール（患者影響確認含む） | 高 |
| 海外接種の記録登録フロー | 英文母子健康手帳・予防接種記録の日本の制度への対応 | 高 |
| HPVワクチンの積極的勧奨告知内容 | 接種勧奨時の説明書・動画・Q&A の整備 | 中 |
| クーポン有効期限管理 | 定期接種は当該年度のみ有効、任意接種は要件で異なる基準 | 中 |
| 予防接種医療事故時の報告フロー | 副反応報告から予防接種健康被害救済制度への案内 | 中 |
| 未接種者の追跡・勧奨 | 案内送付後の接種確認、未接種理由の聴き取り方 | 低 |

## 差分の構造的な意味

```
予防接種の「穴」は個人の判断と全体防疫の両立の矛盾から生じる

  定期接種スケジュール（法令要件）
    ↓ VRS記録の遅延・不完全性
    ↓ 市町村の個別案内方法のばらつき
    ↓ 保護者の「いつ受けるのか」「何が無料なのか」の混乱
    ↓ 接種漏れ、または不要な再接種、または任意接種への過度な不安
      ↑転出入時にこの混乱が増幅される

定期接種率の維持は、個別案内の「質」（わかりやすさ・信頼性）に依存している。
特に社会的議論のある接種（HPV等）は、市町村独立ではなく全国標準の説明資料があれば、
「どこに住んでもおおむね同じ説明」→「接種率の全国格差是正」につながる。

VRS導入で技術的基盤は整ったが、人間の「手続きと判断」の部分での標準化が課題。
```
