# INC-005: 転入時の生活保護移管漏れによる無保護状態

## 何が起きるか

生活保護受給中の住民が転入した際、市民課が転入届を処理しても、福祉課への引き継ぎが発生せず、一時的に保護費を受け取れない無保護状態が生じる。

```
生活保護受給者が転入届を提出
  ↓
市民課が転入届を受け付け（転出証明書のみ確認）
  ↓
「福祉のことは福祉課へ行ってください」と口頭で案内するのみ
  ↓（受給者本人は転入先での生活保護申請手続き方法を知らない）
旧自治体：翌月から保護費の送金を停止（転出処理完了）
転入自治体：申請がないため保護費を支給しない
  ↓
数日〜数週間、食費・家賃・医療費の支払い手段がない
  ↓
食料不足 → 食べないで家賃に充てる
医療中断：処方箋が切れても薬局に行けない
服薬中断：高血圧・糖尿病・精神疾患の薬が途絶える
家賃滞納：大家からの督促が始まる
  ↓
症状悪化 → 重篤化
```

## 法的根拠

**生活保護法第19条（保護の実施機関）**:
市区町村が住所地の属する都道府県知事を経由して、保護の実施機関として責任を持つ。

転入自治体が「申請がないから」と不保護を続けることは、法的には「保護を受ける権利」を阻害する行為に該当しうる。

**生活保護法第7条（申請保護の原則）**:
保護は申請に基づいて開始するが、旧自治体から転入情報を受け取った場合、転入自治体は「申請受け付けの案内」をする義務がある。

## 現在の対処と属人度

市民課担当者の個人的な気づきに依存している。

- 市民課担当者が「生活保護を受けていますか？」と聞くかどうかは個人判断
- 生活保護受給証明書を受給者が持参していても、市民課がその重要性を認識していなければ対応しない
- 福祉事務所が「転入者の中に生活保護受給者がいないか」を市民課に定期問い合わせする運用を取っているかは自治体によってまちまち
- 係長が「転入届時に生活保護の有無を確認するように」と指示しているかどうかで対応が大きく異なる

## 対策案

**今すぐできること（コスト: 数万円以下）**
- 転入時チェックリストに「生活保護受給中か確認」を追加
- 「もし生活保護を受けていたら、福祉課で申請手続きをしてください。書類はこちらです」というチラシを市民課で配布
- 生活保護受給証明書の提示を受けた市民課が、その場で福祉課に電話して手続き案内をする運用を作る
- 市民課と福祉課の「転入引き継ぎリスト」（月次）を新設

**中期対策（コスト: 数十万円）**
- 住民基本台帳システムと福祉システムの転入連携を強化する（転入データを自動で福祉システムに連携）
- 福祉事務所が転入者名簿から「受給者本人またはその親族」を検出し、その場で新規申請を勧奨する体制

## 全国の類似事例

- 保護の空白による医療中断・服薬中断事例が全国の福祉事務所で報告されている
- 患者会からも「転居後、一時的に医療費が払えなくなった」という相談が複数寄せられている
- 転入手続きと生活保護の連携不備を指摘する自治体職員の論文・報告も複数存在（全国福祉事務所協議会資料等）
