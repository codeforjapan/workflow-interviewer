# INC-007: 精神障害者保健福祉手帳の更新失効によるサービス停止連鎖

## 何が起きるか

精神障害者保健福祉手帳（有効期限2年）の更新期限が迫っているが、自治体からのリマインド通知がない、または本人が体調不良で来庁できないため手帳が失効する。その結果、障害福祉サービスの利用根拠が失われ、就労継続支援やグループホームの利用が停止され、収入・住居が不安定化して精神症状が悪化する。

```
精神障害者保健福祉手帳（有効期限2年）の更新期限が近づく
  ↓（自治体からのリマインド通知なし、または本人が体調不良で来庁できない）
手帳が失効
  ↓
障害福祉サービス受給者証の更新不可（手帳が要件）
  ↓
就労継続支援・グループホーム等のサービス利用が停止
  ↓
収入・住居が不安定化
  ↓（精神症状悪化）
入院・救急搬送
  ↓（入院中は手帳更新の受診もできない悪循環）
手帳更新のための診断書取得に3〜4か月かかる
  ↓
数か月間、無サービス状態が継続
```

## 加えて国民年金との連鎖

```
手帳失効
  ↓（手帳に基づく障害基礎年金2級受給者の場合）
法定免除の根拠が失われる
  ↓
市区町村が免除解除を気づかず放置
  ↓
未納扱いになり受給資格期間が損なわれるリスク
```

## 法的根拠

**障害者総合支援法第19条（支給決定の有効期間）**:
手帳失効を放置した状態でのサービス継続は給付根拠を失う。更新案内の義務は法定されていないが、個人の権利が実質的に剥奪される状態を放置した行政の説明責任が問われうる。

## 現在の対処と属人度

一部の自治体は更新期限の2か月前に通知を送っているが、義務ではなく実施しない自治体も多い。転居により通知が届かないケースも頻発。手帳更新と年金免除手続きの連携は市区町村によって大きく異なり、障害福祉課と年金担当課の連絡が非公式運用に依存している。

## 対策案

**今すぐできること（コスト: 数万円以下）**
- 手帳有効期限の2か月前に自動アラートを送る仕組みの整備（住基データとの突合で実現可能）
- 手帳失効前に障害福祉課が対象者リストを把握し、訪問・電話による更新勧奨を実施する体制構築
- 障害基礎年金受給者向けに「手帳更新時は年金課にも報告」という案内を明記したチェックシートの配布
- 年金課と障害福祉課の情報共有フロー（失効検知時の庁内連絡票）の整備

**中期対策（コスト: 数十〜数百万円）**
- 手帳有効期限を健康管理システムに登録し、自動で福祉事務所に通知する仕組みの構築
- 手帳失効時の障害福祉サービス受給者証自動更新停止ルール（誤給付防止）を明文化
- 精神保健福祉センター・医療機関との連携で、手帳更新が困難な者（入院中など）への代替申請プロセス構築

## 全国の類似事例

精神障害者の手帳更新失効によるサービス停止は全国的に報告されており、NPO・支援団体から制度改善の提言が出ている。手帳失効により無サービス状態となった利用者が入院・危機的状況に陥るケースは毎年複数報告されている。
