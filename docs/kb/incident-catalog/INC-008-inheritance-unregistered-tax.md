# INC-008: 固定資産税の相続未登記継続による課税誤りと不良債権化

## 何が起きるか

所有者が死亡した後、相続人が不動産登記をしないまま放置し、市区町村の固定資産税課税台帳が被相続人名義のままになる。自治体は連絡先を把握できないまま納税通知書を郵送できず、対象地が「空き家」として放置され、やがて税債権が時効で消滅してしまう。

```
所有者が死亡
  ↓（相続人が不動産登記をしないまま放置）
市区町村の固定資産税課税台帳：被相続人名義のまま
  ↓
法務局の登記情報との乖離が続く（数年〜数十年）
  ↓（自治体は連絡先を把握できない）
納税通知書が届かない → 未納扱い
  ↓
督促・差押えを行おうとするが「所有者が死亡している」ため執行不能
  ↓
時効（5年）経過 → 税債権消滅（自治体の損失）
  ↓（相続人が複数いる場合）
誰に請求すべきかわからず課税事務が停滞
```

## 自治体の法的リスク

**地方税法に基づく徴収困難化**:
法定相続人全員への課税通知が必要だが、相続人の特定コストが高い。相続人が複数いる場合、共有状態での持ち分に応じた課税が必要となるが、登記がなければ持ち分把握も困難。

2024年4月から相続登記が義務化（不動産登記法改正）されたが、義務化以前の未登記案件は残存しており、自治体の固定資産税徴収漏れは地方税法上の徴収義務に抵触する。

## 現在の対処と属人度

死亡届受理後に「固定資産税課への連絡票」を起こす自治体もあるが、多くは戸籍担当者の個人判断に依存。課税台帳の名義変更は「相続人からの申告」を待つ受動的運用が一般的であり、自治体側が死亡情報を基に主動的に課税名義を変更する運用は少ない。

## 対策案

**今すぐできること（コスト: 数万円以下）**
- 死亡届受理時に固定資産税課へ連絡する「死亡情報通知票」を整備し、市民課から税務課への連絡フローを自動化
- おくやみ手続き案内票に「固定資産税の名義変更（相続登記）」を必須項目として追加
- 相続登記義務化（2024年4月施行）について、おくやみ窓口で遺族に対して書面で周知
- 法務局の登記簿と固定資産税課税台帳の照合作業を年1回実施し、乖離案件を特定する仕組みの構築

**中期対策（コスト: 数十〜数百万円）**
- 死亡情報の庁内連携を自動化（市民課→税務課）し、課税台帳の名義変更を自動実施するシステム整備
- 相続人特定が困難な案件については、相続財産管理人制度の活用や相続放棄等の法的手続きを案内するプロセス構築
- 空き家管理台帳と固定資産税台帳を連携させ、所有者不明土地の早期把握機構の構築

## 全国の類似事例

相続未登記による所有者不明土地問題は全国で約410万haに上るとされている（国土交通省推計）。固定資産税の不良債権化も複数の自治体で問題化しており、総務省・国土交通省が相続登記義務化と庁内連携強化を促進している。
