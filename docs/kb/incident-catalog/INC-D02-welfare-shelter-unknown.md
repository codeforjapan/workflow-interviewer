# INC-D02: 福祉避難所の存在が周知されず、要配慮者が一般避難所で劣悪な状況に置かれる

## 何が起きるか

要配慮者（車いす利用者・人工透析患者・妊産婦・医療的ケア児等）が一般の指定避難所（学校体育館等）に避難するが、バリアフリー設備がない・専門スタッフがいない・プライバシーが確保できない等の劣悪な環境に置かれる。福祉避難所の存在を知らない住民・職員が多く、移送調整も機能しないため、必要な人が福祉避難所に届かない。

```
大規模災害発生
  ↓
要配慮者が「避難所＝学校の体育館」と認識して一般指定避難所に避難
  ↓
体育館：段差あり・スロープなし・床は板間・大勢の避難者
  ↓【問題1: 場所が非公表】
「福祉避難所の場所を住民に教えない方針」の自治体では、住民が行き先を知らない
  ↓【問題2: 直接来所禁止ルール】
「市区町村の窓口を経由して移送してもらう必要がある」が、
  その市区町村の本部が混乱しており連絡がつかない
  ↓【問題3: 受け入れ容量不足】
福祉避難所（事前協定の施設）自体が被災、または利用者で満員
  ↓
要配慮者が一般避難所の通路・玄関・車中に留まり続ける
  健康悪化・医療処置の遅延・精神的苦痛
```

## 法的背景と2021年改正

- 災害対策基本法第49条の7第3項: 福祉避難所の指定
- 2021年の内閣府ガイドライン改正: 「直接受け入れ型」の福祉避難所を認める（従来は原則禁止）
- しかし、直接受け入れ型を実施している自治体は全体の一部にとどまっている（2023年時点）

## 現在の対処と属人度

- **属人度: 高い**
- 避難所の担当職員が「この人は福祉避難所に移ってもらう必要がある」と判断しても、福祉避難所の場所・連絡先を知らないことがある
- 「現場の判断」で体育館の隅に段ボールで仕切りを作って対応する事例が多い
- 地域包括支援センター・ケアマネ・相談支援専門員が独自に連絡を取って移送を調整している場合がある（個人の善意と関係に依存）
- 福祉施設側も「誰を受け入れるか」の基準が明確でなく、施設の裁量で対応しているケースが多い

## 対策案

**今すぐできること（コスト: 数万円以下）**
- 福祉避難所の場所・連絡先・受け入れ対象者を避難所担当職員全員に共有（非公表の場合も職員は把握）
- 一般避難所での要配慮者の早期把握と福祉避難所移送の判断基準を明文化
- 福祉避難所との事前協定の内容確認（「どんな状態の人が何人受け入れられるか」を具体化）

**中期対策（コスト: 数十〜数百万円）**
- 「直接受け入れ型」福祉避難所への移行（対象者と場所をあらかじめ公表し、直接避難できるようにする）
- 福祉避難所の受け入れ訓練（年1回以上）の実施
- 避難行動要支援者の個別避難計画と福祉避難所の移送計画を連動させる
- 事前協定施設の被災時のバックアップ施設リストの整備

## 全国の類似事例

- **熊本地震（2016年）**: 体育館での車いす利用者・人工透析患者の苦境が報告された。特別支援学校等を福祉避難所として開設したが、情報が届かず活用されないケースがあった。
- **東日本大震災（2011年）**: 障害者・高齢者が一般避難所で長期間過ごし、健康悪化・死亡（関連死）に至る事例が多数発生。「震災関連死」の多くが避難所環境の問題と指摘されている。
- 内閣府「令和3年度 避難に関する総合的対策の推進に関する実態調査」において、福祉避難所の周知不足が全国共通の課題として指摘されている。
