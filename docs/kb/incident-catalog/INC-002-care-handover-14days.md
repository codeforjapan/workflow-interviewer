# INC-002: 介護認定引き継ぎ14日切れ

## 何が起きるか

要介護認定を受けた状態で転入してきた住民が、転入窓口で「後で介護保険課に行けばいい」と案内されたまま14日が過ぎ、通常申請扱いになる。

```
要介護者が転入
  ↓
市民課で転入届受付
  ↓（市民課担当者が介護保険証に気づかない、または案内しない）
14日経過
  ↓
介護保険課で申請 → 通常申請扱い（引き継ぎ不可）
  ↓
認定まで最大30日間、サービスを受けられない
  ↓
家族介護者が30〜44日間、無支援で介護継続
  ↓
介護疲弊 → 放棄・虐待のリスク
```

## 現在の対処と属人度

みなと市では市民課A主任が「介護保険証を持参していたら介護保険課に電話する」という個人ルールで対処。A主任不在日・新人担当者は機能しない。

## 対策案

**今すぐできること（コスト: 数万円以下）**
- 転入受付チェックリストに「65歳以上の転入者→介護保険証の有無を確認→14日ルールを案内」を追加
- 市民課・介護保険課の連絡票（紙）を整備

**中期対策（コスト: 数十万円）**
- 標準準拠システムへの移行時に、転入処理と連動した介護保険課への通知機能を要件定義に含める
