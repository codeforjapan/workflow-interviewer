# INC-006: 退職後の国保・年金未手続きによる無保険状態と受診抑制

## 何が起きるか

退職後、退職者が国民健康保険・国民年金の手続きを行わないまま時間が経過し、医療を受ける際に無保険と判明。受診抑制や医療費の支払い不能に陥る。

```
退職
  ↓
「ハローワークへ失業給付の手続きに行かなければ」と思い込む
市区町村窓口に来ない（国保・年金の手続き時期を逃す）
  ↓
国民健康保険：未加入のまま3ヶ月以上経過
国民年金：未届けのまま未納期間が発生
  ↓
病気またはケガ → 医者に行く → 受付で「保険証がない」と判明
  ↓
医者が自費扱いで対応 → 10割請求（保険診療の3倍）
（例: 初診料5,000円程度 → 15,000円程度に）
  ↓
高額請求に驚愕 → 支払い拒否 → 医療中断
  ↓
症状悪化 → 重症化 → 急性増悪で救急車搬送 → 入院（さらに高額医療費）
  ↓
医療費が払えない → 仕方なく国保に遡及加入
  ↓
加入から過去期間の保険料を一括請求（数十万円の請求）
  ↓
医療費 + 遡及保険料 → 支払い困難 → 生活困窮 → 生活保護申請
```

## 国民年金の穴

```
退職後に免除申請をしないまま未納が2年以上継続
  ↓（時効成立）
過去分の遡及申請・追納が不可能に
  ↓
受給資格期間（10年）を満たせない可能性
  ↓
65歳で老齢基礎年金ゼロ
  ↓
老後の生活保護申請 → 行政負担増（年間100万円以上）
```

## 法的根拠

**国民健康保険法第6条（加入資格）**:
退職して社会保険を喪失した場合、14日以内に国民健康保険に加入する義務がある。期限を過ぎての加入は「加入までの期間、無保険」となる。

**国民年金法第89条（免除）**:
経済困難な場合は保険料免除を申請できるが、申請しないまま未納が続くと資格期間に含まれない。

未加入・未納は法令義務違反だが、**行政が周知・案内しないことで「実質的に住民が義務を果たせない状態」を放置している**側面がある。

## 現在の対処と属人度

退職者への案内は極めて非統一的である。

- ハローワーク：失業保険の手続き時に国保・年金の案内をすることもあるが、義務ではない
- 市区町村：「広報誌に載せている」程度の場合が多く、転出者全員へのアナウンスではない
- 企業退職者向けの集団説明会を開いている自治体は限定的
- 「退職後は自分で手続きするもの」という認識が住民側にあり、自治体からの主動的な案内がないと動かない

## 対策案

**今すぐできること（コスト: 数万円以下）**
- 「退職者向けワンストップ案内チラシ」を整備（国保・年金・住民税の3点セット、手続き場所・必要書類をA4両面に集約）
- ハローワークとの連携協定を締結し、ハローワーク施設内に「市区町村手続き相談ブース」を月1回設置
- ハローワークで失業保険の申請を受け付ける際に、市区町村の案内ブースへの来庁を勧奨する（ハローワーク職員への働きかけ）
- 退職後「30日以内に市区町村の福祉相談窓口へ」という企業向けの啓発文書を作成（商工会議所経由で配布）

**中期対策（コスト: 数百万円）**
- ハローワークと市区町村の情報連携システムを構築（失業給付申請者のリストを市区町村が受け取り、国保・年金加入勧奨通知を自動送付）
- 退職者・高齢者向けの「人生設計セミナー」を年3回以上開催（年金・国保・税制の基礎知識を集中講座）
- 経理担当者・人事担当者向けの「退職予定者の手続き案内マニュアル」を整備し、企業に配布

## 全国の類似事例

- 生活困窮者支援窓口に「退職後、無保険・無年金になって医療が受けられなくなった」という相談が多数寄せられている
- 年金事務所の窓口相談でも「昔、支払わなかった期間があって、今加入しても年金がもらえないと言われた」という相談が多数
- 医療機関の相談室からも「自費診療での支払い能力がない患者の生活保護申請サポートが増加している」という報告がある
