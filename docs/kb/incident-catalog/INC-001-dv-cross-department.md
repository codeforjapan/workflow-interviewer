# INC-001: DV情報の課横断連携失敗

## 何が起きるか

DV被害者が転入する際、市民課がDV支援措置フラグを設定しても、その情報が子育て支援課・福祉課に伝わらないまま各課が通常通り処理を進める。

```
市民課: DV支援措置を設定・新住所を秘匿処理
  ↓
子育て支援課: 児童手当の給付通知書に新住所を印字して郵送
  ↓ （加害者が請求者だった場合、通知書から住所が判明）
福祉課: 生活保護の扶養照会で夫（加害者）に連絡
  ↓
加害者が転入先市区町村・住所を特定
  ↓
接触・傷害事件
```

## 現在の対処と属人度

多くの自治体で「係長間の電話連絡」という非公式運用で補われている。

- 連絡するかどうかは市民課係長の判断と知識に依存
- 子育て支援課が給付通知書の住所を「窓口渡し」に変更する運用も、担当者の個人判断であることが多い
- 係長・ベテラン担当者の異動・退職で即時消失するリスクがある

## 対策案

**今すぐできること（コスト: 数万円以下）**
- DV案件発生時の「3課連絡票」を紙で整備し、電話後に票を起こす習慣を作る
- 給付通知書の「窓口渡し」運用をマニュアルに明文化する
- 生活保護の扶養照会判断チェックシートを整備する（根拠: 厚労省通知 R3.3.30）

**中期対策（コスト: 数十〜数百万円）**
- 住基システムのDVフラグを給付管理システムに連携する機能を追加
- DVフラグがある場合の通知書を自動的に「窓口渡し」に設定

## 全国の類似事例

DV被害者の転居先が加害者に判明したケースは全国で複数報告されている。
住民票の閲覧制限制度（支援措置）が整備されている一方、給付通知書・扶養照会といった「支援措置の外側」からの漏洩については制度的な手当が遅れている。

参照: 総務省「配偶者からの暴力を受けた被害者等の住民基本台帳事務における支援措置について」（H20.3.3 総行市第31号）
参照: 厚労省「生活保護問答集について」扶養義務者への照会に関するQ&A（R3改正）
