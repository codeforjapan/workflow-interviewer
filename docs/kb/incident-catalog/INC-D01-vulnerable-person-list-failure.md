# INC-D01: 避難行動要支援者名簿が機能せず、要配慮者が取り残される

## 何が起きるか

市区町村が法的義務として作成している「避難行動要支援者名簿」が、災害時に実際の避難支援に活用されず、要配慮者（高齢者・障害者・医療依存者等）が取り残される。名簿の作成は義務化されているが、「誰が・どうやって・いつ助けに行くか」の個別避難計画の整備が追いついておらず、名簿の存在だけでは機能しない。

```
市区町村が避難行動要支援者名簿を作成
  ↓（平時）
名簿が「個人情報保護」を理由に関係機関に共有されていない
  または「同意した人のみ」共有されており、多くの要配慮者が含まれていない
  または名簿情報が古く（2年前等）実態と乖離している
  ↓
大規模災害が発生（夜間・早朝）
  ↓
消防・民生委員・自治会が名簿を持っていない、または持っていても古い
  ↓
要配慮者への訪問・救助が誰の責任で行うか不明
  「民生委員が行くと思っていた」「消防が来ると思っていた」「市の職員が来るはずだ」
  ↓
要配慮者が自力で避難できず、または搬送が遅れる
  ↓
取り残し・遅延避難による被害の拡大（孤立・低体温・転倒等）
```

## 法的背景

- 災害対策基本法第49条の10：市区町村に名簿作成を**義務化**（2013年改正）
- 災害対策基本法第49条の14：個別避難計画の作成を市区町村の**努力義務**化（2021年改正）
- 個人情報保護法・自治体条例：平時の第三者提供には原則本人同意が必要

⚠️ 「名簿を作っているから大丈夫」という認識は誤り。名簿の活用（関係機関への共有・個別計画の策定）がなければ機能しない。

## 現在の対処と属人度

- **属人度: 非常に高い**（地域差が大きい）
- 機能している地域では、民生委員が独自に要配慮者を把握・訪問しており、名簿に依存しない個人的な関係で支援が成立している
- 民生委員の高齢化・なり手不足により、この属人的な支援体制の維持が困難になりつつある
- 自主防災組織が機能している地域では、地区独自の名簿・避難訓練で補完している場合がある
- 民生委員・自主防災組織のない地区では、誰も責任を持って要配慮者を助けに行かない状態になる

## 対策案

**今すぐできること（コスト: 数万円以下）**
- 避難行動要支援者名簿の「関係機関への共有」の見直し（条例改正による非同意者の掲載・共有の検討）
- 名簿情報の年次更新ルールの策定（介護認定・障害手帳の更新と連動）
- 民生委員・消防団・自治会への名簿共有の実態確認と文書化

**中期対策（コスト: 数十〜数百万円）**
- 個別避難計画の作成率向上（2021年努力義務化を受けた具体的な計画作成の推進）
- 避難支援者（民生委員・消防団員等）との事前合意（「誰が行くか」を計画に明記）
- 災害対応訓練への要配慮者本人の参加（実際に避難できるか確認）
- デジタル技術（マイナンバー等）を活用した名簿の自動更新・共有システムの構築

## 全国の類似事例

- **東日本大震災（2011年）**: 宮城県・岩手県等で多くの要配慮者（特養・グループホーム入居者含む）が津波に飲まれた。事前の個別計画がなく、施設職員・民生委員等が独自判断で対応したが間に合わなかったケースが多数報告された。
- **熊本地震（2016年）**: 仮設住宅での孤立死問題が発生。名簿があっても仮設住宅入居後の追跡ができていなかった。
- 総務省「避難行動要支援者の避難行動支援に関する取組指針」（2021年改定）にも多数の事例が掲載されている。
