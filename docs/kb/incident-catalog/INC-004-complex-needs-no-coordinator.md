# INC-004: 複合課題の調整者不在による孤立死

## 何が起きるか

80代の親（要介護3）と50代の子（精神障害・無職）という複合課題世帯。各課が自分の領域のみで対応し、誰も世帯全体を見ていないため、世帯が徐々に孤立していく。

```
80代の親（要介護3）× 50代の子（精神障害・無職）
  ↓
介護保険課 → 親のケア（訪問介護・デイサービス）のみ対応
福祉課 → 子の障害手帳・障害年金のみ対応
就労支援課 → 子の就労支援のみ対応
  ↓（誰も世帯全体の経済・生活を見ていない）
親の年金が途絶える（死亡または認知症悪化で銀行印が見つからない等）
  ↓
食料・光熱費が尽きる → 生活保護申請の気力が出ない
  ↓
子が支援を求める気力・能力を失う → ケアマネ・福祉事務所への連絡なし
  ↓
介護サービス利用中止 → 親が転倒・誤嚥 → そのまま放置
  ↓
孤立死（親または子、または両方）
  ↓（発覚した場合）
行政の「支援していたはずなのに見落とした」責任が問われる → 新聞報道・議会質問
```

## 法的根拠

**社会福祉法第106条の4（重層的支援体制整備事業）**:
複合課題を抱える世帯に対して、包括的な支援を行う体制整備は「努力義務」である。

複合課題世帯へのアウトリーチ・調整が機能しなかった場合、行政の説明責任が問われる。なお、相談支援専門員やケアマネジャーが「異変に気づいていながら報告・連携しなかった」場合、民事責任が生じうる。

## 現在の対処と属人度

現状は非公式な個人の「気になる世帯リスト」で管理されている。

- ベテランケースワーカーや地域包括支援センターの職員が個人的に気にかけて月1回連絡している
- その職員が異動・退職すると、その関心と連携が完全に消失
- 令和4年から制度化された「重層的支援体制整備事業」があるが、参加自治体は全国でも限定的（約300自治体）
- 参加していない自治体では制度的な手当がなく、どの課が「世帯全体」を見るべきか不明確なままになっている

## 対策案

**今すぐできること（コスト: 数万円以下）**
- 「複合課題世帯チェックリスト」を整備（介護×障害、介護×生活保護、障害×生活困窮の組み合わせを検出）
- 福祉課・介護保険課・就労支援課による「定期合同ケース会議」（月1回）を新設し、該当世帯を抽出・情報共有
- ケアマネジャー・相談支援専門員の連携フローを文書化（「世帯に異変があったら福祉課に報告」を義務化）

**中期対策（コスト: 数百万円）**
- 重層的支援体制整備事業への参加申請（厚労省補助金あり）
- 専任の「複合課題調整員」ポストを設置（福祉課配置が推奨）
- 各課システムの「世帯タグ」情報を共有する仕組みを構築

## 全国の類似事例

- 2020年代に入り、「8050問題」（80代親×50代子）に関連する孤立死が全国で報道されている
- 東京都内の複数事例では、複数の支援制度の対象であったにもかかわらず、誰も世帯全体の状態を把握していなかった
- 福祉事務所・ケアマネジャー双方が「相手方の領域」と考えて情報共有していなかった事例が多い
