# INC-009: おくやみ後の年金停止遅延による過払いと遺族への一括返還請求

## 何が起きるか

被相続人（年金受給者）が死亡したが、年金事務所への死亡届出が遅れるため、年金機構が死亡を把握せずに年金を振り込み続ける。その後、過払いを検知した年金機構が遺族に対して過払い分の一括返還請求を行い、既に相続財産を使用・分割済みの遺族が困窮する。

```
被相続人（年金受給者）が死亡
  ↓
遺族が死亡届を市区町村に提出（戸籍・住民記録の手続きをこなすだけで精一杯）
  ↓（「年金の停止は日本年金機構へ」という案内がない、または聞き流す）
年金事務所への死亡届出が遅れる（数か月〜数年）
  ↓
年金機構が死亡を把握せず年金を振り込み続ける（過払い）
  ↓
年金機構が過払いを検知（本人の確認書類未提出などで発覚）
  ↓
遺族に対して過払い分の一括返還請求
  ↓（遺族が既に相続財産を使用・分割済みの場合）
返還できず困窮
  ↓（悪質と判断された場合）
不正受給として刑事告発リスク（刑法246条：詐欺罪）
```

## 加えて未支給年金・遺族年金の請求漏れ

```
死亡届提出に追われ「未支給年金の請求」を知らない
  ↓
5年の時効で請求権消滅
  ↓
受け取れたはずの数十万〜数百万円の給付が失われる（経済的損失）
```

## 法的根拠

**国民年金法第105条（死亡の届出）**:
死亡届の提出義務は遺族にあり、怠った場合は不正受給として返還義務が生じる。悪質なケースでは詐欺罪が適用された判例もある。一方、未支給年金の請求権は時効（5年）で消滅するため、案内しない自治体の「不作為」も問題となりうる。

## 現在の対処と属人度

死亡届受理時に年金停止の案内を行う自治体もあるが、口頭説明のみで書面を渡さないケースが多い。遺族が複数いる場合、誰が手続きを行うのかが曖昧なまま進むことも多い。「おくやみ窓口」を設置している自治体では改善が見られるが、まだ全国的ではない。年金事務所との情報共有フロー（システム連携）がない自治体がほとんど。

## 対策案

**今すぐできること（コスト: 数万円以下）**
- 死亡届受理時に「年金停止の手続き案内」を書面で必ず渡す（年金事務所への連絡先・手続き期限を明記）
- 「未支給年金の請求期限（5年）」と「遺族年金の請求可能性」を同じ案内票に記載し、相続人全員に周知
- 死亡届受理者（喪主など）の氏名・連絡先を記録し、後日の確認に使用できるようにする
- 年金機構に対して「市区町村での死亡届受理情報を迅速に受け取る仕組み」の構築を提案

**中期対策（コスト: 数十〜数百万円）**
- おくやみワンストップ窓口の設置（市民課・税務課・年金課・福祉課が一同に対応）
- 市区町村と年金事務所との情報連携協定を締結し、死亡情報を自動共有するシステムの構築
- 年金過払い返還請求時に分割納付や返還猶予の案内を必須化する制度提案（国への要望）
- 高齢者・認知症等で対応困難な遺族のための「年金停止手続き代行」サービスの整備

## 全国の類似事例

年金の過払い問題は毎年度発生しており、日本年金機構が返還請求を行うケースは年間数万件規模とされる。遺族が高齢・認知症などで対応困難なケースや、既に相続財産を使用済みで返還不可能なケースも報告されている。相続税と年金過払いの二重負担により困窮する事例も複数報告されている。
