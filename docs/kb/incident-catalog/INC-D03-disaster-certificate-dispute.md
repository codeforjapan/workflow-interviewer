# INC-D03: 被害認定区分への不服で第2次調査を申請しないまま支援を受け損なう

## 何が起きるか

罹災証明書の被害認定区分（例: 「半壊」と認定されたが実態は「大規模半壊」相当）に住民が不服を感じながら、「第2次調査（不服申立）」という制度を知らないまま諦め、本来受けられるはずの被災者生活再建支援金・仮設住宅入居資格等の支援を受け損なう。

```
大規模災害発生
  ↓
市区町村が被害認定調査を実施
  ↓（調査員不足・被害の複雑さ・外観調査の限界）
住民が「半壊」と認定される（実態は「大規模半壊」に近い状況）
  ↓
「半壊」では被災者生活再建支援金の基礎支援金（50万円）が出ない
  「大規模半壊」なら50万円 + 加算支援金（最大200万円）が受けられた
  ↓【制度の不周知】
「第2次調査（不服申立）が可能」ということを、
  罹災証明書の交付時に説明されなかった、または見落とした
  ↓
不服申立の期限（多くの場合30日〜60日程度）が過ぎる
  ↓
被災者が「半壊」のまま確定し、本来受けられるはずの支援を受けられない
  住宅再建ができず、仮設住宅の入居資格もなく、長期的な生活困窮に
```

## 関連する制度的問題

### 問題1: 「一部損壊」（損害割合10%未満）の壁

一部損壊では、ほとんどの公的支援制度の対象外になる。しかし実態として修繕費用が数十万〜数百万円かかる「一部損壊」が存在し、住民の生活への影響は深刻でも支援がほとんどない状態が生まれる。

```
一部損壊（損害割合10%未満）
  ↓
被災者生活再建支援金: 対象外
応急修理制度: 対象外
仮設住宅: 対象外
固定資産税減免: 自治体によっては対象外
```

### 問題2: 第1次調査と第2次調査の区分変更

第2次調査（再調査）の結果、区分が「半壊→大規模半壊」に変更されるケースがある。しかし変更後に受けられるはずだった支援の申請期限がすでに経過している場合、区分変更だけでは遡及して支援を受けられないことがある（自治体・制度によって異なる）。

### 問題3: 賃貸借主への説明不足

建物の被害認定は建物所有者（家主）が対象。賃貸借主は「家財の被害」については罹災証明書を取得できるが、「住居の被害区分（全壊・半壊等）」は自分の認定ではなく家主の認定となる。賃貸借主は被災者生活再建支援金の対象外（建物所有者でないため）であることが、証明書交付時に説明されないケースがある。

## 現在の対処と属人度

- **属人度: 非常に高い**（自治体の説明体制に左右される）
- 罹災証明書の「交付スタッフ」がいれば、その場で不服申立の説明ができるが、大規模災害時は証明書発行が機械的・大量処理になり、説明が省略される
- 「知っている住民」は自分から不服申立を請求するが、「知らない住民」は泣き寝入りする
- 支援団体（NPO・弁護士等）が被災地に入って、申立制度を住民に説明している事例がある

## 対策案

**今すぐできること（コスト: 数万円以下）**
- 罹災証明書の交付時に「第2次調査（再調査）の申請が可能です」という一文と申請期限を書面で必ず渡す
- 一部損壊・半壊の住民に対して「利用できる支援制度の一覧（支援の有無の確認表）」を罹災証明書と同時に配布
- 交付窓口での「不服申立の手順案内」を掲示・配布できる体制を整備

**中期対策（コスト: 数十〜数百万円）**
- AI・ドローン等を活用した被害認定調査の精度向上（第1次調査での精度を上げて、不服申立を減らす）
- 区分変更後の支援申請に遡及適用を認める自治体独自の救済措置の設計
- 賃貸借主向けの別途の支援案内（家賃支援・生活再建資金貸付等の案内）
- 罹災証明書発行と被災者支援制度の案内を一体化した「被災者ワンストップ窓口」の設置

## 全国の類似事例

- **熊本地震（2016年）**: 被害認定の不服申立が相次ぎ、第2次調査で区分が変更されるケースが多発。弁護士会・NPO等が被災者への説明活動を行った。
- **東日本大震災（2011年）**: 津波被害と液状化被害の認定基準が違い、液状化被害の住民が「一部損壊」と認定されて支援対象外になる問題が発生した（千葉県浦安市等）。浦安市は独自の支援制度を創設して対応。
- **台風19号（2019年）**: 河川氾濫による浸水被害で、床上浸水の深さによって区分が変わるため、数センチの差で支援対象が変わることへの住民の不満が報告された。
