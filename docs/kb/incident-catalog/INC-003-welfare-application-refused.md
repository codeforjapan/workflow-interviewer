# INC-003: 生活保護申請書の不交付

## 何が起きるか

申請意思を示した住民に申請書を渡さない（または渡すことを先送りさせる）ことで、生活保護の申請が阻まれる。

```
困窮者が相談に来る
  ↓
「まず親族に相談してください」
「まだ働けそうですね」
「書類が揃ってから来てください」
  ↓（申請書を渡さない）
保護を受けられない状態が継続
  ↓
食費・医療費が尽きる
  ↓
餓死・病死・自殺
  ↓（発覚した場合）
報道・刑事訴追・行政訴訟・担当者・管理職の処分
```

## 法的根拠

**生活保護法第7条（申請保護の原則）**:
保護は要保護者、その扶養義務者またはその他の同居の親族の申請に基づいて開始する。

申請意思を示した住民に申請書を渡さないことは**この原則に反する違法行為**。

## 現在の対処と属人度

みなと市では係長が「申請書は必ず渡せ」と口頭指示しているが文書化なし。善意の新人CWが「まだ頑張れそう」と判断して先送りするリスクが常在している。

## 対策案

**今すぐできること（コスト: 数万円以下）**
- 「申請書は必ず渡す」を明文化したチェックシートを全CWに配布
- 月1回のロールプレイ研修（困窮者の相談シナリオを想定）
- 新人CW研修に「水際作戦とは何か・なぜ違法か」を必須化

**システム的対策（コスト: 低）**
- 初回相談の受付記録を必須入力にする（「相談を受けたのに申請書を渡さなかった」を記録に残す）

## 全国の類似事例

- 北九州市事件（2007年）: 生活保護を断られた後に餓死
- 同種事案はその後も複数の自治体で発生・報道
