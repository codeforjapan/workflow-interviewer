## 組織概要

| 項目 | 内容 |
|---|---|
| 正式名称 | 児童相談所（○○児童相談所、中央児童相談所等、名称は都道府県による） |
| 設置根拠 | 児童福祉法第12条 |
| **設置者** | **都道府県・指定都市**（義務設置）。中核市・特別区は任意設置（2024年から中核市にも設置努力義務） |
| 対象年齢 | 0〜18歳未満の児童（必要に応じて20歳未満まで対応可） |
| 主な職員 | 児童福祉司・児童心理司・医師・弁護士（配置義務）・保健師等 |
| 財源 | 都道府県・指定都市の一般財源 + 国庫補助 |

⚠️ **児童相談所は市区町村の機関ではない。一時保護の権限は児童相談所のみが持つ。**

---

## 主な機能

| 機能 | 内容 | 根拠 |
|---|---|---|
| **相談・調査** | 子どもや家庭に関するあらゆる相談の受付・調査 | 児童福祉法第26条 |
| **一時保護** | 子どもの安全確保のために施設等に一時的に保護（保護者の同意不要） | 児童福祉法第33条 |
| **措置決定** | 施設入所・里親委託等の行政措置を決定する | 児童福祉法第27条 |
| **里親・養子縁組** | 里親の認定・委託・特別養子縁組のあっせん | 児童福祉法第34条の19等 |
| **家庭裁判所への送致** | 非行少年の送致、親権喪失審判の申立て | 少年法・民法 |

## 市区町村との役割分担（2022年児童福祉法改正後）

2022年の児童福祉法改正により、市区町村に「こども家庭センター」の設置が努力義務化（2024年4月施行）。市区町村と児童相談所の役割分担がより明確になった。

| 場面 | 市区町村（こども家庭センター等） | 児童相談所 |
|---|---|---|
| **虐待通告の受理** | ○（市区町村が受理可） | ○（189番への通告も含む） |
| **初期調査・リスクアセスメント** | ○（軽度・中度ケース） | ○（高リスクケース・一時保護が必要な場合） |
| **一時保護の決定・実施** | ✕ | ○（児童相談所長の権限） |
| **施設入所・里親委託の措置** | ✕ | ○ |
| **在宅支援・見守り** | ○（主担当） | ○（高リスクケースの指導） |
| **要保護児童対策地域協議会（要対協）の運営** | ○（設置義務） | ○（構成機関として参加） |
| **特別養子縁組のあっせん** | ✕ | ○ |

**重要**: 「一時保護が必要かどうか」の判断権限は**児童相談所のみ**。市区町村が「この子は危険だ」と判断しても、一時保護は児童相談所に依頼するしかない。

## 虐待対応フロー（市区町村から見た視点）

```
虐待・ネグレクト等の疑いの把握
  │
  ├─ 通告義務：発見者は市区町村または児相に通告義務（児童福祉法第25条）
  │
  ▼
市区町村（こども家庭センター）が受理
  │
  ├─【軽度・在宅支援で可】→ 市区町村が主担当で継続支援
  │
  └─【高リスク・一時保護が必要】→ 児童相談所に送致・協議
                                    ↓
                               児童相談所が調査・アセスメント
                                    ↓
                               一時保護の判断（児相長の権限）
```

## 要保護児童対策地域協議会（要対協）

虐待・非行・障害等の要保護児童に関する情報共有・支援調整のための協議機関。市区町村が設置（設置義務）し、児童相談所・学校・保育所・警察・保健センター・民生委員等が参加する。

**重要**: 要対協で共有された個人情報は**守秘義務の対象**（児童福祉法第25条の5）だが、支援目的での共有は可能。「個人情報だから共有できない」という誤った理解が支援の空白を生みやすい。

---

## 構造的ギャップ

### ギャップ1：「通告したが動いてくれない」という認識のズレ

市区町村や学校が「虐待の可能性がある」として児童相談所に情報提供・相談しても、「一時保護が必要なほど重篤ではない」と判断されて返ってくることがある。通告した側は「無視された」と感じるが、児相側は「在宅支援のレベル」と判断している。この認識のズレから、情報共有が止まり重大事態が起きるまで放置されるケースがある。

### ギャップ2：「一時保護後の情報が市区町村に戻ってこない」

一時保護が行われた場合、その後の処遇（施設入所・在宅復帰等）の情報が市区町村に共有されないことが多い。在宅復帰した子どもへの見守りを市区町村が担うべきところ、「児相案件は終わった」と双方が思い込んで空白が生じる。

### ギャップ3：中核市・特別区の設置状況の格差

設置が進んでいない中核市では、都道府県の児童相談所が広い管轄エリアを担当するため、地理的に遠い相談所に相談・通告しなければならない。管轄の児童相談所が「どこか」を現場担当者が知らないことがある。

### ギャップ4：ヤングケアラーとの境界

ヤングケアラーのうち、虐待・ネグレクトとの境界が曖昧なケースで「通告すべきか支援すべきか」の判断が現場で迷われる（→ `concepts/young-carer.md` 参照）。

---

## 連携のポイント

- 「この子どもの安全が心配だ」という状況では、まず**189番（児童相談所全国共通ダイヤル）または管轄児童相談所に相談**する。通告は義務であり、「確信が持てないから」という理由で躊躇しないことが重要。
- 要対協の定例会を使って、「高リスクではないが気になるケース」を定期的に情報共有する。一時保護になるまで児相に連絡しない運用は危険。
- 一時保護・施設入所措置後も、在宅復帰の情報を必ず市区町村が受け取る仕組みを作る（引き継ぎシートの活用等）。
