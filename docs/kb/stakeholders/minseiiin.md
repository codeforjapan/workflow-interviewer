# 民生委員・児童委員プロファイル

> **このプロファイルの役割**
> 民生委員・児童委員の制度的位置づけ、市区町村との役割分担の典型パターン、
> 連携が機能しないときの具体的なギャップを記述する。

---

## 組織の役割

### 法的根拠と組織属性

| 項目 | 説明 |
|---|---|
| **法的根拠** | 民生委員法（1948年制定） |
| **組織属性** | **非常勤の地方公務員**。ただし報酬なし（実費弁償のみ）|
| **選任方法** | 市区町村長が推薦し、都道府県知事が委嘱（任期3年） |
| **選任者の背景** | 一般市民。自営業者・会社員・農業者など。特別な資格・試験はない |
| **総数** | 全国約23万人 |
| **配置単位** | 一定数の世帯（約70～100世帯）ごとに1名配置 |

### 民生委員の身分の特殊性

```
【通常の公務員】
└─ 市区町村の職員として給与・福利厚生あり

【民生委員】
├─ 地方公務員（身分あり。秘密保持義務など）
├─ 報酬なし（完全ボランティア）
├─ 実費弁償のみ（月数千円程度の活動費）
├─ 兼職が一般的（別に仕事を持っている）
└─ 身分保障なし（更新されないこともある）

【児童委員】
└─ 民生委員が兼務する（約8割の民生委員が児童委員でもある）
```

**重要：民生委員は「自治体の末端職員」ではなく「市民代表」**

## 主な機能と活動

### 法定責務

民生委員法で定められた活動：

| 責務 | 内容 |
|---|---|
| **相談・援助** | 生活上の悩みや問題について相談に応じ、関係行政機関への橋渡し |
| **情報提供** | 福祉サービス・福祉制度に関する情報提供 |
| **見守り・実態把握** | 担当区域の生活困窮者・高齢者・児童等の実態把握と見守り |
| **関係機関への通報** | 要支援者を発見したときの行政機関への連絡 |
| **福祉増進活動** | 地域内の福祉活動の企画・実施（町内会との連携等） |

### 児童委員としての特有活動

| 活動 | 内容 |
|---|---|
| **子育て家庭の訪問** | 妊産婦・新生児家庭への訪問（健診機関と連携） |
| **児童虐待の通告** | 虐待の疑い発見時は児童相談所に通告 |
| **母子家庭の支援** | ひとり親家庭の情報提供・相談 |

### 実務的な活動（地域差が大きい）

- 敬老会の企画・運営
- 防災訓練への参加・呼びかけ
- 地域福祉ネットワーク会議への参加
- 学校との連携（欠席児童への声かけ等）
- 児童虐待防止啓発

## 自治体との典型的な役割分担

### 要支援者の発見・初期対応

| 段階 | 自治体の役割 | 民生委員の役割 |
|---|---|---|
| **発見** | 福祉事務所への申告 / 相談窓口 | **日常生活の中での気づき** / 「この家族、最近元気がない」という目撃 |
| **初期接触** | 相談面接 / 要件確認 | **最初の声かけ** / 「大丈夫ですか」という話しかけ |
| **信頼構築** | 制度説明・サービス紹介 | **定期的な見守り** / 顔なじみの関係性からの相談受け |
| **支援実施** | 給付決定 / 支援の実行 | **生活の傍にいる** / 困ったときの頼れる人 |

**→ 「行政の目が届かない隙間」を民生委員が埋める構図**

### 災害時要支援者

| 局面 | 自治体 | 民生委員 |
|---|---|---|
| **平時の名簿管理** | ○（要支援者名簿作成） | ×（秘密保持の観点から非公開が多い） |
| **顔なじみ関係** | ×（官的・距離がある） | **○（日常の接触がある）** |
| **災害発生時の安否確認** | 動員不足の中で難しい | **○（区域内を歩き回る）** |
| **避難所での見守り** | 行政職員は統括業務が多い | **○（居住者の動向把握）** |
| **復興期のケア** | ○（生活再建支援制度等） | **○（精神的支援・寄り添い）** |

**→ 日常のネットワークが災害時の生命線になる**

### 乳幼児訪問・母子保健

| 機能 | 自治体（保健センター） | 民生委員・児童委員 |
|---|---|---|
| **新生児把握** | 出生届から把握 / 乳幼児健診の案内 | ×（個人情報制限で連携困難） |
| **予防接種の勧奨** | 健診時の説明 / ハガキ通知 | △（町内会経由で周知を手伝う場合も） |
| **産後うつスクリーニング** | 保健師の健診での発見 | ×（医学的な判断ができない） |
| **児童虐待の早期発見** | 健診・学校の報告 | **○（日常の訪問で気づきやすい）** |
| **育児困難家庭への寄り添い** | 支援プログラム実施 | **○（定期的な見守り・相談相手）** |

**→ 「健診は一時点、民生委員は継続」という棲み分け**

## 連携が機能しないときに起きるギャップ

### ギャップ1：民生委員が行政に情報を届けたくても「誰に連絡すればいいかわからない」

**現象**
- 民生委員が「この高齢者、ガス代が払えないようだ」という情報を入手する
- だが「福祉課のどの部署に、どのように報告すべきか」がわからない
- 電話して「福祉課のことは福祉課に」と言われるが、「誰を探せばいい」かわかない
- 結局報告しないまま、数ヶ月が経つ

**原因**
- 自治体側に「民生委員からの報告を受ける窓口」が不明確
- 民生委員向けの「報告マニュアル」がない
- 福祉課の組織図が公開されていない（地域福祉担当課など、部署名が地域によって異なる）
- 民生委員の研修で「報告先」が教えられていない

**防止策**
- 自治体から民生委員へ「生活困窮情報の報告先」を明記した書類を配布
  （例：「福祉課 地域福祉係 ○○（内線○○） ○○○－△△△△」）
- 民生委員を対象にした年1回の説明会で、組織図と報告フローを説明
- 報告様式をテンプレート化（必須項目：相談者名・内容・発見日・報告日）
- 報告後、自治体側からフィードバック（「対応しました」「継続支援します」など）を返す

### ギャップ2：民生委員の交代期（3年ごとの改選）に情報と人間関係が引き継がれない

**現象**
- 3年間かけて「この区域の見守り対象者10名」を把握していた民生委員が退任
- 新任民生委員が赴任したが、「前任者の見守り対象者が誰か」を知らない
- 自治体側も「前任者から情報を受け取ったか」を把握していない
- 数ヶ月間、見守りが空白になる

**原因**
- 民生委員の引き継ぎは形式的（辞令・看板の受け渡しのみ）
- 見守り対象者のリストが自治体に集約されていない（民生委員の個人ノート任せ）
- 新任民生委員の担当区域が広すぎて、年内に全世帯を認識できない
- 自治体から新任民生委員への情報提供が遅い

**防止策**
- 自治体で「民生委員の見守り対象者リスト」を一元管理
  （個人情報保護に留意しつつ、福祉課内で共有）
- 交代時に「申し引き書」を形式化（前任者に内容確認させる）
- 新任民生委員の赴任から1ヶ月以内に、福祉課職員が「区域内の相談事例・支援中の世帯」をブリーフィング
- 3ヶ月目に再度確認（「新しく気づいたことはありますか」）

### ギャップ3：高齢化した民生委員が担当区域を十分カバーできないエリアの増加

**現象**
- 民生委員自身が70代後半。足腰が弱く、広い区域の全戸訪問ができない
- 「平時は自治会役員の互選で形式的に決まっている」状態が続く
- 実質的には「自治会長の知人」という配置になり、見守り機能が失われている
- 特に独居高齢者が集中しているエリアでカバー不足

**原因**
- 民生委員の担当区域が一律決まっている（70～100世帯）が、実際の負担は地形・密度で大きく異なる
- 複数の民生委員が連携する仕組みがない
- 後継者不足（引き継いでくれる人がいない）
- 民生委員の無給化で、若い人がなり手になりにくい

**防止策**
- 担当区域の見直し（山間部など地形が複雑な地域は世帯数を減らす）
- 複数民生委員による「ペアリング体制」（経験者と新任者を組ませる）
- 民生委員補充員制度の活用（専任ではなく、補助的に支援する制度がある自治体がある）
- 活動費・研修費の増額検討（わずかでも待遇改善が候補確保につながることもある）

### ギャップ4：児童虐待の疑いを発見しても「通告かどうか判断がつかない」で報告が遅れる

**現象**
- 民生委員が「その子どもの様子がおかしい、怪我が多い」と気づく
- だが「これは虐待か、それとも不注意か」判断がつかない
- 児童相談所に「報告していいのか」と迷っている間に、日時が経つ
- 結果的に対応が遅れる

**原因**
- 虐待の兆候について、民生委員への教育が十分でない
- 「通告の義務があることは知っているが、何を通告するのか」が曖昧
- 児童相談所への連絡方法（電話番号・流れ）が周知されていない
- 「通告=警察に突き出す」と誤解している民生委員もいる

**防止策**
- 児童虐待防止啓発の研修を、民生委員向けに年1回実施
  （「怪しいな」という段階で通告していい、という説明が重要）
- 児童相談所から直接民生委員へ「通告方法」を案内
- 「児童虐待相談電話」のポスター・シール配布（民生委員の自宅に貼ってもらう）
- 通告実績を匿名で民生委員にフィードバック（「情報ありがとうございました」という返事）

## 連携ポイント（ベテランの暗黙知の構造化）

「この場面では民生委員に相談する」という典型的なトリガーのリスト。

### 福祉事務所ケースワーカー向け

- **困窮世帯の転入があるとき**
  転入後、その世帯の民生委員に「困窮者です」と事前情報を伝える（同意のもと）。民生委員が「生活スキル向上」や「親戚への働きかけ」などの相談相手になりうる。

- **高齢者の孤立リスクが高い場合**
  「この人、病院にも行かず、社協の福祉用具貸出サービスも使っていない」という気づきは、定期訪問する民生委員からよく聞く。

- **児童虐待の事前通告がされていない時点で、学校から相談があったとき**
  民生委員は家庭訪問の機会が多いため、家庭内の様子をより詳しく知っていることがある。

### 市町村福祉課（地域福祉担当）向け

- **地域での生活課題（ゴミ屋敷・野良犬・不法投棄等）が報告されたとき**
  民生委員に情報を共有し、「この家族の背景にどんな問題があるか」を聞く。福祉課職員では気づかない事情（親族の死亡、精神疾患の悪化など）が見えることがある。

- **健康寿命が短い地域での健康寿命延伸プロジェクト**
  「運動習慣がない人をどう巻き込むか」は、民生委員のネットワークが有効。

- **防災訓練で「要支援者の安否確認」をシミュレーションするとき**
  実際の災害時には、自治体職員では足りない。民生委員をどう動員するか、事前に打ち合わせが必要。

| **個人の強み・課題** | 「○○さんは高齢で訪問が難しい」 / 「△△さんは積極的に虐待通告している」など |

| **過去の事例** | 「去年、民生委員が3ヶ月で交代したため、見守りが停滞した」など |
| **連携の形式** | 「引き継ぎ書の様式」 / 「情報提供のマニュアル」など、既にあれば整理 |
