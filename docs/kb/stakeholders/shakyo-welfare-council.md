# 社会福祉協議会（社協）プロファイル

> **このプロファイルの役割**
> 社会福祉協議会の制度的位置づけ、自治体との役割分担の典型パターン、
> 連携が機能しないときの具体的なギャップを記述する。

---

## 組織の役割

### 法的根拠と組織属性

| 項目 | 説明 |
|---|---|
| **法的根拠** | 社会福祉法 第109条（都道府県・市区町村に設置の義務） |
| **設置者** | 地方自治体（だが運営は独立） |
| **組織属性** | 民間（行政ではない）。ただし補助金で運営されるため、準公的機能を担う |
| **基本理念** | 「社会福祉に関する活動への参加」と「連帯」を基本とする民間福祉活動 |
| **経営体制** | 理事会・評議員会による民主的運営。職員は常勤スタッフ |

### 社協と自治体の法的な違い

```
【自治体（福祉事務所・福祉課）】
├─ 行政機関（法的拘束力あり）
├─ 実施責任あり（例：生活保護の決定権）
├─ 個別の権利・給付の根拠法規を直接適用
└─ 強制力あり（処分・指導など）

【社協】
├─ 民間団体（法的拘束力なし）
├─ 実施責任なし（支援活動は任意）
├─ 法人の定款と事業計画に基づく活動
└─ 強制力なし（説得・調整のみ）
```

**重要：社協は「福祉業務を代行する機関」ではなく「補完する機関」**

## 主な機能と事業

社協が全国的に実施している標準的な事業：

| 事業 | 内容 | 自治体との関係 |
|---|---|---|
| **日常生活自立支援事業** | 認知症高齢者・知的障害者の日々の金銭管理・書類整理をサポート | 福祉課が利用者を紹介することが多い |
| **ボランティアセンター** | 地域ボランティアのコーディネート・育成・活動支援 | 災害時に自治体と連携して支援物資・人員配置 |
| **生活福祉資金貸付** | 低所得世帯への生活費・医療費・住宅改修費などの貸付（厚労省補助）| 生活保護申請前の相談窓口として機能 |
| **成年後見支援** | 成年後見人養成研修、社協独自の法人後見受任 | 自治体が申し立てた後見人候補者になることも |
| **地域ケア会議事務局代行** | 介護保険法で定める地域ケア会議の事務運営・書記 | 介護保険課から委託。会議の場は自治体主催が原則 |
| **福祉用具・車いす貸出** | 地域住民への短期貸出 | 低所得者対象。有料（一部自治体補助） |
| **住民相談窓口** | 困りごと・福祉サービスについての一般相談 | 自治体の福祉窓口と並行。相談内容で相互紹介 |

## 自治体との典型的な役割分担

### 生活困窮者支援における分担

| 段階 | 自治体（福祉課）が担う部分 | 社協が担う部分 |
|---|---|---|
| **発見** | 生活保護申請 / 福祉事務所の相談受付 | 日常の見守り / ボランティア経由の情報 |
| **初期相談** | 制度説明 / 要件判定 | 困りごとの受け止め / 生活保護のイメージ払拭 |
| **支援開始前** | 生活福祉資金の該当確認 | 生活福祉資金貸付の審査・実行（独立） |
| **保護決定後** | 生活保護の実施・給付決定 | 日常的な見守り・寄り添い（福祉課と並行） |
| **退出支援** | 就労支援プログラム / ハローワーク連携 | 生活スキル向上研修（定義あいまい） |

**→ 「保護決定は自治体」「日常支援は社協」という棲み分けが理想だが、実務的には曖昧**

### 地域ケア会議における分担

| 要素 | 自治体 | 社協 |
|---|---|---|
| **会議の召集・主催** | ○（必須） | ×（事務局のみ） |
| **実施方針の決定** | ○ | × |
| **議題設定** | ○ | △（提案はできるが自治体が判断） |
| **事務局・書記** | △（多くは社協に委託） | ○ |
| **関係者調整** | △ | ○ |
| **アクションの決定・管理** | ○（責任） | △（実務） |

**→ 「誰がアクション管理するか」が曖昧なことが多い**

### 成年後見制度における分担

| 場面 | 自治体（福祉課等） | 社協 |
|---|---|---|
| **後見申し立ての判断** | ○（実施責任あり） | ×（相談受け受けのみ） |
| **申し立て実行** | ○ | ×（自治体に提案可） |
| **法人後見の受任** | ×（利益相反） | ○（中核的な役割） |
| **後見人候補者の確保** | △（親族がいない場合） | ○（市民後見人育成） |
| **成年後見制度利用促進** | △（法的義務あり） | ○（啓発・研修） |

## 連携が機能しないときに起きるギャップ

### ギャップ1：社協が把握している困窮者情報が自治体に来ない

**現象**
- 社協の相談窓口に「ガス代が払えない」という相談が来ている
- ボランティアから「一人暮らしの高齢者が食事をしていないようだ」という通報がある
- だが自治体の福祉課に情報が届かない
- 数ヶ月後、その住民が孤立死することもある

**原因**
- 個人情報保護の名目で、社協と自治体の情報共有ルールが不明確
- 「同意なしに福祉課に伝えるのは違法では」という過度な心配
- 逆に自治体が「社協に情報をくれ」と要求する方法がない
- 情報連携の担当者が定まっていない

**防止策**
- 「困窮者情報の共有に関する申し合わせ」を文書化
- 個人情報保護法の「公益性のある支援」という例外を明記
- 定期的な情報共有ミーティング（月1回など）をルール化
- 緊急時の即報体制（電話・ホットライン）を決める

### ギャップ2：地域ケア会議で「アクションが決まらない会議」が常態化

**現象**
- 「80歳の独居、ADL低下が目立つ。転倒リスク高い」と事例が報告される
- 「ケアマネと福祉用具屋と訪問看護と社協で見守ろう」ということになる
- 会議終了。2週間後、誰も動いていない。再び会議で同じ事例が報告される

**原因**
- 会議資料に「○○さんは訪問看護を検討」と書いてあるが、誰が提案するのか不明
- 社協事務局が「それは介護保険課が？」と思っている
- 介護保険課職員は「ケアマネが提案するもの」と思っている
- アクションの所有者がいない

**防止策**
- 会議で「決定」ではなく「実行形」で記録（例：「社協が○月○日に訪問看護事業所に紹介する」）
- アクション担当者を必ず指名、内線番号まで記載
- フォローアップ日を決める（○月○日に進捗確認ミーティング）
- 「決まったのに動かない」ケースを四半期ごと分析

### ギャップ3：成年後見の受任者が見つからず支援が止まる

**現象**
- 「この人は法人後見が必要」と判定されたが、社協も忙しく、個人後見人の候補もいない
- 結果、6ヶ月間、誰も財産を管理していない状態が続く
- その間に住宅ローンが滞納される

**原因**
- 社協の法人後見の受け入れ限度が決まっている（人員不足）
- 市民後見人養成研修の修了者数が少ない
- 自治体との連携で「申し立ては自治体の責任」となり、後見人探しは社協任せになっている
- 長期的な支援計画がない

**防止策**
- 市民後見人養成研修を毎年実施、修了者100人規模を目指す
- 社協の法人後見職員を増員するための補助予算を自治体で確保
- 弁護士・司法書士との連携体制を構築
- 申し立ての際に「後見人確保計画」を自治体・社協で事前に話し合う

### ギャップ4：社協職員と自治体CWが「お互い何をやっているかよく知らない」まま並行支援

**現象**
- 同じ困窮世帯について、社協の福祉サービス提供者（相談員）と自治体CW（ケースワーカー）が別々に対応
- 社協は「この人は就労支援が必要」と考えている
- 自治体は「この人は就労可能でない」と評価している
- 支援者間で矛盾した対応になり、本人が混乱

**原因**
- 定例の事例検討会がない
- 「支援の仕分け」が決まっていない（自治体が全て対応するのか、社協に任せるのか）
- 記録の共有化がなされていない
- 職員の異動時に引き継ぎがない

**防止策**
- 月1回、自治体と社協の関係職員による事例検討会をルール化
- 「共通支援対象者」を定期的にリストアップし、分担を確認
- 支援記録の共有ルール（何を共有するか、共有頻度）を文書化
- 人事異動時には新任職員向けの「連携研修」を実施

## 連携ポイント（ベテランの暗黙知の構造化）

「この場面では社協に連絡する」という典型的なトリガーのリスト。

### ケースワーカー向け

- **困窮世帯の日常的な見守りが必要になったとき**
  生活保護開始後、「毎日の生活スキル向上」「金銭管理」などの支援が必要な場合、社協の福祉サービス提供者に相談。

- **高齢者の財産管理能力が怪しくなったとき**
  「最近、訪問販売に騙されたようだ」「通帳管理ができなくなった」という情報キャッチ時は、社協の成年後見相談窓口に即相談。14日ルール（申し立てから認可まで）がある。

- **地域での生活課題（ゴミ屋敷・孤立等）を発見したとき**
  町内会から通報があった場合、社協のボランティアコーディネーターに相談すると、近所の見守りボランティアネットワークと連携しやすい。

### 介護保険課向け

- **要介護認定者の転出入があるとき**
  社協が地域ケア会議の事務局担当の場合、転出前に本人の支援内容をヒアリングし、新自治体に引き継ぎ資料を作成。

- **地域ケア会議のアクション進捗が停滞しているとき**
  社協に「この事例のフォローアップを主導してほしい」と依頼。社協は関係者調整に強い。
