## 組織概要

| 項目 | 内容 |
|---|---|
| 正式名称 | ○○都道府県後期高齢者医療広域連合（都道府県ごとに47連合） |
| 組織属性 | **特別地方公共団体（地方自治法第284条第2項に基づく広域連合）** |
| 設置根拠 | 高齢者医療確保法 第48条（都道府県ごとに全市区町村が加入義務） |
| 保険者 | 広域連合（47連合が全国の75歳以上の後期高齢者を担当） |
| 財源 | 保険料（1割）+ 公費（5割：国・都道府県・市区町村）+ 支援金（4割：現役世代から） |

⚠️ **後期高齢者医療は「市区町村」が保険者ではなく、「広域連合」が保険者。**
住民は市区町村の窓口で手続きするが、制度を運営しているのは別の組織（広域連合）である。

---

## なぜ広域連合が必要か

75歳以上の高齢者は疾病リスクが高く、市区町村単位で保険を運営すると高齢者の多い自治体の財政が成り立たない。そのため全国共通の財政調整の仕組みとして、**都道府県内の全市区町村が共同で設立する「特別地方公共団体」**に保険者機能を集約している。

---

## 市区町村と広域連合の役割分担

| 業務 | 市区町村 | 後期高齢者医療広域連合 |
|---|---|---|
| **資格取得・喪失の届出受付** | ○ | × |
| **保険証の発行** | × | ○（広域連合が発行、市区町村を通じて配布） |
| **保険料の賦課（計算・通知）** | × | ○ |
| **保険料の徴収** | ○（市区町村が特別徴収・普通徴収を実施） | × |
| **各種申請の窓口受付** | ○ | × |
| **保険給付の決定・支払** | × | ○ |
| **高額療養費の支給** | 受付のみ | ○（決定・支給） |
| **不服申立ての審査** | × | ○（広域連合が審査） |

**重要**: 住民から見ると「市役所で全ての手続きができる」ように見えるが、保険料の決定・保険給付は全て広域連合が行っている。「なぜこの保険料になったのか」という問い合わせに市区町村が答えるためには、広域連合からの賦課情報を把握している必要がある。

## 75歳到達時の自動移行フロー

```
75歳の誕生日の前日まで
  └─ 国民健康保険 または 健康保険（被扶養者含む）に加入

75歳の誕生日当日
  ├─ 後期高齢者医療制度に自動的に移行（届出不要）
  ├─ 広域連合から後期高齢者医療保険証が送付される
  └─ 国民健康保険は自動脱退（手続き不要だが確認は必要）
```

**問題が起きやすい場面**: 保険証の「切れ目」。旧保険証の有効期限切れ〜新保険証到着の間に医療機関受診が重なると、窓口での資格確認でエラーが発生する（→ aging-and-care.md の穴A）。

---

## 構造的ギャップ

### ギャップ1：「保険者が誰か」を住民も職員も把握していない

住民は「市役所が後期高齢者医療をやっている」と思っている。職員側も「窓口は自分たちだが決定は広域連合」という関係を説明できないことがある。「不服申立てはどこに？」「保険料の計算が間違っている場合はどこに言えばいい？」という問いに正確に答えるためには、広域連合との役割分担の理解が必須。

### ギャップ2：保険料通知と市区町村の徴収の乖離

広域連合が保険料を「計算・通知」し、市区町村が「徴収」するという二段構えのため、「なぜこの金額か」を市区町村窓口が説明できないケースがある。住民が「市役所に文句を言っても解決しない」状態になりやすい。

### ギャップ3：死亡後の手続き情報の連携

後期高齢者医療の資格喪失手続き（死亡時）は市区町村が窓口となるが、高額療養費の精算・保険料の過誤納還付は広域連合が処理する。死亡後の保険料精算や高額療養費の還付が遺族に伝わらないケースがある。

---

## 連携のポイント

- 75歳到達者への後期高齢者医療への移行案内は、誕生月前後に自動的に広域連合から通知が行く。市区町村は手続き不要だが、国保脱退の確認と保険証の切れ目対策の案内が必要。
- 保険料に関する問い合わせは、賦課の根拠・計算方法の詳細は広域連合へ案内する。市区町村は「徴収のみ」担当であることを説明する。
- 不服申立て（審査請求）は広域連合に対して行う。市区町村に持ち込まれた場合は広域連合への案内が必要。
- 死亡時の資格喪失届受付後、高額療養費未申請の有無を確認し、必要であれば遺族に広域連合への請求を案内する。
