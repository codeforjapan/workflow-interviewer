## 組織概要

| 項目 | 内容 |
|---|---|
| 正式名称 | 福祉事務所（社会福祉事務所、市福祉事務所、○○町福祉事務所等、名称は自治体による） |
| 設置根拠 | 社会福祉法第14条 |
| **設置義務** | 都道府県・市・特別区は**設置義務あり**。町村は任意設置 |
| 職員構成 | 所長・査察指導員（スーパーバイザー）・現業員（ケースワーカー） |

⚠️ **「福祉事務所 ＝ 市役所の中の一課」とは限らない。**
都道府県が設置する福祉事務所は、町村部をカバーする独立した出先機関である。

---

## 設置形態と実施機関の関係

```
都道府県設置の福祉事務所
  └─ 管轄: 都道府県内の「町村部」（市・特別区を除くエリア）
      └─ 生活保護の実施責任は都道府県
      └─ 所在地: 都道府県の出先機関（保健福祉センター等に併設が多い）

市・特別区設置の福祉事務所
  └─ 管轄: 市・特別区のエリア
      └─ 生活保護の実施責任は市・特別区
      └─ 所在地: 市役所内（または別棟・出張所として分離）
      └─ 指定都市・中核市は独自に設置義務（都道府県から権限委任）

町村設置の福祉事務所（任意）
  └─ 設置している場合: 町村が生活保護の実施機関になる
  └─ 設置していない場合: 都道府県の福祉事務所が実施機関
```

## 主な所管業務

| 業務 | 根拠法令 |
|---|---|
| **生活保護の申請受付・決定・実施** | 生活保護法第19〜25条 |
| **老人福祉措置（特養入所判定等）** | 老人福祉法第10〜11条 |
| **障害者福祉措置** | 各障害者福祉法 |
| **母子・父子自立支援** | 母子及び父子並びに寡婦福祉法 |
| **ホームレス自立支援** | ホームレスの自立の支援等に関する特別措置法 |

生活保護が最も重要・負荷の大きい業務。現業員（ケースワーカー）が個別ケースを担当し、定期訪問・支援調整・給付管理を行う。

---

## 生活保護の実施における職員構成

| 役職 | 役割 | 標準人数比 |
|---|---|---|
| **所長** | 福祉事務所全体の管理・保護決定の最終権限者 | 1名 |
| **査察指導員（スーパーバイザー）** | 現業員の技術的指導・困難事例のスーパービジョン | 現業員7名につき1名 |
| **現業員（ケースワーカー）** | 担当世帯の訪問調査・保護決定補助・自立支援 | 1名あたり標準80世帯（実態は超過が多い） |

**全国共通の問題**: 標準では1ケースワーカーが80世帯を担当するとされているが、実態は100〜200世帯を担当する自治体も多く、訪問頻度の低下・支援の形骸化につながっている。

## 市区町村窓口との役割分担

多くの市では「市役所の一課」として福祉事務所機能が組み込まれているが、建前上は独立した機関。

| 場面 | 市区町村の一般窓口 | 福祉事務所 |
|---|---|---|
| 生活保護の相談 | 案内・前段の話し聞き取り | 申請受付・審査・決定 |
| 生活困窮者自立支援 | 支援拠点の設置（委託可） | 連携支援（実施責任は市） |
| 老人福祉措置（緊急入所等） | — | 措置権限あり |
| 障害者福祉サービスの支給決定 | 申請受付 | 支給決定（市設置型） |

## 構造的ギャップ

### ギャップ1：「どの福祉事務所に相談すればよいか」が住民にわからない

都市部では「市役所 ＝ 福祉事務所」が実質的に一体になっているが、地方の町村では「都道府県の福祉事務所＝役場から離れた出先機関」が担当しており、住民が自力でたどり着けないことがある。

### ギャップ2：申請権の侵害（水際作戦）

生活保護は**申請権が保障されている**（生活保護法第7条）にもかかわらず、「まず親族に相談してから」「預金を全部使ってから」「住所がないと申請できない」などの誤った案内により申請を妨げる「水際作戦」が全国で問題化している（→ INC-003）。

福祉事務所の所長・ケースワーカーの判断に依存する部分が大きく、担当者が変わると対応が変わることが構造的問題。

### ギャップ3：ケースワーカーの過重負担と訪問頻度の低下

標準80世帯を大幅に超える担当世帯数を抱えるケースワーカーが多く、法定の訪問調査（年4〜6回）が形式化している。この結果、支援が必要な世帯の変化（健康悪化・ネグレクト・暴力等）を把握できず、孤立死・重大事態につながるリスクがある（→ INC-004）。

### ギャップ4：他機関との情報共有ルールの不備

生活保護受給者が入院・施設入所した場合の医療機関・介護施設との情報共有、ケアマネジャー・地域包括支援センターとの連携が属人的。担当ケースワーカーの異動時に引き継ぎが不完全になり、支援の空白が生じる（→ INC-005）。

---

## 連携のポイント

- 「生活が苦しい」「もう生きていけない」という相談には、福祉事務所への案内を迷わず行う。相談者に申請意思がある場合は「申請書を渡す・同行支援を案内する」まで責任を持つ。
- 町村部の場合は「どの都道府県福祉事務所が管轄か」を事前に確認しておく。担当エリアの境界で混乱が起きやすい。
- 地域包括支援センター・社協と連携して困窮者を把握した場合、福祉事務所への情報連携フローを決めておく（個人情報保護の観点からの申し合わせが必要）。
- 生活保護受給者の転入・転出時（INC-005参照）は、転出元の福祉事務所との引き継ぎを確認する。
