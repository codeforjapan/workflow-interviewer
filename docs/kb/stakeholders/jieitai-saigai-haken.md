## 組織概要

| 項目 | 内容 |
|---|---|
| 正式名称 | 陸上自衛隊・海上自衛隊・航空自衛隊 |
| 所管 | 防衛省（国の機関） |
| **組織属性** | **国の機関**（市区町村・都道府県の指揮下にない） |
| 災害派遣の根拠 | 自衛隊法第83条（都道府県知事の要請に基づく） |
| 派遣指揮 | 防衛大臣・自衛隊の最高指揮官（都道府県知事や市区町村長の指揮下にない） |
| 主な災害対応機能 | 人命救助・行方不明者捜索・物資輸送・入浴支援・医療支援・道路啓開 |

⚠️ **自衛隊は市区町村の指揮下にない。派遣要請は都道府県知事が行う。**
市区町村長が直接自衛隊に要請することはできない（知事への要請経由が原則）。

---

## 災害派遣の要請フロー

```
大規模災害発生
  │
  ├─【市区町村の対応】
  │   市区町村長 → 都道府県知事に「自衛隊派遣の要請」を依頼
  │
  ├─【都道府県の対応】
  │   都道府県知事 → 防衛大臣（または地方防衛局）に「自衛隊の災害派遣要請」
  │
  └─【自衛隊の対応】
      防衛大臣の命令 → 部隊が派遣される
      ※ 要請がない場合でも「自主派遣」が可能（人命に危険が迫る場合）
```

⚠️ **自主派遣（自衛隊法第83条第2項）**: 被害が明らかで緊急を要する場合、知事からの要請を待たずに自主的に派遣できる。東日本大震災・熊本地震等で適用された。

## 市区町村との役割分担

| 機能 | 市区町村 | 自衛隊 |
|---|---|---|
| **人命救助・捜索** | ○（消防・警察が主体） | ○（大規模・広域・特殊装備が必要な場合） |
| **避難所の運営** | ○（運営主体） | △（物資輸送・炊き出し支援等） |
| **物資の輸送** | ○（陸上・近距離） | ○（ヘリ・大型車両による遠距離・大量輸送） |
| **入浴支援** | △（施設確保） | ○（野外入浴設備の展開） |
| **医療支援（DMAT等）** | △（受入調整） | ○（衛生部隊による医療支援） |
| **道路・橋梁の応急復旧** | ○（発注・管理） | ○（重機・工兵部隊による緊急啓開） |
| **孤立集落へのアクセス** | △（限界あり） | ○（ヘリ・船舶・特殊車両） |
| **遺体収容** | △（警察が主体） | ○（大規模捜索・収容に参加） |
| **原子力災害対応** | △（初期対応） | ○（NBC対応・専門部隊） |

## 自衛隊派遣の終了基準と「撤収問題」

自衛隊の派遣は本来「緊急性が解消された時点」で終了するが、実際には撤収の判断が難しい。「自衛隊が撤収したら支援が途絶える」という懸念から、派遣継続を求める自治体と、訓練・即応態勢維持のために撤収を求める自衛隊側のバランスが問題になることがある。

---

## 構造的ギャップ

### ギャップ1: 市区町村長は自衛隊に直接要請できない

市区町村長が「今すぐ自衛隊が必要」と判断しても、知事経由の要請が原則であるため、緊急時の迅速な対応が遅れることがある。このため、事前に都道府県・自衛隊と「具体的な連絡ルート」を確認しておくことが重要。

### ギャップ2: 「自衛隊が来たら何でもやってくれる」という誤解

自衛隊の活動範囲は自衛隊法上の制約がある。民間業者が通常実施すべき業務（廃棄物処理・一般的な建物解体等）を自衛隊に依頼することはできない。「自衛隊が何でもやってくれる」という誤解が、民間業者への発注遅れを招く場合がある。

### ギャップ3: 自衛隊の派遣と地元業者の調整

自衛隊が一部の復旧作業を行うと、地元建設業者・ボランティアとの役割分担が曖昧になる。平時から「自衛隊がやること」と「地元業者・ボランティアがやること」の役割分担を防災計画に明記しておくことが重要。

---

## 連携のポイント

- 自衛隊の派遣要請は知事に行うことを防災計画に明記し、連絡先・手順を事前に確認しておく。
- 自衛隊の来援が見込まれる場合、「受け入れ場所（ヘリポート・野営地）」と「要請内容（何をやってもらうか）」を事前に準備しておく。
- 撤収時期は自治体側から早めに申し入れる（自衛隊に長期依存しないための地元業者・ボランティアとの分担計画を立てておく）。
