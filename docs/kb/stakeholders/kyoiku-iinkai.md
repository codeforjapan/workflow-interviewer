## 組織概要

| 項目 | 内容 |
|---|---|
| 正式名称 | ○○市（区町村）教育委員会 |
| 設置根拠 | 地教行法第2条（市区町村・都道府県に設置義務） |
| **組織属性** | **首長から独立した執行機関**（行政委員会） |
| 構成 | 教育長（常勤）+ 教育委員（非常勤・4名以上）= 合議制 |
| 所管事務 | 学校の設置・管理・廃止、教育課程・教職員人事、社会教育、文化財保護等 |
| 首長との関係 | 教育方針は首長とは独立して決定。ただし大綱方針は首長が策定（H27改正で総合教育会議を設置） |

⚠️ **教育委員会は「市役所の一部局」ではなく、首長から独立した機関。**
首長が直接指揮・命令できないため、首長部局の福祉・保健担当との情報共有に**制度的な障壁**が生じる。

---

## 首長部局との役割分担

| 場面 | 首長部局（福祉・保健課等） | 教育委員会（学校含む） |
|---|---|---|
| **就学援助（学用品費・給食費等）** | 実施主体が首長部局の自治体あり | 実施主体が教育委員会の自治体あり（**自治体によって異なる**） |
| **ヤングケアラー支援** | 福祉的支援の調整 | 学校での把握・スクールソーシャルワーカー配置 |
| **虐待対応（学校での把握）** | 要保護児童対策地域協議会（市区町村設置） | 学校が通告義務者として通告・要対協に参加 |
| **不登校対応** | 生活保護世帯・困窮世帯の家庭支援 | 教育支援センター・フリースクール連携 |
| **障害のある子どもの支援** | 障害者手帳・障害福祉サービス（首長部局） | 特別支援学級・特別支援学校への就学指導 |
| **保育（0〜5歳）** | 認可保育所の管理・入所判定（こども家庭庁系） | 幼稚園（教育委員会所管） |
| **放課後児童クラブ（学童保育）** | 首長部局が実施する場合あり | 教育委員会が実施する場合あり（**自治体によって異なる**） |

## 就学援助の実施主体問題

就学援助（学用品費・給食費・修学旅行費等の補助）は、**自治体によって教育委員会実施型・福祉部局実施型・混在型がある**。

```
就学援助の実施主体（自治体によって異なる）
  │
  ├─ 教育委員会が実施
  │   → 申請先：学校（担任経由）or 教育委員会窓口
  │
  ├─ 福祉部局（子ども家庭課等）が実施
  │   → 申請先：福祉窓口
  │
  └─ 両者が別々に実施（就学援助 + 市独自補助の二本立て）
      → 住民が申請先に迷う
```

生活保護世帯は自動的に就学援助（教育扶助）の対象となるが、準要保護世帯（生活保護基準の1.1〜1.3倍以下）の申請ルートが自治体によって違うため、支援が必要な家庭に情報が届かないことがある。

## 構造的ギャップ

### ギャップ1：「教育の独立性」が情報共有の壁になる

首長部局と教育委員会は独立した機関であるため、児童の情報（欠席状況・家庭の困難・虐待の疑い等）を首長部局の福祉担当に共有する際に「守秘義務」「個人情報保護」を過度に解釈して情報が出てこないことがある。要保護児童対策地域協議会（要対協）が整備されていても、「要対協の案件になっていないもの」の情報は流れない。

### ギャップ2：スクールソーシャルワーカーの配置と活用の格差

スクールソーシャルワーカー（SSW）は教育委員会が配置する福祉専門職で、学校と福祉のつなぎ役を担う。しかし配置数は全国で大きく格差があり、SSWがいない学校では担任が福祉的判断をしなければならない。またSSWが「教育委員会の職員」として位置づけられると、福祉部局との情報共有が「部局間の連絡」となり手続きが重くなる。

### ギャップ3：就学援助の申請が届かない

準要保護世帯の就学援助は、制度を知っていて申請した人だけが受けられる。学校からの案内が年1回の文書配布のみであったり、申請が「恥ずかしい」という心理的ハードルがある。生活保護CWが子どもの就学援助を案内できるケースでも、教育委員会と福祉部局の情報連携がなければ漏れる。

### ギャップ4：特別支援学級と放課後等デイサービスの連携不足

特別支援学級に通う子どもが放課後等デイサービス（障害福祉サービス）を利用する場合、学校の情報と障害福祉の支援計画が連携されないため、二重の情報収集・アセスメントが発生する。教育委員会（学校）と障害福祉担当（首長部局）の間に情報共有の仕組みがない自治体が多い。

---

## 連携のポイント

- 虐待が疑われる児童の情報は、**学校（教育委員会）が要対協への共有または189への通告を行う義務**がある。福祉部局との情報共有を「個人情報」を理由に拒む教育委員会に対しては、要対協の目的（情報共有が法的に認められた目的）を明示する。
- 就学援助の申請ルートは自治体によって異なるため、生活保護世帯・困窮世帯への案内時に「どの窓口で申請するか」を確認した上で案内する。
- スクールソーシャルワーカーの役割と連絡方法を福祉・保健担当者が把握しておくことで、学校→福祉部局の橋渡しをスムーズにできる。
- ヤングケアラー支援については、首長部局・教育委員会・児童相談所の三者が「誰が主担当か」を明確にした協定・申し合わせを事前に作っておくことが全国共通の課題。

## 総合教育会議（H27地教行法改正）

首長と教育委員会が教育施策について協議・調整する場として、2015年から「総合教育会議」の設置が義務化された。大綱方針は首長が策定するが、個別の教育課程・教職員人事は引き続き教育委員会の権限。

この会議を通じて「子どもの貧困対策」「ヤングケアラー支援」等の首長部局との横断的課題を協議できるが、活用の深さは自治体によって大きく異なる。
